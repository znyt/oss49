　　那群丧尸去哪了？
<br/>　　安静，周围实在太安静了，第五小队引开的那群丧尸是离他们太远，还是踪迹难寻？
<br/>　　“要不要我过去查看一下？”一名第一小队的成员问道。
<br/>　　“不。”徐锋立即阻止，道：“别分散。”
<br/>　　话音刚落。
<br/>　　咻！
<br/>　　一道犀利的破空声响起，一下子从一人的耳畔擦过。
<br/>　　那名成员也是下意识反应极快，瞬间侧挪身体，但那不知名的物体还是擦破了他的耳旁皮肤，丝丝鲜血溢出。
<br/>　　五人顿时提高警惕注意周围，可一片黑暗中什么都看不见，声音气味也没有异常。
<br/>　　咻咻！
<br/>　　又是两道声音快速响起，擦着三个人的身躯边缘而过，其中一次更是距离一人的脖颈动脉极近，危险异常。
<br/>　　就这样时不时的，总有隐藏在暗处的攻击出现，大多数时候都只是擦肩而过，可随着时间的递增，已经有人被划伤皮肤流血不少，再这样下去第一小队早晚会被耗尽力气。
<br/>　　最重要的是，有不明物种就在周身，哪怕离开也不现实。
<br/>　　噗！
<br/>　　忽的一人被重重划破手臂，鲜血大量涌·出在地上，诡异的是气味并不浓烈，也不知是不是被四周的腐臭气掩盖。
<br/>　　那人立即在其他人的掩护下包扎封住伤口，抽空说道：“速度太快了，而且根本不知道是什么东西在攻击我们。”
<br/>　　可就是这么一会儿的功夫，掩护他的第一小队成员也纷纷中招，两人的身上背后都被划伤，更有少许伤口尤其贴近致命部位。
<br/>　　事态一下子严重起来，那隐藏在暗处的攻击也忽然加快了速度和频率，疯狂进攻。
<br/>　　有人开始尝试反击，可每次都无功而返不说，还落得身上鲜血淋淋。
<br/>　　“很奇怪。”一人喘着气说道：“你们不觉得很熟悉吗？对方似乎知道我们的弱点，小队模式的弱点。”
<br/>　　此话一出，相互间集合成圆的五人都心头一紧，五人小队的作战模式通常为大、中型战役之内的小组团队，组队成员带有极强的针对性。
<br/>　　很不巧的是当下正属于中型战役之中，他们刚刚完成了炸出埋骨之地一个外圈的任务，接下来的不过是继续前行，占领那个圈地而后朝内进攻。
<br/>　　所以在这一战之中的小组针对性并不是高端战斗，相互间的配合也并非以反击屠杀为主，这样一来在忽然遇到此番情况之时，憋屈感就极其强烈。
<br/>　　因为哪怕任何五个精英队成员互组成队，亦或是激进派的杀羽战队成员五个，都不至于说因为缺少某个合作者或配合不恰而导致被站着打。
<br/>　　那名成员所说的小队弱点，也就是在这里！
<br/>　　徐锋紧握金枪没有反击，而是一直在观察。
<br/>　　目光所落之处的前方都是正常视野范围，可偏偏余光每一次的移动都能诡异的察觉到什么不一样，眉头皱起，徐锋一边装作没有察觉的样子继续四处观察，一边不停变化角度感受自己的余光。
<br/>　　终于在这片范围走了好几次后，徐锋发现自己的余光总能时不时的看到一个阴影，体型时大时小每一次都是快速闪过，似乎是刻意躲着他们五人的视觉死角。
<br/>　　但每一次被余光捕捉之时的地方，却又出乎意料的很有局限性，只在以他们为中心的四、八、十二点钟这三个点附近，绝不超出一米。
<br/>　　察觉到这一规律的徐锋没有立即出手，而是缓慢的抬起头看向那三个点的上方，果然不出所料。
<br/>　　他竟然看到了三棵树！
<br/>　　目光快速下移，下方没有见到树干，一抬头却看到了那三个方向点上有树枝树梢！
<br/>　　没有扎入泥土难道生长在了半空中不成？
<br/>　　诡异的状况连徐锋都有些发愣，以至于让他又一次变幻方向确定此事真伪。
<br/>　　而就在此时，四点钟方向的那棵树梢忽然抖动了一下，诡异的树叶摩擦中都没有发出任何声音，反而是紧接着的一秒钟后，又是‘咻’的一声，直冲着徐锋的眉心而来！
<br/>　　噌！
<br/>　　一枪将什么东西挡住弹开，徐锋双手握枪紧盯着那个方向，一秒钟前看到了那棵树稍的动静，一秒钟后才进攻而来，在他这个等阶想挡住轻而易举。
<br/>　　且可以说明对方的速度并不快，甚至还很慢！
<br/>　　徐锋忽的眉头舒展，忽然抬脚快步朝前，紧接着——
<br/>　　噌！
<br/>　　又是一声被挡住的攻击，金属物与杀羽金枪的碰撞发出的声音，在这一片安静之中显得尤其响亮。
<br/>　　诡异的响亮！
<br/>　　“你们不觉得很安静吗？”忽的徐锋笑着开口，看着上方的那三棵树梢，道：“我们的五感被屏蔽了，对方不过是个战五渣。”
<br/>　　此话一出，其余的四名成员都一阵恍惚，猛然惊觉除了攻击时刻，这里竟然连寻常的风声都没有，更别说明明好几人受伤，可偏偏气味都是一成不变的丧尸腐臭气。

<br/>　　这也是为什么他们什么都听不到看不到的原因，对方没有隐身，只是反其道行之的将他们的五感扰乱了而已。

<br/>　　只是这些往往属于新人类的能力，怎么会出现在埋骨之地的敌人身上？

<br/>　　剩下的四人一开始还没反应过来，但在看了徐锋所看的方向后，也纷纷面色大变跟上徐锋步伐。
<br/>　　那棵树的晃动方向所呈现的并非进攻，而是逃跑！
<br/>　　朝前追击的五人一下子冲出了那围成三角地带的区域，往后望去还能看到身后的那三颗粗·壮树干，这些统统都在五感被屏蔽干扰时隐藏，此时脱离了屏蔽范围自然就能看见。
<br/>　　风声回归，嗅觉正常，当一切感受都回到正轨之时，第一小队也朝前方追击而去。
<br/>　　只是干扰屏蔽五感这么特殊稀少的能力，整个末世纪元三年以来他们都只听说过一次，那便是江凌若的弟弟江凌天的好友，有着五感干扰能力的新人类，八月。
