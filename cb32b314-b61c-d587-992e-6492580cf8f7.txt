        落以柔的小手情不自禁地抚上傅夜沉的脸庞。
<br/>
<br/>        她用心去感受属于他的温度。
<br/>
<br/>        “大哥，我是以柔……”落以柔眼神暧昧，声音柔情似水地轻唤。
<br/>
<br/>        其实，她早已在暗中独自唤过无数次对他的昵称。
<br/>
<br/>        清脆的嗓音回荡在寂静的病房里，那是她全心倾恋一个男人时才有的怦然悸动。
<br/>
<br/>        然而，回应她的却是一声轻浅而伴随着心碎的呓语。
<br/>
<br/>        “筱颖……”傅夜沉忽地翻过身，整个人压在她身上。
<br/>
<br/>        落以柔被他突如其来的翻身动作吓得动都不敢功一下，而原本该沉沉睡去的他，却忽然睁开一双深邃的黑眸。
<br/>
<br/>        心跳在这一瞬间漏了好几拍，落以柔气都不敢喘一下的望着这双如黑钻般的眸子。
<br/>
<br/>        那茫然而无焦距的眼眸明白的告诉落以柔，傅夜沉并不知道正被他抱着的是别的女人。
<br/>
<br/>        “筱颖……”被治疗药物深深影响的脑子混沌得无法思考，长相甜美而温柔的落以柔，在傅夜沉的眼里变成了他心爱的童筱颖。
<br/>
<br/>        “筱颖……”
<br/>
<br/>        他不断地喊着“筱颖”这个名字，双手紧紧搂着怀中柔软的身躯。
<br/>
<br/>        落以柔身心一怔。
<br/>
<br/>        他把她当成童筱颖了！
<br/>
<br/>        “筱颖，你可不可以爱我？”傅夜沉声音哑然，呢喃质问。
<br/>
<br/>        鼻息间涌入的属于女人身上才有的芳香，就像是罂粟花的香气，让他慢慢地上瘾……
<br/>
<br/>        傅夜沉的黑眸迷离，心里想的却是，筱颖是不是回心转意了？是不是愿意放弃阿景了？是不是不再跟他闹离婚了？是不是愿意回到他的身边了？
<br/>
<br/>        “大哥，我爱你。”落以柔呢喃细语地告白，她知道，他一定听不到。
<br/>
<br/>        但是，他好像听到了什么。
<br/>
<br/>        所以，她终于又回到他身边了，是吗？
<br/>
<br/>        傅夜沉的内心不断地渴望这个问题，能够有一个答案。
<br/>
<br/>        然而不断自问的结果，只得到“她不爱他”的事实，他的心逐渐崩塌。
<br/>
<br/>        筱颖不爱他……
<br/>
<br/>        筱颖从始至终都不会爱上他……
<br/>
<br/>        在心痛到无法呼吸，临近崩溃的边缘时，一双纤细的藕臂悄悄圈住他的颈项，给了他一个温暖的拥抱。
<br/>
<br/>        是筱颖吗？
<br/>
<br/>        是筱颖回到他身边来了吗？
<br/>
<br/>        这种让他从来没有过的安全感，仿佛被一双强而有力的手臂拥抱住般的令人安心。
<br/>
<br/>        傅夜沉甩了甩脑袋，用力眯了眯眼睛，想看清楚被自己压在身下的女人到底是谁。
<br/>
<br/>        甚至，他在怀疑，自己是不是又在做梦。
<br/>
<br/>        筱颖的容颜在视野里渐渐地变得模糊起来，再慢慢地出现在自己眼中的倩影变得逐渐清晰。
<br/>
<br/>        不是筱颖……
<br/>
<br/>        是落以柔！
<br/>
<br/>        傅夜沉蓦地清醒，猛然弹起身子，从落以柔身上翻身坐起。
<br/>
<br/>        落以柔顿感尴尬，连忙坐起身，挪着双腿下了床，穿上鞋子站在床边，聋拉着脑袋，羞涩不已地解释道：“那个……大哥……刚刚……刚刚是你把我给……”
<br/>
<br/>        其实，明明是她自己先爬上他的病床，依偎在他怀里的，但是，此刻，她不能让他知道这件事情。
<br/>
<br/>        “滚！”傅夜沉坐在床边，眸光微沉，不等落以柔把话说完，冷不丁地呵斥道。
<br/>
<br/>        落以柔倏然抬眸，看了傅夜沉一眼后，心痛，却斩钉截铁地反驳道：“我不能滚，是连医生让我来照顾你的。至少，我得等你的病情完全康复了，才能离开。这是我答应连医生的事情，说到就要做到。”
<br/>
<br/>        “……”傅夜沉淡漠地瞥了落以柔一眼，索性直接拔掉了手背上的留置针，一丝鲜血瞬间从针孔处喷射了出来。
<br/>
<br/>        落以柔吓得惊慌失措，大呼小叫道：“大哥，你这是做什么？你的点滴还没打完，你不能这样糟蹋自己！”
<br/>
<br/>        “给连华生打电话，让他来接我回私宅。”傅夜沉则淡定地从床边的置物柜里拿出棉签，按在了自己的手背上止血。
<br/>
<br/>        “是。”落以柔不再跟傅夜沉纠结这个话题，而是非常识趣地颔首点头，听从命令。
<br/>
<br/>        她每次都能做到收张有度，不让傅夜沉对她产生厌恶感。
<br/>
<br/>        这也是她唯一隐藏自己的心意，能长久待在他身边的办法。
<br/>
<br/>        连华生被落以柔一个电话叫来VIP病房后，和颜悦色地跟落以柔说好话，让她先回去休息。
<br/>
<br/>        落以柔一走，连华生便条条是道地数落起傅夜沉的不是来：“阿沉，那个童筱颖到底给你灌了什么迷魂汤？把你整个人折磨成这样？我觉得她现在压根就配不上你了，你为什么还一门心思地扎在她身上？”
<br/>
<br/>        “筱颖她人呢？”傅夜沉完全无视掉连华生的话，质问道。
