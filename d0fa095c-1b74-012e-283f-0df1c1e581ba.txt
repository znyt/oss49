　　对于洪荒之中的修炼者来说，时间是最不值钱的东西。十年时间转瞬即逝，瞬间便到了灵神界开启的时间。
<br/>　　“嗡！”
<br/>　　这一日，洪荒之中顿时荡起一层无形的波纹，紧接着，一道神秘的声音在众生心神之中响起，那声音，好似在梵唱，似祝福，似赞颂，似诸天大道之音，顿时让人忘却了所有烦恼。
<br/>　　声音连响九道，之后才缓缓平息。
<br/>　　“轰！”
<br/>　　紧接着，一道轰然爆炸声在众人心神之中炸了开来，一个浩瀚的世界，出现在众人心神之中。
<br/>　　所有人都知道，灵神界，开启了。
<br/>　　众人心中默念一声“进入”，顿时，一道神光扫过全身，一个眩晕，众人便进入到一方幽暗的通道之中，四周迷雾笼罩，让人看不清虚实，四周没有别人，只有一个人孤零零的，只能向前走去。
<br/>　　渐渐的，雾霭越来越少了，渐渐接近一个光明的世界。最终，众人各自踏出幽暗通道，进入一方青色的神殿之中。
<br/>　　神殿很小，只有一个一般屋子般大小，神殿上空，有“接引殿”三个金色潦草大字，让人知道这是一个什么样的地方。
<br/>　　在进入接引殿的瞬间，便有一道消息传入众人脑海之中，那些消息是灵神界的一些简单信息，当然，也只是最基本，最简单的一些信息。
<br/>　　灵神界中，每次出现，都只能从接引殿中出现，每次返回肉身，也是只能从接引殿返回，不能从其他地方返回。否则大战或者遇到生死危机之时，随时可以退出，那就起不到历练的效果了。
<br/>　　在灵神界中，打破前人的记录能够得到奖励。这些记录什么都有，有偏向于速度的，有炼器、炼丹的，有创造功法的，有比拼力量的，有比拼斩杀妖兽的，等等……
<br/>　　在灵神界之中，众人的名字也可以不用自己真实的名字，可以随便起一个，当然，也可以用真名。这也是为了保证个人现实世界之中的隐私。当然，这种隐私也只是暂时的，随着在灵神界之中创出名声，众人迟早会知道的。
<br/>　　灵神界之中，每一重天都有一百个接引殿，它们分布在灵神界的各个地方。
<br/>　　踏出接引殿，一阵白光闪烁，便会出现在灵神界之中。四周的人顿时便增多了许多，一眼望去，看到的全是各种各样的种族，有凶残的妖兽，有魔气逼人的魔兽，有瑞气缭绕的神兽，有相对正常的人类，有头角峥嵘的异族，有煞气惊天的魔头，有缥缈神秘的神仙……各种种族，应有尽有。
<br/>　　“这……这就是灵神界！”有人看着这宛若真实的世界，惊叹道。
<br/>　　“如此真实，真是不可思议！”有人感受了一下自己，那宛若肉身降临般的真实，让人不敢相信。
<br/>　　“轰！”
<br/>　　有人一掌拍出，轰在虚空之中，令虚空一阵波动，一层层涟漪荡漾开来。
<br/>　　“我的实力竟然完全存在，我也可以施展自己的神通！”
<br/>　　在实验一番之后，所有人都激动了。
<br/>　　“灵神界，真是修士的天堂！好真实啊，就连我自己都丝毫感觉不出来！”
<br/>　　“真是好地方，我喜欢！洪荒万族修士们，我来了！”
<br/>　　“哈哈，我看见，我征服！”
<br/>　　“同级之中，我倒是想要看看，我自己处在哪个阶段？”
<br/>　　“……”
<br/>　　所有进入灵神界之中的人，在了解了灵神界的功效后，彻底沸腾了。众人纷纷向着四周散开，既然是试炼之地，自然是要出去试炼啊，在接引殿前，对他们并没有太多吸引力。
<br/>　　有人出去探险了，有人去约战了，有人去猎杀灵神界之中的妖兽去了，有人去寻宝去了……
<br/>　　进入灵神界的生灵很多，在知道灵神界的各种好处后，众人自然都会去做自己的事。毕竟，此刻是灵神界初开，宝物按理来说，此时最多，先去先得，去晚了，都被别人得到了，那时才是欲哭无泪。
<br/>　　就在众人都要离去做各自的事情之时，一个身赤金色道袍，身负一柄凌厉长剑的修士站了出来，他眼神凌厉，喊道：
<br/>　　“吾乃人族武者，阳光彼岸，众所周知，灵神界是一方新的世界，有着许多神秘之处，也有着许多危险在等着我们，一人之力终究有限，吾欲邀请几位道友一同探险，可否有人愿意？”
<br/>　　此处是第五重天金阳天，能进入此天的修士，自然都是金仙修士，武者也就相当于神通王者境。这位自称‘阳光彼岸’的人族武者，看上去还非常年轻，能够以如此年纪便修炼如此境界，显然是天资卓绝，有大气运之辈。
<br/>　　听了阳光彼岸的话，有的人则是若有所思，觉得此话有理，有的人则是冷笑不已，心中不以为意，有人有些犹豫，显然是心动了，也有人对此不屑一顾，转身便走，毫不理会，有人对自己的实力极为自信，选择单干……
<br/>　　两个人，就会有两种心思，数万人，便会有数万种心思，更何况这些人是来自洪荒万族之人，心思不一更是常见，毕竟，非我族类，其心必异！
<br/>　　也有人觉得，我自己一样可以组建团队，为何非要与你组队，受你限制？
<br/>　　在各种各样的心思下，有极少的人来与阳光彼岸组队，出去冒险、探宝。阳光彼岸也毫不在意，大家都是陌生人，之前毫不认识，因为灵神界的原因才聚集在一切，想要一见面就信任对方，那是不可能的，不与他组队也说得过去。
<br/>　　更何况，他也不指望众人能够听他的，他知道那是不可能的。但万事开头难，有了开头，等他在灵神界之中闯出名声来，还怕找不到人吗？
<br/>　　“我们小队起一个名字，叫‘王者’，如何？”
<br/>　　几人面面相觑，你看看我，我看看你，只觉得一脸懵逼，什么鬼？你想的才周到呢，名字都有了，我们哪有想那么多啊。不过，既然别人都说出来了，那就同意吧，不就是个名字而已。
<br/>　　“可以。”
<br/>　　“好，从此以后，你们就是‘王者小队’的成员了，将来我一定要将‘王者’两个字响彻灵神界。”
<br/>　　“呃？……”
<br/>　　众人都不知道该说什么为好了，只好默默无语。
<br/>　　“好了，我们走！”
<br/>　　等了一会儿，见不再有人过来，阳光彼岸便带着招到的几人向着外面走去。
<br/>　　有了阳光彼岸的开头，其他人自然纷纷效仿，组建小队探测灵神界。有名为殷杰的幽冥界修士建立的“轮回”小队，有称号为“梦牵绕了谁的灵魂”的魔族修士，建立了“梦魇”小队，有……
<br/>　　“打打杀杀有什么好的，世上没有宝物解决不了的问题，如果有，那就用更多的宝物！”一个人族胖子突然出现，看着众人出去冒险，他在心里默默道。
<br/>　　“嗯？灵神界中貌似有很多的宝物啊，我是不是该在这灵神界中建立一个交易场所呢，赚钱这种事情，我最喜欢了。嗯，就这么愉快的决定了。”
<br/>　　这个称号为“—”的胖子心中如此想到，脸上顿时露出财迷般的笑容。
<br/>　　“我还是一如既往的聪明啊，哈哈！”
<br/>　　神界初开，各种各样的人都会出现，各种种族势力也会随之建立起来，世界开始热闹起来了。
<br/>　　……
