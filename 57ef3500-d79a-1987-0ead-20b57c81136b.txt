        &lt;h3&gt;第三卷第二百三十六章  江俞轩回北京&lt;/h3&gt;
<br/>
<br/>        西城发生了一系列的变故，让江俞轩感到很愧疚，人的精力毕竟是有限的，他无法兼顾到两边的工作。
<br/>
<br/>        西城房产项目轰轰烈烈，杨总想要和他一起搞高新区的开发，他的三期项目也马上要面临交工，他感觉没有精力顾及陈子昂公司的事情，他生了辞职的念头。
<br/>
<br/>        陪着杨总体检完，他准备去一趟北京，当面向陈子昂辞职，也去看看张倩楠。
<br/>
<br/>        “杨总，我这几天去一趟北京，有什么事情我们电话联系，等我回来我给你拿体检报告去。”
<br/>
<br/>        “行，去看看也好，当散散心，陪着我这糟老头子加班很多天了，高新那边尽快拿出来一个方案。”
<br/>
<br/>        “我已经安排人按照你的要求在做了，航拍已经完成了，只是后期还是必须得找专家勘测一下。”
<br/>
<br/>        “陈子建已经辞职很久了，说实话，西城在这方便我还是很相信他的。”杨东叹了口气。
<br/>
<br/>        “陈子建就是陈子昂的二哥，如果你不放心的话，我找陈子昂请他二哥出山，为你把把关。”
<br/>
<br/>        “我看这个可以有。你最近就找一找陈子建，在你的方案落地之前来看看。”
<br/>
<br/>        “你这个糟老头子总是喜欢将我往坑里带，一不小心就着了你的道。”
<br/>
<br/>        “嘿嘿，你臭小子，有那么好的关系不用，非要我这个糟老头子求着你。这么多年，你也得体谅体谅我这个糟老头子。”
<br/>
<br/>        “行，你说了算，我从北京回来我就亲自去将陈子建给你请过来。”江俞轩突然就想起来手镯之事，这一段时间太忙，竟然给忙忘记了，大哥最近好像也没有提起这件事情了。
<br/>
<br/>        刚好去找陈子建的时候看看陈子昂她老爹，看看是什么样的父母亲教育出来了陈子昂这样的一个人。
<br/>
<br/>        想到陈子昂江俞轩心里多了一丝温柔，也多了一丝愧疚，陈子昂冰雪聪明，大家却当她什么都不知道，一味的以为替她着想，谁知道她才是为了大家着想，不想大家难堪，不想拂了大家的面子，到现在他竟然不懂她了，不知道她想要什么。
<br/>
<br/>        “轩轩啊，人这一生能遇到适合自己的人真的不多，有些人一错过就是一生，在你的心还没有确定之前不要太早下定论，免得以后后悔。娶妻生子很容易，想和自己心爱的人共度一生却不易。”杨总突然就发了感慨。
<br/>
<br/>        “怎么就发起感慨了？”
<br/>
<br/>        “哎，我啊是感慨我自己呢，年轻的时候喜欢一个女生，女生也喜欢我，可是我就是不敢对她说，一直等，等到想说的时候，她已经有了男朋友。”
<br/>
<br/>        “你这现在还惦记这呢？”
<br/>
<br/>        “是啊，心里藏着的那份美好怎么可能忘得掉了，我觉得人到了一定的年龄对过去的事情反而记忆深刻。
<br/>
<br/>        所以啊，轩轩，认真的正视自己的内心，想好自己想要什么，在合适的时间里去做自己想做的事情，不要等到老了时候心里都是遗憾。”杨总最近总是感慨颇多，江俞轩归结于杨总喜欢给他上课。
<br/>
<br/>        不过杨总说的话很有道理。
<br/>
<br/>        “哈哈哈，我似乎还没有什么遗憾的事情，但是我一定会谨记你的教诲。”江俞轩打着哈哈。
<br/>
<br/>        “那就好，你还年轻，以后遇到的事情很多。”杨总切切叮咛。
<br/>
<br/>        “好，我明天去北京，这边的事情我让人和你对接。”
<br/>
<br/>        “你准备去几天？”
<br/>
<br/>        “两三天吧。”江俞轩也不确定他去北京能呆几天，有些事情总归是要面对的。
<br/>
<br/>        张倩楠确实是一个好妻子的人选，既然证都领了，仪式也简单的举行了，他是应该去看看他的妻子，商量一下今后的生活，总不能让他的妻子总是住在娘家。
<br/>
<br/>        他现在的工作在西城，家在广州，之前在北京工作，公司也是有给他分的房子，虽然去了西城但是房子一直都在，他要离开陈子昂的公司，也就没有必要去住公司的房子，他们在北京的产业是有酒店的，也不可能总是住酒店，去北京和张倩楠商量商量看张倩楠喜欢哪一片的地方，张倩楠在北京工作，他们需要在北京安家。
<br/>
<br/>        有些事情已经错过了，那么就错过吧，他不能辜负张倩楠，这几年他们虽然没有在一起，却也是以男女朋友相处。
<br/>
<br/>        江俞轩到了北京，没有直接去找张倩楠，而是回到了房子里，这个房子他住了整整六年，这几年偶尔会回来住，因为有保姆要来经常收拾卫生，陈子昂也就有他房间的钥匙。
<br/>
<br/>        打开房门，家里干干净净，原来养的绿植兴奋的生长着，屋子里暖融融的，江俞轩心里一暖，这可能是陈子昂让保姆收拾的，花好像也是刚刚修剪不久，鱼缸里的金鱼欢快的游来游去，似乎不知道他的主人经常不在家。
<br/>
<br/>        江俞轩脱掉外套，舒服的将身子窝在沙发里，伸开他的大长腿搁在茶几上，这个家里的感觉很不错，虽然他已经离开了好久，但是那种熟悉的感觉还在，靠在沙发上他竟然睡着了，一觉醒来已经是晚上十点钟，江俞轩习惯性的走到冰箱，准备看看冰箱里有什么吃的，打开冰箱他惊呆了，冰箱里面有新鲜的素菜、鸡蛋、牛排、虽然不是很多，但是足够新鲜。
<br/>
<br/>        “这个疯子，我又不回来做饭吃。”江俞轩迅速关掉冰箱的门，退回到沙发上。
<br/>
<br/>        他不知道陈子昂抱着怎样的心理让保姆买这些菜，她是一直希望他留在北京？难道她一直都这样？原来回来的时候都是白天，来去匆匆，都不知道冰箱里是不是会有素菜，要不是今天太晚，有点饿，他都懒得打开冰箱。
<br/>
<br/>        江俞轩的心思有些烦乱，他走到书房，书房的桌子上放着一张纸，纸上龙飞凤舞的写着一些字，他看了一下，是陈子昂的笔迹，应该是陈子昂打电话时记的东西。
<br/>
<br/>        陈子昂到家里来了？
<br/>
<br/>        他急切的拿起电话，给张倩楠打电话，响了很久，电话才被张倩楠接起来。
<br/>
<br/>        “倩楠。”江俞轩像是抓住了一根救命稻草。
<br/>
<br/>        “俞轩。”张倩楠兴趣缺缺。
<br/>
<br/>        “倩楠，你在哪里？我回北京了，我来接你？”
<br/>
<br/>        “我在西城，这边有事情，我和函哥到西城了。”
<br/>
<br/>        “我以为你在北京！”江俞轩声音低沉。
<br/>
<br/>        “我到西城好几天了，因为工作的原因我没有给你说。”
<br/>
<br/>        “没事，那我把北京的事情处理完，我就回西城吧。”
<br/>
<br/>        “行，到西城了联系，我挂了啊。”
<br/>
<br/>        江俞轩看着挂断的电话，他失眠了。
<br/>
<br/>        江俞轩第二天就回到了西城，他没有去找陈子昂，也没有提出辞职。
