　　第707章
<br/>　　相府要比想象中大，赫赫一片庄园，假山回廊，崔相竟丧心病狂引一条活水进来，在庄园内形成一片湖泊。
<br/>　　双方展开白刃战，李乐天一马当先，厮杀在队伍最前方。一国之君尚且如此，何况身后的鱼龙卫。天子都可以不要命，大家自然可以提着脑袋随他拼命。
<br/>　　在如此势头下，李乐天的队伍一直向前推进。但他心里并没有半点喜悦的心情，战斗刚刚开始，目前还不是开心的时候。现在他的困惑是，直到现在自己还没有看见相府的主事人。
<br/>　　相府的家丁护院死士几乎各自为战，这也令李乐天的推进变得容易。但李乐天提着的心却不敢放下去，相府就算反应有些慢，但慢到这种地方仍是令他觉得不可思议。
<br/>　　该不会是发生什么意外了？
<br/>　　真正决定这一局胜负的并不在相府，而是在于那两万城防军。自己平掉相府后，便必须立刻赶到兵营，安抚那里的大军。目前百里胜正带人在大街上阻击城防军，怕他们也撑不了太久时间。
<br/>　　“哇呀呀。”
<br/>　　一个拎着双斧的壮汉从人群中跳了出来，口中怪叫，抡着双斧向李乐天扑来。
<br/>　　因为程大雷的缘故，帝国还颇兴起一股用斧之风。无论是山贼还是军将，但凡有些力气，都喜欢用斧头做战。如此导致帝国用锤棍这样的大将少了很多。
<br/>　　李乐天不是以力气见长，长枪直刺对方面门，然后横甩，将对方逼退。那用斧的狠人，只是稍稍回避，便又重新扑了过来。
<br/>　　此子的确凶悍。
<br/>　　李乐天心中还真知道对方，其人姓柳，名唤柳弄月。名字听上去十分文雅，性情却残暴的很，以杀人为乐。相府多得是这种人，崔相用人唯才是用，也不去计较出身。
<br/>　　铛！
<br/>　　有一用大枪的汉子拦在柳弄月面前，晃了晃脑袋，双臂一伸将对方推开。
<br/>　　“再来！”
<br/>　　“兀那娃娃，毛都没长全就敢在爷爷面前撒野，吃我一斧。”
<br/>　　来人正是马超马孟起，他不是很明白今夜的状况，不过他也不是计较小节的人物。吃他杀得痛快便可，一杆大枪迎上了柳弄月的双斧。
<br/>　　只是三招两式，便稳稳占了上风，瞬间，又有三四个人跃了出来，将他围在当中。
<br/>　　一路杀过来都是如此，马超同时要面对四五人的围攻。可即便如此，他也显得从容不迫，反倒是柳弄月几人被压得死死的，只有招架之力而无还手之功。
<br/>　　“陛下，小心。”
<br/>　　寒芒一闪，斜刺里一柄刀撞过来。田七见机的快，斜着将李乐天撞开，用身体迎上了对方的刀。刀顺着皮甲的缝隙刺入，再往里进，血便涌了出来。
<br/>　　田七咬紧牙关，狠狠攥住对方手腕。李乐天也是一惊，来人出招稳准狠，看来是相府精心培养出来，专门为了暗杀用的。
<br/>　　如果不是田七关键时刻为自己挡了一刀，自己一条命已经要交待在此地。李乐天用长枪直刺，在对方喉咙留下一个血窟窿。而这个时候，田七也已经气绝身亡。
<br/>　　李乐天轻轻摇摇头，连叹口气的时间都没有。今日血战，注定死伤惨重，而自己要做的，本就是在注定的死局中挣扎出一丝生机。
<br/>　　用血和肉挣扎出一丝生机。
<br/>　　形势渐渐对李乐天不利，似乎对方也认定了擒贼先擒王的道理，发现李乐天后，同时七八个人向自己涌来。李乐天身边死的人越来越多，而马超被人缠住，现在也没办法向他援手。
<br/>　　恰在此时，一队人从大门外涌了进来。一身材魁梧的大汉冲到李乐天身边，同样用一杆大枪，立刻将围攻李乐天的几人斩杀。
<br/>　　“好汉子，你叫什么名字？”
<br/>　　“启禀陛下，草民卢俊义。”
<br/>　　“过了今夜，你便不是草民了。”
<br/>　　随之，李乐天的目光落在百里胜身上。百里胜双手一抱拳，此时节也没可能行大礼。
<br/>　　“禀陛下，属下无能，城防军杀过来了。”
<br/>　　果然，凭百里胜这八百人挡不住两万城防军。难道今日自己注定要死在这里么？李乐天深吸一口气，此时节也不可能想太多，必须斩断后路，一往无前。
<br/>　　“与我杀！”
<br/>　　长枪一提，天子威风凛然而起。百里胜干脆答应一声，此刻谁都是将生死置之度外，拼得今夜再说其他。
<br/>　　在百里胜带人过来，尤其是卢俊义出现后，李乐天已经占据当前战场的全部优势。
<br/>　　一个卢俊义一个马孟起，二人都有万夫不敌之勇。不夸张说，二人联手便足可横扫相府。
<br/>　　一路向前冲，一路向前杀，二人厮杀在队伍最前方，互相也有较劲的意思。
<br/>　　李乐天心中暗暗点头，如此雄夫，今日竟都出现了，当真是天不亡我。假若自己手边早有这样的大将，何至于沦落到今日地步。
<br/>　　相府的家丁护院，渐渐都做鸟兽散。而崔家的小姐公子，碰到了就成刀下亡魂。如果这一局李乐天胜了，他们必然是满门抄斩的结局。如果李乐天胜不了，现在自然是多杀一个是一个。
<br/>　　“老贼在哪里，老贼在哪里？”
<br/>　　李乐天提着大枪，口中哇呀呀大叫。杀多少人也不如杀一个崔相管用，但直到现在，崔相还没有露面。李乐天心中激灵一下，老贼该不会逃了吧？
<br/>　　自己倒是疏忽了这一节，老贼一旦逃走，城防军就不可能服从自己，结果只可能是功亏一篑。
<br/>　　不过这也不能怪李乐天，他的兵力不过两千，闯破相府大门已不容易，根本没有富裕的兵力将相府包围起来。
<br/>　　“老贼，你在哪里！”
<br/>　　又怒又惧的李乐天咆哮一声，一脚将面前的房门踹开。刚踏入屋子，他一下就愣住了。
<br/>　　屋中亮着烛火，此外还有一口黑漆漆的棺材。
<br/>　　崔相……已经死了。
<br/>　　李乐天不愧是帝国天子，在这种事上半点也不糊涂。他立刻就明白了，崔相已经去世，相府一直秘不发丧，也是因为担心崔云间压不住局势。
<br/>　　当自己最艰难的时候，敌人也不容易。
<br/>(本章完)
