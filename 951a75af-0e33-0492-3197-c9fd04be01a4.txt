　　星月奴微微一滞，冲钟馗说道：“钟大士，你乃阴间天师，也要与我轩辕山作对吗？”不等钟馗开口，又补充道：“我可打开虚空，放你回去，你我之间就此息战，如何？”
<br/>　　钟馗捻须一笑，“我钟馗纵横三界几十年，从来没临阵脱逃过……今日正要拿你，做我的下酒菜，岂能就这么走了？”
<br/>　　星月奴冷笑，“我不想杀你，只是不想与阴司开战，你当我真就怕你？”
<br/>　　“哦，这个你不用担心，阴司早把你列为头等要犯，就是轩辕上帝来，也保不住你！”当下灌了一口酒，用力吸气，身形暴涨数倍，上去一把抓住青州鼎，双手擒住，操控帅旗，朝与林三生战斗的星月奴的一个分身掷过去。
<br/>　　帅旗瞬间裹住了分身，死死贴住，不断缩紧，任凭她如何挣扎，都无法挣开。
<br/>　　“快动手啊，干看着？”钟馗冲林三生骂道。
<br/>　　林三生回过神来，上前用天地规尺用力击打星月奴分身的脑袋，一声惨叫，有一道魂魄从分身上飞出，被星月奴收了回去，雕像仍然是雕像，等钟馗收了帅旗，雕像也坠落在地上，崩毁了。
<br/>　　“钟馗！！”
<br/>　　星月奴大怒，“你以为你这样就挡得住我的玄功！”
<br/>　　当下接连作法，青州鼎上绿光大放，青铜之力提升了数倍，试图往钟馗身上反噬，钟馗也感到了吃力，青州鼎在他怀抱中剧烈摇动起来。
<br/>　　“你大爷的！”钟馗灌了一口酒，喝了一半，剩下一半喷在青州鼎上，化作一团黑气，绕着青州鼎旋转，让青铜之力始终无法反噬到钟馗身上，也无法从他怀中挣脱。
<br/>　　不愧是道门七绝之一……叶少阳望着钟馗，心中很明白，钟馗这抱着青州鼎的动作看上去简单，实际上却是面对了巨大的压力，也就是他，换成在场任何一个人……包括自己，都扛不住那可怕的青铜之力。
<br/>　　他平时在阴司游荡，极少跟人动手，三界之内都知道他的威名，但已经很久很久没人见过他真正出手，似乎都已经忘了他有多强。
<br/>　　今天，在这一战之中，他终于显出出了自己的实力，强大到夸张……连星月奴也有点不敢相信。
<br/>　　“小天师……你特么发愣什么，擒贼擒王，你去对付她本尊，只有杀了她，这一切才能结束，我帮你抓着这青州鼎，其余的交给你们了……”
<br/>　　“多谢天师！”
<br/>　　叶少阳摸出六张暗金神符，在七星龙泉剑上帖成一排，朝星月奴奔了过去。
<br/>　　星月奴看了他一眼，袖子一扇，摸出了一把赤红色的宝剑，剑锋上的光芒炽热而雄浑，与七星龙泉剑竟不相上下。
<br/>　　传说中的轩辕剑……
<br/>　　叶少阳咧嘴笑起来，他感到自己体内每一滴血都沸腾了起来，充满了对战斗的渴望，还有仇恨。道渊真人的，张无生的，还有那些惨死在法术公会手下的那许多生灵。
<br/>　　“锵——”
<br/>　　两把宝剑不间断地缠斗起来，气势上不相上下，叶少阳一只手持剑，另只手不断从背包里摸出法器，配合进攻，一时间不落下风。
<br/>　　“你已经是上仙牌位了……若是假以时日，必成气候。”
<br/>　　“可惜你看不到那一天了。”
<br/>　　“哈哈！”星月奴大笑，单手结印，将手中轩辕剑往前一送，在空中抖出了万道金光，从上往下刺来，等叶少阳防守的时候才发现，明明是剑光，交缠在一起，却隐隐组成了山岳的形状，仿佛有一座山从头顶压下来。
<br/>　　这大概就是轩辕剑的剑气吧。本来轩辕剑就是五行属土。
<br/>　　“天地无极，应变无明，静水流深，逢凶化吉！太上老君急急如律令！”一边念咒，一边将七星龙泉剑丢出去，在山岳一般的剑光之中流转，犹如一道紫色溪涧，在山岳之中穿梭，水滴石穿，山岳无论多么厚重，总能让水流找到空隙。
<br/>　　两把三界顶级法器，展开了针锋相对的对决。
<br/>　　叶少阳神识的全部，都投入在了七星龙泉剑上，心随意动，巍巍山岳，势不可挡，但不管山用多重、有多么严丝合缝，流动的水总是能够找到出路，一次次被逼到绝境，又一次次从夹缝中绝处逢生。
<br/>　　叶少阳进入了一种物我两忘的境界。
<br/>　　这场险恶的战斗，却让他真正领会到了七星龙泉剑的奥义，这种领悟，是被星月奴的轩辕剑逼出来的。叶少阳很想跟她说声谢谢。
<br/>　　“哦？”
<br/>　　叶少阳的表现，让星月奴也感到了震惊，纵然她的剑势丝毫不乱，但她也明白，靠这个是奈何不了他了，当下冷哼一声，召回来一道分身，前后夹攻他。
<br/>　　叶少阳陡然感到吃力，这时小青小白赶来，清风明月剑合体，三个打两个。
<br/>　　星月奴又召回一道分身。雪琪现出天罗夜叉真身，冲了过来，刚到跟前，还没加入战斗，突然感觉一股庞大的气势横冲过来，转头一看，是四宝观观想出了韦驮天，手持禅杖，气势逼人。
<br/>　　“大和尚！”
<br/>　　雪琪喊了一声，四宝却好像没听见，操控着韦驮天，手中禅杖朝她打过来。
<br/>　　雪琪只好咬牙顶上。
<br/>　　本来是叶少阳和星月奴的一场单挑，现在演变成了群殴。
<br/>　　星月奴哈哈大笑，丝毫不惧，对四宝呵斥道：“速来护驾！”
<br/>　　四宝立刻冲到她面前，作法召唤出金身罗汉，挡在她面前，那几道分身依次挡在边上，布置出了一道“V”形状的结界，四宝就是箭头。
<br/>　　星月奴大袖一挥，打破了身后岛上的封锁，狂狼涌来，托住了四宝等人，形成了一股强大的旋流，倾注到他们布置成的结界上，形成了一道高大的水墙。
<br/>　　叶少阳一干人愣了下，试图打碎结界，但结界被破坏之后，水并没有流出来，而是快速填充，结界也完好无缺。
<br/>　　星月奴退到后面去，飞到空中，一个浪花用来，将她裹在水中，形成了一道水柱，她盘膝坐在中间，四周的海水不断汇聚过来，接触到她身体表面，立刻蒸发成了气体，在她身边环绕。
