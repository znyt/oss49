　　时间眨眼间便已过十个元会，百万余年转瞬即逝！
<br/>　　“嗯！”
<br/>　　狂暴的混沌之气中终于平息下来，旋涡之中出现一个青年男子，身穿青袍，一头黑发飘散，英俊无比，周身散发着混沌的气息，目中沧桑之意闪过，一瞬即逝！
<br/>　　仿佛亘古长存，天地同在！
<br/>　　元始此刻也停止了参悟法则，从元始印中走出来，与青年男子对视，虽然两人为一体，但相貌没有一点相似之处，一个是二十来岁的青年，一个是三十来岁的中年男子，两人站在一起，并没有丝毫的不和。
<br/>　　两人这一对视，不由疯狂大笑了起来。
<br/>　　“见过道友，吾乃混沌道君！”
<br/>　　“见过道友，吾乃元始天尊！”
<br/>　　“哈哈哈！”
<br/>　　“哈哈哈！”
<br/>　　说完，两人再次疯狂大笑。良久，才同时息声。
<br/>　　此时，元始对着混沌道君说道：“你我同为一体，既然你为混沌第一灵根混沌树，那么以后就在混沌修行了。而本尊，则在那洪荒天地之中修行。”
<br/>　　“大善，吾之本体为混沌树，经过这百万年的契合，已同为一体，不分彼此。此世间三种最为古老崇高的至高起源大道，象征灵魂起源之鸿蒙大道，象征能量起源的太初之道，象征物质起源的混沌之道。
<br/>　　本尊想要走聚众道为一道的元始大道，就必须修此三种起源大道。如今，吾既为混沌第一灵根——混沌树，当修这物质起源混沌大道，至于其余两种起源大道，则需本尊想办法了。”
<br/>　　“无妨，吾既然有这鸿蒙至宝元始印，就一定能成功，你也不必担忧。汝只需专心修混沌大道即可。你如今已经已有准圣实力了，你我可通过灵魂上的联系，共同参悟元始印，争取早日证道混元第三步。”元始对着混沌道君说道。
<br/>　　“可！”混沌道君明显话少。
<br/>　　“既然如此，汝就在此修行，本尊先回洪荒了。”元始显然不像混沌道君一般，虽然话不多，但也不少。
<br/>　　“嗯！”混沌道君点头，当下，便不再多言。
<br/>　　元始也不甚在意，当下收起因果莲台，转身直奔洪荒天地而去，而混沌道君却转身向着混沌深处而去。
<br/>　　……
<br/>　　元始返回洪荒之后并没有回东昆仑山谷，而是向南方赶去。因为之前的混沌之行让元始认识到了自己的玄黄不灭真身还不足够的强大，连抵挡混沌之气都十分困难，这可不行。这让元始不得不为自己的肉体真身考虑。
<br/>　　如今元始的《玄黄不灭》已经修炼到了第三重，若想要第三重大成，就必须凑齐四象神兽之力，以四象之力汇聚地水火风，构成天第四极，对应元始的双手双脚，融会贯通之后，元始就可以练成四象真身。
<br/>　　《玄黄不灭》是一部绝世仙法，每一重都对应着一种绝世体质。如：第一种大成之后就是无上玉身，第二重大成之后为五行圣体，第三重大成之后为四象真身，第四重大成之后为真龙体，第五重大成后又为星辰圣体，第六重大成后为阴阳圣体，第七重大成后为虚空体，第八重大成后为混洞源体，第九重大成后才可称为玄黄不灭真圣体。
<br/>　　九种无上体质合一才可证玄黄不灭真圣体，可见此体制的霸道绝伦之处。可想而知，若是修成此体质，不仅在霸道之上能与混沌真体争锋，而且它还有许多混沌真体没有的奥妙，毕竟混沌真体只能修混沌大道，而玄黄不灭真圣体却能修众多大道。
<br/>　　元始如今就是要去天地极南之地寻找四象神兽中的朱雀神兽。四象神兽乃天地之初负责镇守天地四极的神兽，它们分别是：
<br/>　　东方青龙神兽；
<br/>　　南方朱雀神兽；
<br/>　　西方白虎神兽；
<br/>　　北方玄武神兽。
<br/>　　要问元始为什么要先去南方找朱雀神兽，而不是去找其他神兽，只因为元始从混沌之中回来，最靠近极南之地。故而，元始最先找朱雀神兽。
<br/>　　朱雀是四灵之一，为南方七宿的总称。很多人将其认为是凤凰或者凤凰的一种，但实际上这是错误的观点，真正的大神通者，都不会这样认为。朱雀与凤凰有极大的不同，且作为天之四灵与四方星宿之一的朱雀比凤凰要更加尊贵。
<br/>　　一般人想要找到朱雀十分困难，甚至很多人一生都难以见到朱雀。但，这对于元始来说并不算什么。元始是什么人，没有开天之前元始可是执掌因果法则的因果之主啊，因果的力量，用来推算什么的，再好不过了，这比伏羲的先天八卦、周公的后天八卦要厉害多了。
<br/>　　只要还存在天地之中，就会与这天地有因果存在，既然有因果存在，那么元始就能够算到。虽然元始还没有恢复之前的修为，因果法则也因为重新转生的关系被封了，但这对于元始来说并不算什么，以他如今的修为，已经能够解开一部分封印了，虽然这点力量相较于完整的因果之力只是九牛一毛。但，对于眼前的情况来说，已经足够了。
<br/>　　元始将这种用因果法则来推算事物的过去、现在、未来的发展与存在的方法称作《元始金章》。了解要推算事物的信息越多，推算越容易。
<br/>　　因为元始知道朱雀的存在与大致地点，故而，方向已经缩小了许多，所以，推算起来也比较容易，没有费多大法力。元始十指飞快运转，因果之力浮现，玄之又玄。不到半分钟，元始就推算到了朱雀所在之地。
<br/>　　“南岭焚炎谷吗？”元始淡然一笑，不再停留，化作一道清风，眨眼间便已消失不见。
<br/>　　……
<br/>　　“嗯？这就是焚炎谷？”一道身影闪现，停留在一个巨大的火山口附近。
<br/>　　火山附近，又有数十个火山，此处就是一片巨大的火山群，最中间一个巨大火山口，直径数百米，周围数十个火山大大小小将其包围，像是在朝拜最大的一个火山。火山之中，岩浆涌动，方圆数百里，寸草不生。
<br/>　　“嗯？”元始皱了皱眉，一个闪身，来到最大一个火山口前，向下望去。
<br/>　　洞口巨大，山洞极深，洞内漆黑一片，什么也看不见，仿佛一个巨大凶兽张开血盆大口。但元始却在其中感受到一股极为强大隐晦的力量，里面仿佛蛰伏着一个巨大凶兽。
<br/>　　元始没有任何犹豫，开启身体防御，一纵就跳了进去。到了如今他这种实力，已经没有多少令他畏惧的了。
