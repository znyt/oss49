　　第461章
<br/>　　如此，程大雷就糊涂起来。
<br/>　　现在他身处的草原，是并州与幽州边界，而同时，这里也和北蛮部控制的草原接壤。
<br/>　　而程大雷在北蛮部的名声……根本就不是好或不好的问题，假若北蛮部要众筹埋葬程大雷，那么程大雷坟上的土绝对会超过青牛山。
<br/>　　三千戎族来也匆匆，去也匆匆，在浩荡的草原上留下几道烟尘，便消失在远处的地平线上。
<br/>　　程大雷心中的困惑，无人可以回答。
<br/>　　极远处的草原上，一棵枯死的槐树，三千来去如风的戎骑来到槐树下，聚集在这里。
<br/>　　在槐树的枯杈上坐着一个少年，一匹马拴在树根处，他晃悠着双腿，如一个没长大的顽童。但三千凶恶的戎骑停在树下，却都保持着安静，可见他们对这『顽童』的尊重。
<br/>　　在戎族，尊重的原因只有一种，那就是畏惧。而畏惧的原因是，大家承认这少年比他们强。
<br/>　　骑术比他们强，箭法比他们强，杀人……也比他们强。
<br/>　　“王，人已经救出。”黑雁在马上道。
<br/>　　“嗯，我知道啦。”少年拍了拍干枯的树干，回头向远处望了一眼，口中呐呐道：“大当家，恩已清，仇未了，下次见面，就是敌人了。”
<br/>　　灼眼的光芒照在他脸上，本来木讷的一张脸罩着一个黑色眼罩，平白有了些狰狞味道。
<br/>　　……
<br/>　　程大雷在戎骑的帮助下，突破重围，暂时没时间去想心中的疑惑，因为还有更重要的事情要处理。
<br/>　　就算突围，现在也远远称不上安全。稍整以暇，帝国军就会杀过来，自己能利用的，只有这短暂的空隙而已。
<br/>　　程大雷回过头，看着跟着自己突破重围的残兵，拼杀一阵，又是损兵折将。一张张脸上挂满鲜血，麻木而呆滞，就算此刻已突围，但他们还为回过神来。
<br/>　　“好啦，醒醒吧。”程大雷突然喝了一声，冲这些人道：“谁是你们的头目，谁能回答我的问题，站出来一个说话？”
<br/>　　程大雷问完之后，数着呼吸，十息之后，一个人站了出来。
<br/>　　“启禀圣主大人，我是正义教步卒统领白平。”
<br/>　　“这里没什么圣主大人，你们可以叫我大当家，再让我听到这两个字，我先斩了你。第二，你不再是什么步卒统领，你是义字军的营长。”程大雷。
<br/>　　“是。”白平恭敬的回答，更多是畏惧。
<br/>　　“好啦，给你点时间，告诉我现在还有多少人？”
<br/>　　肯在绝地站出来的都不是一般人，白平此刻保持着冷静的理智实属不易，他在很短时间内，便给了程大雷想要的答案。
<br/>　　“启禀大当家，义字军包括我在内，还有三百一十三人。”
<br/>　　三百一十三……程大雷倒抽一口凉气，林少羽提了十万人打长安，打到现在，只剩下这可怜兮兮的三百一十三人。
<br/>　　仗打到这个份儿，真是丢人丢到家了。
<br/>　　不幸中的万幸，生命之神的祝福还在发挥作用，这三百一十三人身上都有伤，但都保持着战斗力。毕竟，那些失去行动力的人，已死在乱军厮杀之中。
<br/>　　“好啦，我简单给你们说几句话，都与我听仔细了。”程大雷清了清嗓子：“你们不是我的队伍，说实话，我并不喜欢你们。现在要走的，走好不送，草原很大，藏身应该不难，你们找地方窝着，我也省点力气。”
<br/>　　程大雷问完等了片刻，没人离开队伍，于是继续道。
<br/>　　“既然没人走，那这支队伍就归我带了。林少羽将你们交给我，我就会带着你们离开这杀戮场，往后走，会很难，还会死很多人，或者你们都会死光。我只能说尽力，但不能说保证，你们都听明白没有？”
<br/>　　三百一十三名残兵同时跪道，白平道：“我们弟兄的命都是大当家的，大当家让我们现在自戳，我们也绝无怨言。”
<br/>　　程大雷点点头，不愧是从尸山血海走出来的人，都知道杀人和被杀是怎么一会事。虽然只有三百一十三人，却有一支队伍的锐气，况且是在刚刚战败的情况下。
<br/>　　“现在，原地休整，开始吃东西。”程大雷道。
<br/>　　“大当家，吃什么？”白平虽然有些忐忑，但还是问了出来。
<br/>　　“这些不都是食物么，都不能吃么？”程大雷指着游散在身周的战马。
<br/>　　戎族这次做事特别地道，还给程大雷留下了几百匹马，白平之所以没有想到，是因为战马对一名军人的意义实在重大，假若问他们战马和老婆掉进水里，要先救那个，他们肯定会选前者。
<br/>　　程大雷可没有这方面的心思，人本就是从野兽中走来的，肚子都填不饱了，装什么大尾巴狼。
<br/>　　而这些起义军，包括程大雷，都太需要食物的补充了。
<br/>　　在程大雷下令之后，白平开始带人宰杀战马，杀了几匹马给大家分食。过程中还颇出现几幕感人的事情，有人捂着眼睛不忍看，有老马临死前落泪不止。
<br/>　　程大雷假装瞧不着，装什么装呀，还真以为自己是圣人了。
<br/>　　趁着这个时间，程大雷检查自己的系统界面，看刚获得的隐藏属性是什么。
<br/>　　嘟，获得隐藏属性：暴恶
<br/>　　嘟，强者之路，必然以暴制暴，以恶制恶，用鲜血和白骨铺成一条属于你的路。
<br/>　　程大雷原先的隐藏属性是『一个好人』，现在变成了『暴恶』很显然是截然相反的隐藏属性。以前系统都称呼自己为正义的少年，得，现在自己不正义了。
<br/>　　“系统精灵，你给我出来，出来。”程大雷低声吼。
<br/>　　『……』
<br/>　　“你给我解释解释，什么叫暴恶。”
<br/>　　『所谓暴恶，简单来说，就是你杀人太多，邪恶值达到一个极限，触发了隐藏属性』
<br/>　　“那么，有什么改变？”程大雷言简意赅，现在没时间和系统纠结。
<br/>　　『确实还是有些改变的，以前已说过，一个好人会帮你将邪恶值清零，失去隐藏属性后，你的邪恶值一直在累积，邪恶值越多，招募到的英雄也就……』
<br/>　　“也就只能招募到坏人么，现在还一样？”
<br/>　　『不，不一样。』
<br/>(本章完)
