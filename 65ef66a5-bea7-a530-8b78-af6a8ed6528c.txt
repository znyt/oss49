　　“这，都行。”
<br/>　　叶少阳心神恍惚地起身，想到昨晚的事，走过去问她：“你昨晚上……”
<br/>　　“好了，都过去了。”
<br/>　　周静茹笑了笑，不再让他说下去。
<br/>　　她又恢复了平时的样子。
<br/>　　这一天，叶少阳去警局录口供，打听到那个女警官刘琪，当初跟他一起被法阵吸进去的，也跟着一起出来了，正在医院里接受检查。
<br/>　　叶少阳把事情经过说清楚，在口供上签了名，想到什么，问道：“这档案不会公开吧。”
<br/>　　“当然，就是入档而已，这件事，没有后续了吧？”
<br/>　　叶少阳想了一下，摇了摇头。
<br/>　　“那就好，不管怎么样，只要能结案就行。”那警官送他离开。
<br/>　　回到酒店，叶少阳跟老秋等几个当地法师见了下，得知龙虎山的人已经都走了，大家坐下吃了个饭，叶少阳跟他的一群小伙伴也离开了。
<br/>　　周静茹帮他们定的机票，因为陈悦没有身份证，比较麻烦，好在有周静茹，通过陈涛把机票搞定了，一行人检票上飞机。
<br/>　　“少阳哥。”叶少阳突然听见周静茹叫自己，回头看去，周静茹站在安检口，双手背在身后，笑吟吟地看着他。
<br/>　　“你怎么不走？”
<br/>　　“我要出国一趟，得一段时间才回去，干脆就从这里飞过去好了。”
<br/>　　叶少阳觉得有点不对劲，但她的理由没法反驳。
<br/>　　周静茹冲他招手，“保重啊，少阳哥。还有大伙。”
<br/>　　大伙都跟她道别。
<br/>　　“你……你注意安全啊。”
<br/>　　“我会的。你们快上去吧。”
<br/>　　大伙开始从升降梯往里走了。
<br/>　　叶少阳还站在入口，工作人员开始催促他。
<br/>　　“少阳哥，你快走吧，我会照顾好自己。”周静茹用力朝他挥手。
<br/>　　叶少阳被工作人员推着往里走，到了转弯口，身后突然响起周静茹的呐喊：“少阳哥！”
<br/>　　叶少阳回头，看了她一眼，最后看到她一眼：穿着牛仔裤和粉色长袖毛衣、运动鞋，头上戴着淡蓝色棒球帽，十分干练休闲，像是要去旅行。
<br/>　　然后他就被推进了机舱。
<br/>　　一丝不详的感觉涌上叶少阳心头，等他来到座位上，立刻拿出手机，给周静茹在微信上发消息，但是没回。
<br/>　　后来飞机起飞，不地不把手机关上，等到了石城，飞机落地，他赶紧打开手机，有周静茹发来的一条微信：
<br/>　　少阳哥，我走了，我去欧洲游学了，可能几年都不会回国，也可能永远不回去。我爱过你，但不能留在你身边了，我要开始新的生活，我会把手机换了，你不必联系我，我会寄明信片给你，祝你一切安好，也许，我会一直爱你下去，随便吧。
<br/>　　忘了我，但不要忘得那么彻底，至少，记得曾经有我这个人。
<br/>　　叶少阳眼睛湿润了，看不清屏幕上的字迹，脑海里反复都是周静茹在机场里冲自己微笑着摆手的那一幕。
<br/>　　我当然不会忘记你，因为，你会一直在我心里啊。
<br/>　　虽然不舍分别，但叶少阳也很欣慰周静茹的选择。
<br/>　　也许随着时间过去，她会慢慢忘记自己，然后遇到一个能跟她携手到老的人。但愿。
<br/>　　谢雨晴在机场迎接，上来就是一个大大的拥抱给他。
<br/>　　刚送走了周静茹，再看到谢雨晴，想到他们是一样的状态，叶少阳有点愧疚，觉得自己终归得做点什么。
<br/>　　他没跟任何人说周静茹一去不归的事。
<br/>　　老郭出来也挺久了，立刻匆匆回家去见媳妇了。叶少阳带着大伙回家，家里一切乱糟糟的，自己卧室的门框什么的还坏了。
<br/>　　谢雨晴把他拉到房间，将那个假叶少阳冒充自己的经过说了一遍，叶少阳之前虽然听秋莹和林三生都说过，但当时不知道细节，如今听到这些细节，感觉匪夷所思。
<br/>　　“那你……你没事吧，他没欺负你吧？”叶少阳问谢雨晴。
<br/>　　“欺负我干什么，他要假装成你啊，为什么要欺负我。”说完突然明白过来，瞪了叶少阳一眼道：“得了吧，就算是真的你，我也不会让你占便宜的！”
<br/>　　叶少阳这才放心。
<br/>　　瓜瓜说道：“老大，你应该庆幸啊，那个冒充你的家伙没有毁你的名声，万一他去到处犯罪，或者当街裸奔什么的，人家还以为那是你……”
<br/>　　谢雨晴扑哧一笑。
<br/>　　叶少阳狠狠瞪了他一眼，“你脑洞真的大！”
<br/>　　在一起吃了午饭，然后谢雨晴把雪琪带走了，陈悦留下，好在叶少阳家里房间多，收拾了一间房出来给她住。
<br/>　　叶少阳修整了一下，傍晚跟老郭联系了一下，本来想找他补充一些法药，正好老郭是在家里接的电话，郭大嫂抢过手机，非要他过去吃饭。
<br/>　　郭大嫂那个热情，叶少阳只好过去。
<br/>　　见面之后，郭大嫂一如既往地亲热，叶少阳倒有点不好意思，毕竟把老郭带走这么久，于是跟她解释，郭大嫂倒是毫不介意。
<br/>　　“只要人好好的，我管他去哪，老夫老妻了，孩子又不在身边，我跟他天天在一起干啥，还得给他做饭。”郭大嫂斜乜了老郭一眼，“少阳我跟你说，我是支持他跟你一起的，总好过他没认识你之前，总往按摩房跑……啊，现在天不去了吧？”
<br/>　　叶少阳连连摇头，转头看老郭，一副如释重负的样子。
<br/>　　在老郭家里吃了饺子，叶少阳跟老郭一起去了店里，配了各种法药，装了满满一背包，满载而归。
<br/>　　老郭送他去打车，路上嘱咐他，“那个，回去万一你嫂子问你，你就说我去你那里帮忙去了……干啥都行，反正就说我在你那儿，记住了啊！”
<br/>　　叶少阳吃惊地看着他，“你实际去哪？”
<br/>　　“去……按摩。”老郭嘿嘿一笑。
<br/>　　“靠，你还真去啊。”
<br/>　　“正规的啊，你想什么呢，不信你跟我一块去！”
<br/>　　叶少阳当然不去，损了他几句，来到路口，等车的时候，老郭问他：“对了小师弟，你到底知不知道山海印在哪？”
<br/>　　“说多少次了，真不知道。”
<br/>　　“你好好想想，既然这么多人都想要，说明真的是个宝贝。”
