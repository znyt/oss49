　　第559章
<br/>　　虽然过程中有些波折，但燕不归最后还是平安离了书院。而且，他还带走了自己想要的华正则。
<br/>　　“杀人啦，杀人啦！”
<br/>　　“有没有人听见，快点过来救命啊！”
<br/>　　一声高一声低，实在搞得燕不归烦不胜烦。他冷喝一声道：“不想死的就闭上嘴，信不信我现在就杀了你。”
<br/>　　华正则立刻将嘴巴闭上，但这种沉默并没有维持太长时间。
<br/>　　“好汉，我与你往日无怨，近日无仇。你好好想想，自己是不是抓错人了。”
<br/>　　“再想想，你还年轻，不要做傻事啊。”
<br/>　　“你不是要我去救人么，不好说什么死不死的，不吉利。”
<br/>　　燕不归紧咬牙关，不断克制自己想要杀死华正则的冲动。他生性沉默寡言，对这种啰嗦的人实在很难产生好感。于是选择闭口不言，不去搭华正则的话，加快脚步向义庄奔去。
<br/>　　华正则可没有闭嘴，仍然一句接一句，连绵不绝的啰嗦着。
<br/>　　虽然有华正则的骚扰，但燕不归的心情总是高兴的。其一是可以救活黑贞的性命，其二是能在三大高手的围攻中脱身，他也足以自豪。
<br/>　　穿过房脊，身子轻飘飘落在义庄院子当中。燕不归立刻就想唤出声，对于他这种性格而言，这倒是很少见的事情。
<br/>　　他立刻闭上嘴巴，警惕的盯着四周。
<br/>　　今夜的义庄，忒安静了些。
<br/>　　燕不归是在草原上长大的，对于危险有种天生的直觉。生活在草原上，当背后有野兽靠近时，周围便会诡异的安静下来。
<br/>　　此时此刻，他又一次觉察到这种安静。
<br/>　　自己留下五个人照顾黑贞，他们不可能一点动静也没有。莫非在自己离去的这段时间，有人潜入义庄，收拾了自己的同伴，然后，让此地变成一座牢笼。
<br/>　　燕不归此刻倒是希望华正则能说些话，可似乎他也觉察到气氛有些不对，竟也闭上了嘴。
<br/>　　只不过一闪念的时间，燕不归决定闯进去看看。是神是鬼，是妖是魔，为了黑贞，自己也只能见神诛神，遇鬼斩鬼。
<br/>　　手提重剑，用剑推开腐朽的木门，门内的情景一览无遗。殿上供奉着狰狞的恶鬼，地上有篝火熄灭后的灰烬，参差摆着几口薄皮棺材。
<br/>　　在一口棺材上，躺着黑贞。
<br/>　　燕不归大踏步走进去，扶住黑贞的肩膀。
<br/>　　那张脸梦里念过无数遍，愿日日夜夜看着，但今天只是看上一眼，他便不敢再看下去。
<br/>　　皮肤呈现一种诡异的黑，如过多的红色调在一起，于是成了这种黑色。
<br/>　　“贞妹，贞妹……”
<br/>　　黑贞虚弱的抬开眼皮，似乎笑了笑，笑得令人有些怕。
<br/>　　“他们都走了……带着书……他们，走了。”
<br/>　　燕不归瞬间明白义庄内发生了什么。自己令柳褐岩等人留下来照顾黑贞，黑贞的性命对自己而言是极其重要的，但对柳褐岩等人来说，还是山水经比较重要些。
<br/>　　于是他们抛弃了黑贞，也同时抛弃了自己，五人带着书逃回戎族草原。
<br/>　　“走了就走了吧，也不太重要。”燕不归冲华正则道：“救人。”
<br/>　　“那个……你是不是先把我放下来。”华正则无奈道。
<br/>　　所谓关心则乱，燕不归一心牵挂着黑贞的伤势，却是闹出了乌龙。他双手扯开华正则身上的束缚，重重道：“救人。”
<br/>　　此时此刻，程大雷身子轻飘飘的落在屋顶上，如一只老猫贴在上面。轻巧的拿掉一片瓦，打量着屋内的动静。
<br/>　　华正则揉了揉发酸的胳膊，抖了抖随身的医囊，道：“病人就是她么，什么情况？”
<br/>　　“你来看。”
<br/>　　燕不归帮黑贞翻了个身，打开她后背的衣服，看到后背上一道黑印。
<br/>　　“啧，追魂棍的打法，柔劲入体，伤及肺腑。张六动的手，这老头一大把年纪，怎么可以对一个女人下这么狠的手。”
<br/>　　“好眼力。”燕不归忍不住赞了一声，对华正则客气许多：“大夫，怎么救？”
<br/>　　“没救啦。”
<br/>　　一句话令燕不归坠入冰窟窿，他抬起头，用虎狼般的目光注视着华正则。
<br/>　　华正则激灵一下，开口觉得嗓子有些发干：“真的没救了，时间拖到现在，淤血已经堵在一起，便是大罗金仙也救不了。”
<br/>　　“不行，你必须救她，一定要救她。”
<br/>　　燕不归声嘶力竭，歇斯底里。
<br/>　　“你这个人，不要太冲动么。我知道，这个人你一定很在乎，但医生治病不治命。现在她这种情况，谁也救不了。趁她现在还有些意识，你有什么话没对她说过的，就抓紧时间说吧。”
<br/>　　“你救不了他，我便杀了你。”燕不归咬牙切齿，不会有人觉得他只是在恐吓华正则。
<br/>　　华正则委实是一激灵，但他仍强鼓起勇气道：“你这个人脑子放清楚些好不好，我只救我能救的人，让我在一个救不活的人身上动刀，我是决计不肯的。”
<br/>　　燕不归的剑压在华正则肩膀上，华正则整个人就是一沉。双腿忍不住颤抖，只要自己出口半个不字，燕不归会立刻杀了自己。
<br/>　　“那个……我是个医者，我有我做的事，也有我不做的事，而你让我做的这件事，恰好是我不会做的。”
<br/>　　燕不归手中的重剑抬起，举在华正则头顶，以这柄剑的重量，哪怕燕不归不用力气，也可以把华正则的身体砍成两半。
<br/>　　程大雷在屋顶上撇了撇嘴，不得已，自己也要救这华正则一救，毕竟，他就是自己要找的华氏一族。
<br/>　　华正则闭上了眼睛，额头一阵阵冒冷汗，脸色变得苍白。他虽然是个大夫，但看上去自己身体不适很好。
<br/>　　程大雷握紧匹夫剑，打算破屋救人。
<br/>　　噗通。
<br/>　　华正则想象中的情景并没有发生，只听得耳边一声响，他下意识睁开眼睛，看到燕不归跪在自己面前。
<br/>　　“先生，戎族狼部大王座下燕不归，愿以毕生尊严，求先生施手救人。”
<br/>　　华正则撇撇嘴，心道：“你的尊严好像不值钱啊”
<br/>　　当然他口中不敢这么说，无奈道：“你这个人还真是，还真是……起来说话，我也只能试一试。”
<br/>(本章完)
