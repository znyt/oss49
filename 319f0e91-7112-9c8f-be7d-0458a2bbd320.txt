　　第434章
<br/>　　自从离开长安城后，林少羽感觉自己就像折断翅膀的飞鸟。从前翱翔九天，直冲云霄，天下之大无处去不得。但现在只能在地上蹦跶，而且蹦跶的速度还不快。
<br/>　　带着十万兵马打长安，当打下长安城后，损兵折将，全部兵马剩下一半。被各路诸侯赶出长安时，又折损一半。一路上遭遇各路兵马的围追堵截，打来打去，手下兵丁死走逃亡，目前还跟着林少羽的，也就只剩下三四千人。
<br/>　　幽州大本营已经被拿下，林少羽再无力拿回来，就只能带着三千残兵，在并州一地，展开与帝国兵的游击战。
<br/>　　当初，林少羽是打穿并州抵达长安脚下的，并州兵对他可以说是恨之入骨。逮着后，便往死里打。双方互相见过几仗，打得很惨，正义军在抛下一些同伴的尸体后，狼狈逃走。
<br/>　　战败，死亡，伤痛，饥饿，寒冷等等在折磨着这支军队，同时，某种对林少羽不信任的情绪在队伍中蔓延。
<br/>　　或许，他并不是什么天命之将，哪怕他是，但很有可能无所不能的蛤蟆大神已经将他抛弃，他已不值得跟随了。
<br/>　　这一日夜里，起义军在并州某个山坳中扎营，幽暗的山地中，一个个黑影躺在地上，珍惜这难得休息的机会。
<br/>　　为避免被帝国军发现，营地中没有生火，诸人只能靠身上单薄的衣物抵挡寒冷。
<br/>　　林少羽坐在一块大石上，两膝上搁着枪，坐在那里闭目养神。
<br/>　　是脚步踏破枯枝的响动声，林少羽睁开眼睛，看到林冲正接近这里。
<br/>　　“少羽。”
<br/>　　林冲从怀里拿出一个麻布包裹，里面是两个馒头，上面沾着一些血迹。
<br/>　　“少羽，你吃些东西吧。”
<br/>　　林少羽摆摆手：“拿下去给伤兵吃吧，他们更需要这个。”
<br/>　　“可你的身体，不吃东西怎么行。况且……”林冲近前一步，小声道：“况且有些人，就算吃了东西，也活不下去了。”
<br/>　　林少羽一怔，推出去的手僵固在空中，竟不知说什么好。队伍缺医少药，有时走在半路上，好好的汉子一头倒在路上，便再也站不起来了。
<br/>　　林冲硬塞给他手中一个馒头，道：“吃吧，你吃了对大家都好。”
<br/>　　林少羽咬了一口，口腔内一股土腥气，像是咀嚼土壤一般难以承受。他愣了半晌，道：“你说，我们是不是做错了什么？”
<br/>　　林冲摇摇头：“老爷的死，总要有个说法，现在我们已经替老爷报了仇，就算立刻死了也是值得的。”
<br/>　　“是呐，就是立刻死了也是值得的。”林少羽叹了口气，道：“可是我们现在还活着。”
<br/>　　“少羽，下一步你有什么打算？”
<br/>　　郭烦人一死，起义军便没了谋士。在如今兵败如山倒，人心惶惶的情况下，能讨论对策的，也就只剩下林冲一个。林冲是沙场一名悍将，但实在没什么帅才。
<br/>　　“现在整个帝国都要杀我们，幽州已经去不得了，整个天下已无我们的容身之地。我们唯一能去的，也就只剩下一个地方。”林少羽的目光望向西北方向：“只剩下凉州了。”
<br/>　　“凉州，琴川关？”
<br/>　　林少羽点点头。
<br/>　　“但想去那里，一路上要穿过很多城池，怕我们已经支撑不了这么远的路。”林冲。
<br/>　　“从帝国境内过去，自然不太容易。”林少羽道：“所以我们突破边关，走草原，直接到琴川关。”
<br/>　　林冲思索片刻：“我们没有粮食了。”
<br/>　　“路上抢戎族的，总可以赶到。”
<br/>　　林冲目光转了一圈，似乎担忧下面的话被外人听见：“少羽，就算能抢到粮食，有很多人也支撑不下去。干脆，将一些伤兵抛下，如此，才有几分赶到琴川关的可能。”
<br/>　　听到林冲的话，林少羽沉默下来，良久，他轻轻的点了点头。
<br/>　　……
<br/>　　而和程大雷预料的一样，帝国一团乱麻，草原上各部落之间也是打得不可开交。
<br/>　　戎族，红石部。
<br/>　　这片草原是属于北蛮部统治的，在整个北蛮部，红石部实在是一个小得不能再小的部落。整个部落能超过马鞭高的男儿，不过才五六十人。
<br/>　　攻击他们的，是戎族烈阳部。
<br/>　　烈阳部在整个北蛮部来讲，其实也谈不上大。但草原上的资源，就只能允许一个部落活下去，所以无论是烈阳部，还是红石部，必须有其中一个死去。
<br/>　　战斗从晨雾初起时开始，骑着烈马的烈阳部闯进了红石部的营地。第一人临死前的惨叫，拉开了这场战斗的序幕，一轮红日从东方探出个脑袋。
<br/>　　男人骑着骏马冲锋，女人握着木棍战斗，丫丫学步的孩童在血泊中哭泣。
<br/>　　红石部首领黑雁，用一根木棍与敌人厮杀。周围抡开了，七八个人难以近前。
<br/>　　但黑雁能保护的也只是自己而已，他看到自己的孩子被拖在马屁股后面，活活勒死，自己的妻女被敌人抢在马背上，发出一声声惨叫。
<br/>　　以有算打无算，烈阳部很快便接近这场战斗的胜利。男人杀死，女人和牛羊一起被掳走，在草原上，一头牛能换两个女人。
<br/>　　黑雁陷入绝望中，当然，他也这样抢过别人，他的妻子就是用这种手段抢来的。
<br/>　　就在这个时候，远远一匹黑马沿着河流向这边过来，在接近战场时停下了马。
<br/>　　刚刚取得胜利的烈阳部首领注意到了他。对方年纪并不大，表情很平静，看上去还很瘦弱。唯一的出奇之处是，一只眼睛瞎了，蒙着一块黑布。
<br/>　　“小子，你是那个部落的？”烈阳部首领烈阳熊问。
<br/>　　“首领，干脆杀了就是，把他的马抢过来，我喜欢他的马。”一个烈阳部汉子道。
<br/>　　独眼少年没有回答他的问题，反而问：“你们是不是愿意，让我当你们的首领？”
<br/>　　烈阳熊一楞，随即一群人哈哈大笑。少年没有反应，就平静的看着对方，似乎在等待对方笑够。
<br/>　　“谁与我去杀了他，那匹马就是他的。”
<br/>　　“嘿，小子，你的脑袋和马都是我的了。”
<br/>　　一名壮汉纵马狂奔，独眼少年微微抬起眼角。
<br/>(本章完)
