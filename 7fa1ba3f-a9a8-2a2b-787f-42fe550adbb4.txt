　　漆黑的天地，几乎没有一丝光亮。沧桑的世界，笼罩着一片无穷的浓雾，寂静而空廖，深邃而幽远，仿佛亘古永存的苍茫。
<br/>　　幽冥界一如既往地黑暗幽深，在这里，死寂成了主题，黑暗是唯一的色彩。
<br/>　　不过，此刻的幽冥界，却是要比元始之前探查过的要广阔许多，显然，幽冥界还在孕育之中，这多出来的天地，便是那新孕育出来的。
<br/>　　此刻，幽冥界最深处，一个高达数丈的巨石屹立其中，巨石之上，原本沉寂的三道道痕此刻在闪闪发光，黑色、红色、白色三种颜色相互替换，相互循环，最后交织在一起，然后一起绽放出幽深苍茫的光彩。
<br/>　　三色光芒交织，照应在这黑暗无边的幽冥界，打破了幽冥界的寂静，使得万古不变的幽冥界多了一丝异样的光彩。
<br/>　　轰隆隆！
<br/>　　也不知过了多久，伴随着轰隆一声，巨石摇摇晃晃，不过片刻时间，光彩大盛，照亮了整个幽冥界。
<br/>　　之后，过了一刻钟的时间，那光彩才缓缓收敛，退回到巨石旁边，氤氲闪烁，明亮不定。“哗啦！”
<br/>　　蓦然，一声轻响，打破了多年的寂静，于那光彩之中，走出一道身影。那是一道二三十岁的青年男子，他身着一声暗红色长袍，乌黑的长发如瀑布般披散到腰边，他的眼眸深邃无比，眼中闪烁着苍茫的感觉，两条眉毛挺翘而起，气势凌人，给人一种不可冒犯的感觉。
<br/>　　在他的眉心处，有一道神纹，那神纹如同一朵小花，花开三朵花瓣，花瓣很细，细得只看得见一条纹路。中间一瓣为鲜红色，直直向上，左边一瓣为黑色，斜向左上方，右边一瓣为白色，斜向右上方。三色神纹在他的眉心之处，没有丝毫的不适，反而给人一种十分妖异的感觉。显得他的气势更加出尘。
<br/>　　只见他此刻轻轻一步踏出，看向幽冥界最深处，在那里，他能够感受到有六道漆黑无边的深邃通道正在孕育，其中一股淡淡的轮回之力在不断涌动，六个通道，仿佛六个巨大的黑洞，让人一不小心便会被吞噬进去。
<br/>　　看着那藏在深处的六道黑洞，他默默不语，沉默良久，那男子轻声道：
<br/>　　“六道轮回，隐匿幽冥；地道不现，轮回不出；
<br/>　　前尘注定，今生不朽，来世缥缈，是名三生！”
<br/>　　“三生永存，是为不朽，自今日起，吾名三生道人！”
<br/>　　显然，这人便是元始的另一道分身，昔日三生石，今日终究化形而出，号称“三生道人”。这比元始预料的还要晚数万年，显然，三生石化形不如想象中那般顺利。
<br/>　　也是，三生石是何等神物，那可是记载着众生前世今生以及来世的神物，若是那般容易，它就不配叫三生石了。
<br/>　　三生道人化形后，并没有出去，而是继续坐在此地，参悟法则。他不愧为天地神物出生，资质逆天，一化形，便有了大罗中期的修为。
<br/>　　三生法则颇为逆天，其中涉及到了时间、空间、命运、因果、轮回、生死、造化等强大无比的法则，若是能够以此证道，也就无愧于三生永存了。
<br/>　　且不管幽冥界之事，毕竟幽冥界还没有真正成型，虽然多了一个三生道人，但它依旧如之前那般运转着，要来的，它迟早会来，不来的，永远不会来。
<br/>　　“混元者，道之先也！混成一体，万物之源，虚空自然之道，超脱天地法门……”
<br/>　　紫霄宫中，道祖鸿钧讲道不过百年时间。此次道祖所讲的内容为成圣后的修行之路，内容颇为晦涩难懂，大多数人都听得糊里糊涂，不懂分毫。其中，也仅有元始能够听懂，他原本就有混元圆满的修为，此刻来听这些内容，尽管大道不同，但他依旧能够借鉴一二。况且，就算听不懂，他还能用元始印记载下来，等以后修为上去了，他再来参悟。
<br/>　　除了元始外，老子、通天也能听懂部分，至于其他的，也就女娲与西方二人略有小得。但他们得到的，还是没有三清多。
<br/>　　道祖此刻所讲的，恐怕就连道祖自己都不太明白，因为此刻是天道在借助鸿钧之口，给未来的圣人铺路。
<br/>　　以元始的底蕴与见识，结合着自己的大道，吸收着道祖所讲内容，他的身体散发着玄奥的光彩，异象纷飞，和紫霄宫中诸多异象相互映衬，仿佛对比一般。而随着时间的流逝，他的修为也在缓缓攀升。
<br/>　　“哎，此番讲道，如此晦涩难明？这让我等如何能悟？”除了元始等未来的圣人外，余下的大神通者，如妖帝帝俊、东皇太一、镇元子等人，却都眉头紧皱，脸色难看，好似死了爹娘一般。
<br/>　　不能听懂，他们心情能好才怪。但现实就是如此残酷，无奈，他们只好借助此处道韵浓厚，进而来修炼自己的神通。
<br/>　　天地悠悠，时间如白驹过隙，忽然而已。讲道的时光，转瞬即逝，如今已过三千年。在这岁月之中，与天道印证的过程之中，元始兼容并蓄，以元始大道，吸收众多大道，他道行大进，已到了亚圣后期极致。
<br/>　　三千年时间，他整个人都仿佛蜕变了，他的气息更加深邃，好似大道无涯。此刻的元始，气息更加飘忽不定了，没人能够猜测出他的真实战力。
<br/>　　“回到昆仑之后，吾便可以着手斩去‘超我’之身，三我尽斩之后，得见真我，那是我便可以借助那一瞬的道悟，顺利踏入证道第三步：混元证天。证道混元之境。而且，以我的积累，不可能只是进步一两重天的境界，那时，我的修为便可以突飞猛进了。”
<br/>　　元始睁开双眼，眼中诸多异象，汇聚成一个圆点，之后消失于虚无之中，体外道韵流转，渐渐收敛。
<br/>　　“三千年已至，紫霄宫三次讲道结束，不久之后，贫道将合道。量劫不出，鸿钧不现！”金钟一响，道祖便停止讲道，他睁开双眼，眼中带着无比淡漠的神色，他淡淡开口。
<br/>　　在道祖鸿钧三次讲道完毕之后，元始能够清晰的感觉到，自身的气运消失了一部分，不仅自己的，就连老子、通天、女娲，帝俊、太一等紫霄宫中客全部修士的气运都消失了一部分，而那消失的部分，全部汇聚到鸿钧身上。
<br/>　　元始眼睛微微一凝，心神微惊，之后身体才缓缓放松。他发现自己有着先天至宝混沌钟与十二品净世白莲的镇压，当然，最重要的是，他有鸿蒙元始印的镇压，使得他的气运流逝最少。若不是他神念强大，仔细观察，他还发现不了这个问题。至于其他人就没有那么好的运气了，毕竟不是每一个人都会有镇压气运的灵宝。
<br/>　　望着高台之上的道祖鸿钧，他若有所思，这一刻他仿佛明白了道祖讲道的真正目的了。
