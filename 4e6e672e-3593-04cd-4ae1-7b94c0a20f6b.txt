陆茵茵和陆子羡坐在了沙发上，陆茵茵笑道，“妈，寒霆，这一次我飞来海城其实也是为了子羡而来的，子羡已经到了适婚的年纪了，他心高气傲，看不上什么姑娘，嫂子挺急的，我认识一个天才医学少女，正好借用两天后的聚餐将那个天才少女介绍给子羡，我觉得他们十分的般配！”
<br/>
<br/>        陆老夫人拿走了陆子羡和夏夕绾的订婚信物，完美的策划了一场替嫁，让夏夕绾嫁给了自己偏爱的孙子陆寒霆，这件事已经兜不住了。
<br/>
<br/>        现在乍然听闻陆子羡要去相亲了，她双眼一亮，开心道，“那真是太好了，茵茵你的眼光很高，看女孩子一定不会看走眼的，那个天才医学少女一定很适合子羡！”
<br/>
<br/>        “妈，那个天才医学少女相当优秀，是一个天之骄女，她十三岁就被保送帝都一等医科学府，十五岁拿下了双硕士学位，刷新了整个医学界的记录，她毕业的那一年我曾经亲自邀请她来圣黎院留学，但是她说离家太远就给拒绝了，这个天才少女性情聪慧灵动，如果妈见了一定也会觉得惊艳喜欢的。”
<br/>
<br/>        陆老夫人听了不禁啧啧了两声，“听着确实惊艳，跟子羡很般配，相亲聚餐我就不去了，寒霆，你代奶奶去看一看，看一看究竟是何等风流的人物。”
<br/>
<br/>        陆老夫人其实才不管什么风流人物呢，只要大家不来抢她的绾绾孙媳妇就行了。
<br/>
<br/>        陆老夫人从来不觉得自己是小偷，订婚信物本来就是当年夏夕绾的妈咪林水瑶和陆寒霆的妈咪柳璎珞指下的，夏夕绾本来就是陆寒霆的小新娘。
<br/>
<br/>        但是陆老夫人还是有点怕，要是被大家知道夏夕绾就是林水瑶的女儿，不知道多少人要从帝都杀到海城来，那这天下就不太平了。
<br/>
<br/>        夏夕绾的身世，必须要守住。
<br/>
<br/>        至于陆子羡当然也是她的孙子，老夫人知道自己的女儿眼光甚高的，能让茵茵这样赞不绝口的天才少女必然惊艳，配给子羡就挺好的。

<br/>        这样各过各的日子，千万不要有什么交集。
<br/>
<br/>        陆寒霆坐在老夫人的身边，他点头，“好，奶奶，两天后姑姑的聚餐我会抽空去看看的，到时回来告诉你。”
<br/>
<br/>        “甚好！”
<br/>
<br/>        这时陆寒霆低眸看了一下腕上名贵的钢表，“姑姑，子羡，你们在这里用餐吧，我就先走了。”
<br/>
<br/>        陆茵茵道，“寒霆，这么晚了你去哪里？”
<br/>
<br/>        陆寒霆勾起薄唇，英俊的眉眼里透出柔软，“我要去接我的陆太太了，她在等我。”

<br/>        陆子羡全程都没有说话，好像那场相亲宴都跟他没有关系，不过当陆寒霆提到“陆太太”时，他抬了眸，看向陆寒霆，自然也看到了陆寒霆因为陆太太而柔软了的眉眼。

<br/>        陆寒霆掀了掀薄唇，“恩，她刚从乡下回来，没什么拿得出手的学历，也没有见过什么世面。”
<br/>
<br/>        陆老夫人手一挥，“还胆小，怕生，茵茵子羡，你们不要惦记，不见也罢！”
