　　“叶少阳！”
<br/>　　有人惊叫起来。
<br/>　　所有人都呆住了，包括星月奴，一时间全场鸦雀无声，都怔怔地望着这个天外来客般的人。
<br/>　　“叶师兄，你终于来了！”徐波倒在地上，心中说不出的激动，挣扎着爬起来，朝叶少阳跑过去。
<br/>　　叶少阳将掌门大印放在他手中，道：“你拿好了！”
<br/>　　“那山之精……”
<br/>　　“搬山咒已经被我们破了！放心。”
<br/>　　曲波更加激动，泪水蒙住了双眼。
<br/>　　龙虎山弟子们都互相望着，眼中都露出了欣喜的神色。
<br/>　　本来已经都绝望了，但叶少阳的一席话，将他们从深渊里又拉了回来。
<br/>　　“叶少阳！”
<br/>　　星月奴咬牙切齿，“你怎么可能会来！”
<br/>　　“你以为，我会在茅山死等你是吧，本来是这样的。”叶少阳一眼看到躺在地上的释信无等人的尸体，立刻走过去，一个个望过去，最后走上前扶起了静慧师太的尸体，不由悲从中来，喃喃道：“我来晚了，师太……”
<br/>　　他们晚上就赶到了山区，但山下有司命星君镇守，天璇八荒星辰大阵有他和彩云仙子等人加持，实在威力巨大。
<br/>　　幸亏道风和李浩然两个证道强者一起出手，才勉强将大阵轰开了一个缺口，他们这才闯进来。
<br/>　　轩辕山这边，都以为大阵万无一失，因此没有提防，他们一路上山，然后悄无声息地爬上了大殿前方的门楼，这才看到里头凄惨的景象，还好，赶在了最后一刻。
<br/>　　“师叔！”
<br/>　　四宝也从侧门冲进来，扑到在逍遥飞的尸体上，无限悲伤。
<br/>　　捉鬼联盟其余成员，逐渐都进来了，静默地在叶少阳身后站成了一排。
<br/>　　“师叔……”吴嘉伟在清长风身后那一群人间法师中，看到了自己的师叔、崂山掌教紫雷子，立刻明白了什么，十分为难地叫了一声。
<br/>　　紫雷子老脸一红，转过脸去，道：“伟儿，为师身不由己，你去做你该做的，勿以为念。”
<br/>　　“叶少阳，道风何在！”清长风冲叶少阳喝道。
<br/>　　叶少阳不理他，抱起了静慧师太的尸体，放在墙角，让她靠在墙上，曲波等人也跟着把那些惨死的法师都抱着放在了一边，靠着墙，让他们看上去比陈尸院中好看一些。
<br/>　　然后，所有人都站到了叶少阳身后，虎视眈眈地望着星月奴三人。
<br/>　　“叶少阳，我是没想到你能来，本来想着以后再收拾你，你既然上门来，那也别怪了。”当下拍了拍手，大声说道：“诸位都现身吧，山下不用守了。”
<br/>　　一个身穿紫色长袍、头戴斗笠的人出现在她身后的围墙上，抱着双臂，冷冷地望着众人，有些不耐烦地说道：“都是一群人类啊，何须费神。哦，九尾天狐么，倒是有点意思……”
<br/>　　“教主，这群人可不容小觑。”星月奴提醒道。
<br/>　　教主？他就是太平教主？
<br/>　　叶少阳心中微怔，果然，轩辕山也都出动了啊。
<br/>　　接着一白一黑两道人影，也落在围墙上，正是白泽和毕方。
<br/>　　岳恒一眼看到毕方，立刻伸手指着他道：“火鸟你过来，咱俩一决高下！”
<br/>　　毕方轻蔑地看了他一眼，没理会。
<br/>　　一个长裙翩翩的仙子，也落在身侧，正是之前稍微出手过一次的彩云仙子，周身祥云缭绕，也是个绝世强者。
<br/>　　她的目光在人群中寻找着，落在杨宫梓身上，道：“你是混沌天魔，不知道你混沌修成了第几重，我俩手段相似，倒可一战！”
<br/>　　“太平教主，我来会你！”
<br/>　　南宫影飞上另一边的围墙，与太平教主隔空相望。
<br/>　　道风不在，司命星君也不在，他们正在山下进行着一场绝代强者之战。道风也没办法，上山去打的确出风头，但没办法，山下还有个最强之人司命星君守着，只能自己扛起来。
<br/>　　轩辕山七大长老，除了死掉的影魅，其余六个都来了，五个在山上，星月奴是证道强者，实力非凡，太平教主为七大长老中排行第二，实力不亚于她，其余几人也都是绝世强者，再加上白泽和毕方，这两人原本就是上古异兽，又被轩辕上帝收归门下，数千年的苦修，实力早已远超大多数异兽。
<br/>　　这八个人，虽然人数上不如叶少阳那边，但都是一流强者，清长风菜一点，也是准一流，随便一个，放在法术界都差不多能横扫天下。
<br/>　　要对付这八个人，叶少阳心中没底，但事已至此，也没什么好说的，生死之战。
<br/>　　“叶少阳，我就知道，想要一统人间，你这道坎是一定要迈过去的。”星月奴气定神闲地望着叶少阳说道。
<br/>　　“就怕你腿不够长！”
<br/>　　叶少阳回看了大伙一眼，低声道：“大家都当心了，都得活下来！”
<br/>　　而后，拔出七星龙泉剑，剑指星月奴一干人，狠狠地吐出四个字：“一个不留！”
<br/>　　星月奴哈哈大笑。
<br/>　　“一起上，剁了她给道渊报仇！”
<br/>　　“为死去的同仁报仇！”
<br/>　　大家一拥而上。
<br/>　　叶少阳剑锋一转，却朝清长风扑了过去，清长风结印作法，上前应战。
<br/>　　本来两人实力差不多，半天也分不出胜负，清长风已经做好了持久战的准备，见叶少阳一见刺来，立刻作法反击，本是一个回合的试探，但是令清长风没想到的是，叶少阳居然丝毫不躲闪，反而加快了脚步，当下一怔，自己手中的判官笔也扫向叶少阳的肋下。
<br/>　　哐当！
<br/>　　叶少阳身后突然闪出一人，挡在了叶少阳身前，竟然是一尊手持禅杖的巨佛，禅杖横扫在判官笔上，阻了一下，被叶少阳躲过。
<br/>　　叶少阳手中的七星龙泉剑却紫气飞腾，刺中了他胸口。
<br/>　　“天地无极，北斗降临！”
<br/>　　清长风丢开判官笔，左手结印，打出七枚铁离子，排列成比斗七星的位置，大放异彩，宛如一道璀璨的银河，黏住了叶少阳手中的长剑，趁着这一滞的机会，人踉跄后退，离开了七星龙泉剑的攻击范围，伸手捂着肋下，诧异地看过去。
<br/>　　四宝站在叶少阳身前，冲他冷笑着。
<br/>(本章完)
