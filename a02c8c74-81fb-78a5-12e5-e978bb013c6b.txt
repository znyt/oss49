        &lt;h3&gt;第三卷第三百零一章  媒婆&lt;/h3&gt;
<br/>
<br/>        陈子昂回到西城的这一段时间，理清了很多的事情，也布局了很多的事情，她现在每走一步都得小心翼翼，无论是工作上的还是她个人的，人生之艰难，可能就从她少年之时就已经开始了。
<br/>
<br/>        如果说她甘于命运，她可能会嫁给泉城里某一户人家，安安宁宁的做一个小妇人，也就不会有现在的这么多事情，可是她偏偏向往长空，向往那波涛汹涌的大海。
<br/>
<br/>        人生有很多选择，向左向右的路途都大不相同，陈子昂义无反顾踏上她自己的征途，不后悔不回头。
<br/>
<br/>        她没有时间去想许弋澄的事情，她将许弋澄交给易倾秋之后就去了医院，她想再陪父亲一晚上，这次回北京之后她就要飞甸城了，北京还有很多事情没有处理，孩子还没有安顿，压在她心里的依然是一块沉甸甸的大石头。
<br/>
<br/>        西城基本没有什么可令她担心的了，有江俞轩坐阵，她还是很安心的，在和江俞轩共事那么久，江俞轩是一个什么样的人她是最清楚不过的，再加上易倾秋和陈子霖，西城应该是稳固的，易倾秋虽然年轻，但是看待问题很透彻，双商很高。
<br/>
<br/>        高新的项目将会拖上江俞轩几年，这几年只要西城不丢市场还是能做下的，她临走会叮咛李秀和谭九儿将耗材市场要牢牢的把控在自己的手中，最起码要在这几年里稳居第一，至于后期市场发展规律在眼前放着，谁也不能垄断整个市场。
<br/>
<br/>        西城因为有陈子霖也因为西城是陈子昂的家乡，所以陈子昂耗费了大部分精力在西城，再加上别的分公司没有那么多的牵绊，一直都顺风顺水，如果仅仅只是商业上的相互竞争，可能还能感觉轻松一些，现在的陈子昂不仅仅是商业上的竞争，而是她感觉到了危险，一种前所未有的的危机。
<br/>
<br/>        陈子昂到医院的时候，发现多了一个人，仔细一瞧是二秀。
<br/>
<br/>        “二秀姐，你怎么来了？”陈子昂对二秀打着招呼。
<br/>
<br/>        “我听说叔叔做手术，我过来看看，顺便把你二哥的笔记本给他送过来。”
<br/>
<br/>        “二哥也真是，大老远的还让你把笔记本拿过来，在公司拿一台不就行了。”陈子昂笑话陈子建节俭。
<br/>
<br/>        “不是，你二哥笔记本里的资料都没有了，我感觉很蹊跷，我就把笔记本给他拿过来了。”二秀解释着。
<br/>
<br/>        “笔记本坏了吗？我看看？”陈子昂是搞信息化出身的，而且还是隐藏着的高手，修笔记本这等小事情对她来说就是小菜一碟。
<br/>
<br/>        “不是坏了，是里面的资料全部被清除了。”陈子建叹了口气。
<br/>
<br/>        “你也真是不小心，怎么能将资料全部清除了呢？”陈子昂等着二秀拿笔记本出来。
<br/>
<br/>        “哪里是他不小心啊，他走的时候笔记本就没有关机，然后一直到现在都没有回家，是我发现他的资料没有了的。”二秀将笔记本递给陈子昂。
<br/>
<br/>        “有人动了你的东西？”陈子昂问陈子建。
<br/>
<br/>        “我也不知道，我让二秀回去帮我把电脑关机了，二秀说里面什么资料都没有了。”陈子建解释着。
<br/>
<br/>        陈子昂看了一眼陈子建又看了一眼二秀，能动笔记本俩个人的关系很不一般啊。
<br/>
<br/>        “我看看吧。”陈子昂将电脑打开，陈子建电脑里每个盘符都在，不过是def三个盘符里面的资料没有了，明显是有人故意删除了的。
<br/>
<br/>        “资料很重要吗？”陈子昂看着陈子建问。
<br/>
<br/>        “我所有的东西都在这个电脑上，如今没有了，连备份也没有，况且还有我最近写的论文，几十万字。”陈子建懊恼的说。
<br/>
<br/>        有可能是冯睿，二秀知道他电脑密码，二秀是不会删除的，冯睿也能知道他的电脑密码，因为密码就是冯睿的生日，但是这话陈子建又不好说出口。
<br/>
<br/>        “没事，我来帮你，有些数据误删除了，是可以恢复的，不过需要几个小时。我先陪爸妈说说话，晚上给你折腾，保证你明天早上就能拿到数据。”陈子昂看了一下，陈子建的硬盘痕迹还在，并没有完全格式化，所以资料是可以恢复的，这在她来说就是小菜一碟。
<br/>
<br/>        陈子建和二秀都是心里一松，大喜。
<br/>
<br/>        “子昂，你也忒厉害了吧，这你都能行？”二秀看着陈子昂像发现了新大陆。
<br/>
<br/>        “我吃饭的家伙，哈哈哈。”
<br/>
<br/>        陈元良和冉灵卉看着三个人在病房里嘻嘻哈哈，心里是满满的幸福和满足。
<br/>
<br/>        “子昂，你就给你二哥修笔记本去吧，我们和你二哥和二秀说说话，你也别折腾的太晚了。”冉灵卉赶陈子昂离开病房。
<br/>
<br/>        “不碍事的，这要不了几个小时，就看二哥的文件有多大，要是文件小的话一个多小时就搞定了。”
<br/>
<br/>        因为有二秀在，陈子昂有些话也不方便说，大家就在病房里家长里短的聊起来了。
<br/>
<br/>        陈子昂这几天也到处跑着，她想起了陈伟冯和陈天天，便问陈子建：“伟冯和天天没有来看爸？”
<br/>
<br/>        “伟冯说最近事情特别多，太忙了，等闲了来看，天天说她妈妈这几天感冒严重了，也是过几天来看。”陈子建解释着。
<br/>
<br/>        “哦。”陈子昂并没有说什么，看来陈伟冯和陈天天已经是和冯睿完全站在了一起，连爷爷做手术都不愿意来看了。
<br/>
<br/>        陈子昂心里有点难受，但是她嘴上说：“如果二嫂感冒的话，还是让天天不要来了，别将病毒带到这里来了，爸手术后身体虚弱。”
<br/>
<br/>        “我也是这样说的。”陈子建微微一笑看着陈元良说。
<br/>
<br/>        陈天天是陈元良和冉灵卉带大的。
<br/>
<br/>        “爸，我明天就要回北京了，你一定要快快的好起来，要记得我们俩人说的话！”陈子昂看着陈元良。
<br/>
<br/>        “记得，记得，你也要早点回来，我等着呢。”陈元良侧着身子看着陈子昂，眼里是希望。
<br/>
<br/>        无论如何，我们都应该有希望，对生命的希望，对信仰的希望，对我们相爱的人也应该充满希望，希望一切都是美好的。
<br/>
<br/>        陈子昂看着时间不早了，她拿起笔记本说：“二哥，我先回去了，明天让七哥将你的笔记本带过来给你，爸妈就辛苦你照顾了。等爸爸好起来了就娶了我二秀姐吧。”
<br/>
<br/>        陈子昂是看出来了，二秀看陈子建的眼神藏着满满的爱意，而陈子建却躲躲闪闪的，不如她挑开了说出来，反正冯睿和陈子建是没有任何可能了，她已经将冯睿加入黑名单了。
<br/>
<br/>        “子昂，你看你说的什么话，人家二秀......”
<br/>
<br/>        “二秀姐，你看我二哥还羞涩的不行，他是个闷骚型的人，你如果想让他主动，怕是得等一辈子了。”陈子昂笑眯眯的看着二秀，她都没有想到她会给陈子建当一回红娘。
<br/>
<br/>        “你这丫头调皮的很，只要你二哥愿意娶我就愿意嫁。”二秀性子火爆，要不然也不会在家里当着冯睿的面拉着陈子建走。
<br/>
<br/>        “二秀......”陈子建老脸一红，他那里想到二秀会这么想，二秀风风火火，对人坦诚热情，他是有些喜欢，但是二秀比他小。
<br/>
<br/>        “嗯，你就说你愿不愿意。”二秀倒也不藏着掖着，她和陈子建知根知底的，俩个人一个离婚了，一个没了男人，有何不可？
<br/>
<br/>        “怎么可能不愿意呢，就怕委屈了你。”冉灵卉开口了。
<br/>
<br/>        “是啊，我怕委屈你了，我比你大。”陈子建期期艾艾的说。
<br/>
<br/>        “我又不嫌你老，你怕什么？”二秀白了一眼陈子建。
<br/>
<br/>        “哈哈哈，成，等我出院就给你们把礼办了。”陈元良高兴的大笑，怎么感觉他这一次生病还赚了很多，妹妹找到了，还顺便将儿媳妇也找到了。
<br/>
<br/>        陈子昂微微一笑对陈子建说:“你们举行婚礼的时候，我在西城送一套房子给你们。笔记本我拿走了，以后记得不要让别人动你的东西了，是你的就要看好。”
