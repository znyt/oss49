　　“少阳哥！”周静茹一干人在远处看到他们出来，立刻大声叫起来。
<br/>　　叶少阳对他们摆了摆手，示意他们不要过来，然后跟吴嘉伟一人把守在门的一边，静静地等着。
<br/>　　“不会从墙壁跑吧？”吴嘉伟不放心。
<br/>　　“我在墙上都贴了血精符，要是他平时，十张灵符也拦不住他，但他现在只能从这里出来了。”
<br/>　　等了一会，黑烟散去，里面却半天没动静。
<br/>　　“别躲了，出来吧。”叶少阳冲里面喊。
<br/>　　又过了一下，叶少阳意识到不对，朝着门外打出一张灵符，贴在了什么东西上，一股紫气立刻荡开，定睛看去，是一个人的形状，通体都是透明的，如果不是有紫气在身上镶了一层边，肉眼根本看不见他的存在。
<br/>　　这人形本来是想逃走的，被灵符贴住之后，身体晃了晃，被定在原地不能动弹。
<br/>　　“影魅的魂魂？”叶少阳走到他面前去，歪头打量着。
<br/>　　“叶少阳……”是影魅的声音。
<br/>　　“还真是你啊。得亏我觉察到你的气息，不然还真让你偷偷跑了。”叶少阳这才对老郭他们招手，所有人都过来了。
<br/>　　“少阳哥你们没事吧？”周静茹第一时间还是关心他的安全。
<br/>　　叶少阳表示没事，冲影魅说道：“有什么好说的吗？”
<br/>　　影魅沉默良久，道：“你知道我只能从门出来？”
<br/>　　“你灵身已经毁了，你是邪灵又不是鬼，你魂魂跟一般鬼魂一样，怎么破我的血精符？”
<br/>　　影魔听了，半天又不作声了。
<br/>　　“叶少阳，这一阵……算我输了，你打算如何处置？”
<br/>　　叶少阳歪着头说道：“暂时还没想好，不然你提议下？”
<br/>　　“你放我离开，我回山之后，总有重谢，我能保证，星月奴不会再找你麻烦。你若不信，我可立誓。”
<br/>　　叶少阳笑笑，“你是不是觉得，我很怕他？”
<br/>　　“她而今大概已证了大道，杀你易如反掌。”
<br/>　　“这样啊，那就让他来吧。”
<br/>　　叶少阳表情严肃起来，道：“你知道我为什么要跟你说这么多？”
<br/>　　影魅当然不知道。
<br/>　　“你只需要回答我一个问题，道渊真人，是不是你杀的？”
<br/>　　影魅不作声。
<br/>　　叶少阳冷笑：“影魅，你也是轩辕山七大长老之一，本来地位还在星月奴上面吧，你敢做不敢认吗？”
<br/>　　“……是我！”影魅咬牙说道。
<br/>　　“好，你现在可以去死了！”
<br/>　　影魅大惊，“叶少阳，你敢杀我？我是轩辕山长老、轩辕大帝嫡传弟子，杀了我，你与轩辕山就彻底无法和解了！”
<br/>　　叶少阳望着他，几乎一字一顿地说道：“我从来，就没想过跟你们和解！这一剑，为了道渊！”
<br/>　　锵的一声，七星龙泉剑拔在手中，一剑刺进了影魅的胸前。
<br/>　　七星龙泉剑的灵力迅速在影魅身上蔓延开来。
<br/>　　他现在是魂魄状态，毫无反击之力，几秒钟中，他魂魄雾化，彻底消失在空气中，精魄飞舞起来，叶少阳用一道灵符收起来，念了一边咒语，灵符丢出去之后，往北而飞，不一会消失了。
<br/>　　要不了多久，这张承载着他精魄的灵符将出现在天子殿崔府君的案头。
<br/>　　这么做，是为了防止他的精魄飞回轩辕山，万一轩辕大帝或别的什么人，有办法给他聚魂重生，自己白辛苦一场。
<br/>　　要做，就做绝。
<br/>　　叶少阳深吸一口气，抬头望天，在心中默默说道：道渊老兄，我总算为你报仇了，不过只报了一半，还有星月奴，你放心。我一定会杀了她！
<br/>　　与影魅这一战，打得十分舒服，算是扬眉吐气了，叶少阳很开心，吴嘉伟也很开心，他之前试验了那么久的剑术总算用在了实践中——之前他一直没动，是叶少阳在神识中提醒他等待时机，然后只一剑，就杀了轩辕山的一个长老，吴嘉伟心里别提有多爽了，更加坚定了自己的修行方向。
<br/>　　下山路上，大家心情都很轻快，山下，很多警察在等着，老郭去跟他们交涉，周静茹早就安排了几辆车等着，把叶少阳一行人舒舒服服地接到酒店休息。
<br/>　　躺在席梦思床上的感觉，真的很爽。
<br/>　　叶少阳很快就睡着了。
<br/>　　再醒来的时候，看外面天都黑了，看身边有微光，转头一看，当时就愣住了，惊道：“碧清你回来拉！”
<br/>　　碧清躺在他身边，捧着一个平板，正在津津有味地看电视剧，头也不抬地对他撇了撇嘴，“不想我回来啊。”
<br/>　　叶少阳嘿嘿一笑，“你误会了，我之前说话是有点重，不过意思不是赶你走，是想让你以后不要再邪修了，我家美华以前也害过人，认识我之后从良了，现在一样是兄弟啊，啊不对，兄妹……好像也不对，她比我大。”
<br/>　　碧清扑哧笑道：“好啦，我走了不久就明白过来了，只是人家怎么也是姑娘，不好意思主动回来，还算你有连心，知道派人去找我。”
<br/>　　“咱们相处时间不长，但一起出身入死的，说实话，我真的不舍得你走。”
<br/>　　碧清靠在被子上，瞥了他一眼，默默说道：“好吧，我不走，往后，我也正式加入你们捉鬼联盟了，但你别想让我献出魂晶，少阳，我至少名分上是老君弟子，我不能认主，不然会折煞你。”
<br/>　　“正合我意！我是实在不想收门人了，够多了！”
<br/>　　碧清冲他摆摆手，“行了，本宫要继续看剧，你跪安吧。”
<br/>　　叶少阳无语。“对了，瓜瓜他们呢？”
<br/>　　“别人都去休息了吧，瓜瓜在你隔壁，跟他娘一起呢，还有，那个周小姐在你左边隔壁，一个人，你回头可以去找她。”
<br/>　　叶少阳起身穿衣服，来到走廊上，想了一下，先去敲了瓜瓜的房门。
<br/>　　瓜瓜过来开门，回身说道：“娘，是老大来了。”
<br/>　　叶少阳一头黑线。
<br/>　　叶少阳进屋，看到陈悦坐在床上，正盯着电视屏幕发呆。
<br/>　　之前从古墓里出来，瓜瓜一直陪着陈悦，叶少阳当时很忙，也没怎么管她，后来瓜瓜去追碧清，把她交给雪琪照顾，一直跟到酒店来了。
