　　　　陆雪琪虽是女流之辈，但实力却丝毫不逊色于林惊羽，且犹有过之，擂台上，两人交战数百招，林惊羽白衫上已经出现几条血痕。
<br/>　　则陆雪琪依旧仙气凛然，冷若冰霜的脸颊上浮现两朵红晕，美得不可方物，显然，斗法消耗了不少体力，相比伤痕累累的林惊羽，孰强孰弱一目了然。
<br/>　　望着台上的冰山美人，慕清霄苦涩的摇了摇头。
<br/>　　虽然他已经俘获陆雪琪的芳心，但后者不是普通女子，刚烈倔强，至情至性，有善良温柔的一面，也有狠绝果断的一面，若是知晓自己拥有数名妻子，不知会不会决然离去。
<br/>　　嘴角牵起一丝自信的笑意，慕清霄将复杂的情绪抛之脑后，若是自己的女人都不能搞定，谈何主宰证道！
<br/>　　平淡的日子确实能够消磨掉人的血性，在青云门修炼的这些年，他心中隐隐觉得，有倾城佳人相伴，若一直如此生活下去，也称得上神仙眷侣的生活。
<br/>　　然而，这种情绪刚刚升起，便被他无情的抹杀掉，自己的修为是元婴境界，在这一方世界也确实是巅峰的存在，可相比广阔无限的星河，自己还是太弱小了。
<br/>　　回想起自己刚刚获得系统的时候，那一腔热血，想起自己的初心，慕清霄目光前所未有的坚定。
<br/>　　总有一日，诸天万界，唯我独尊！
<br/>　　...
<br/>　　...
<br/>　　果然不出所料，仅仅片刻，擂台四周就响起一阵惊呼声，只见陆雪琪手握‘天琊’神剑，剑刃架在林惊羽脖子上，寒芒闪烁，宣布比试结果。
<br/>　　“本次比试，小竹峰陆雪琪胜！”
<br/>　　伴随着裁判长老的高呼声，擂台下青云门弟子一片沸腾，龙首峰弟子则面色如土，心中都不是滋味，本次七脉会武对他们来说，简直就是灾难。
<br/>　　擂台上，陆雪琪收起天琊神剑，不再理会林惊羽，美眸在人群中扫视一圈，看到站在最外围慕清霄后，冷若寒霜的脸上露出若有若无的笑意。
<br/>　　“雪琪...”
<br/>　　就在这时，高台上响起水月的声音，陆雪琪转过身来，见师尊已经离去，向慕清霄投去一个歉意的眼神，连忙追了上去。
<br/>　　待陆雪琪的身影消失在视线中，慕清霄也收拾了下情绪，转身向大竹峰一脉所在走去。
<br/>　　走在路上，慕清霄在青云门弟子异样的目光中渡过，昨日的一战，他的名声早已盖过久负盛名的齐昊，被称为青云门有史以来第一狠人。
<br/>　　虽然没有晋级前四强，但名声丝毫不弱，三场比试下来，废了三名龙首峰弟子，其中一个还是龙首峰的大师兄齐昊，不是狠人是什么？
<br/>　　甚至，沿途中路过，有的弟子避之不及，在他们眼中，慕清霄简直就是恶魔，齐昊本来是夺魁的最大热门，可由于慕清霄的缘故，别说夺魁，全身修为都废了。
<br/>　　陆雪琪倒是气势鼎盛，但曾书书与张小凡进入前四却是出乎绝大多数青云门长辈的料想之外。
<br/>　　在此之前，曾书书以曾叔常的独子闻名，虽然在风回峰一脉中是公认的年轻俊才，但在青云门中并不十分出名，这一次过关斩将，道法精妙，令众人刮目相看。
<br/>　　张小凡的名声也逐渐在青云门中传开，真是让大竹峰一脉倍有面子。
<br/>　　此时，东边擂台之上，三人并排而立，掌门道玄真人与龙首峰首座苍松道人站在前面。
<br/>　　道玄真人的脸上还是挂着微笑，根本看不出他对这次大试中长门弟子意外的全军覆没有任何不满，倒是苍松道人脸色阴沉的可怕。
<br/>　　原本，他认为齐昊能够夺得七脉会武的桂冠，却不曾想到，龙首峰弟子三番两次被废除修为，让他怒不可揭，若非掌门真人和田不易，他真想将慕清霄就地诛杀！
<br/>　　台下，近千的青云门人围在一起，前排坐着的都是各脉的首座长老，苏茹看着台上，低声对田不易道：“小凡真的在不知不觉中，长大了许多啊。”
<br/>　　田不易冷哼一声，没有说话，众目睽睽下，他自然也能看出来，台上三人，陆雪琪冷若冰霜，曾书书含笑而立，张小凡面色平淡，可眼中自信任谁都能看出来。
<br/>　　台上道玄真人看了这三人一眼，嘴角掠过一丝笑意，转过身子对着台下，道：“诸位，到今日为止，七脉会武已决出了前四位弟子，由于慕师侄触犯门规，所以剥夺比试资格，他们天资过人，道法精妙，俱是我青云门中的精英，肩负着日后光大我青云一门的重任。”
<br/>　　“明日比试，由于比试人数为基数，依旧由抽签决定比试顺序，抽中一号签可以直接晋级，二号三号对决，胜者与一号对决......”
<br/>　　近二十分钟后，道玄真人说了一下比试时间与地点，云海广场上的弟子逐渐散去，走在大竹峰一脉弟子身后，慕清霄忽然察觉一道冷冽且杀意四射的目光落在自己身上。
<br/>　　转身一看，却是擂台上苍松道人冷眼盯着他，嘴角不禁勾起一丝邪异的笑容，看来七脉会武后，下山历练的时候不会太平了。
<br/>　　他可不相信，自己废了龙首峰三名弟子，让苍松颜面尽失，后者会如此简单的放过他。
<br/>　　两人目光在空中对视，苍松皱了皱眉，心中更加的愤怒，同时心中有些惊异，他稍稍收敛了些气息，慕清霄是如何察觉到的？
<br/>　　此子诡异至极，却天赋骇人，哪怕有灵尊护着，也必须想尽办法斩草除根，否则龙首峰在无出头之日！
<br/>　　慕清霄不再理会苍松道人的杀意，也不知道他心中的想法，而是转身径直离去，不惹自己还好，若是惹自己，连龙首峰一锅端了。
<br/>　　苍松在青云门是高高在上的龙首峰首座，但在他眼里，也只是稍微强大些的蝼蚁罢了。
<br/>　　虽然来到本世界数年，未曾动手杀人，却不代表他没有杀人的实力，他可也不是善茬，只是低调了些，更不是任人宰割的羔羊，苍松想杀自己，他自然不会手下留情。
