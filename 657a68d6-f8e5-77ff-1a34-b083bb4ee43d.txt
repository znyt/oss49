　　然而，就在他们做出判断，打算去保护同伴的时候，房门猛然打开，一支桃木剑凌空飞来，刺中了叶少阳对面那个小鬼，桃木剑从远处飞来的惯性，将小鬼直接撞了出去，却是无巧不巧地叶少阳手中阴阳镜的一击。在半空中中，小鬼高叫一声，喷吐鬼气，震飞了桃木剑，恶狠狠地朝叶少阳看了一眼，朝窗外飞了出去。
<br/>　　叶少阳身后和头顶上空的两道鬼魂，见同伴脱险，不需要自己救援，临时又改变主意，对着叶少阳扑了过去。
<br/>　　这突然的变故，打乱了叶少阳的计划，情急之中，叶少阳猛地向前扑出去，几乎在他迈步的同时，一只鬼手拍在了他肩膀上，叶少阳浑身一个激灵，向前趴下去，鬼手正好就顺着他后背掠过，那种又酸又麻的感觉瞬间蔓延开来，叶少阳打了个哆嗦，向前翻滚了一圈，一甩手将几枚铜钱打了出去，暂时抵挡一下。
<br/>　　就在这时，从门外冲进来几个道士，最前面一个四十多岁，黑须冉冉，手里拿着拂尘，身后跟着几个像是他弟子的年轻道士。
<br/>　　黑须道士进屋之后，抬头看见几个厉鬼，立刻大喝一声：“列阵！”
<br/>　　几个弟子立刻朝两边铺展开，一个个拿着桃木剑，一脸紧张地望着房间中的三个恶鬼。
<br/>　　恶鬼长出了一口气，盘膝坐在地上，双手结印，用罡气将后背上凝聚的鬼气从体内驱逐出去。
<br/>　　人鬼有别。鬼就算是在人身上摸一把，都会立刻造成淤青，更不要说刚刚还中了一掌。叶少阳现在法力低微，生怕鬼气蔓延，是以见有救兵到来之后，立刻坐下，开始作法。
<br/>　　三只小鬼也不是没有智商，这工夫也顾不上对付叶少阳了，环顾左右，见对方人数众多，三人立刻站在一起，朝黑须道士扑过去。
<br/>　　“妖孽敢尔！”黑须道士大喝一声，身边众多弟子立刻朝中间收缩，一个个手持桃木剑，按照某种阵法的规矩，朝三只小鬼刺了过去。
<br/>　　三只小鬼挡了一下，震倒了两个小道士，然后侧飞而起，顺着窗户钻了出去。
<br/>　　“穷寇莫追！”黑须道士命令道，所有弟子立刻都站着不动，直到那三只恶鬼陆续飞走，黑须道士这才低头上下打量起叶少阳，喝道：“什么人？”
<br/>　　刘四夫妇和刘老汉之前一直没有看见恶鬼，但是听见了恶鬼的笑声和打斗的经过，吓的缩在墙角不敢动，被黑须道士突然这么一问，这才回过神来，刘四立刻爬起来，对黑须道士作揖打躬，然后扑到床上去，看着自己一动不动的儿子，有点不知所措。
<br/>　　“见过张道长，我家孩子不知道……”
<br/>　　张道长走过去，看了那孩子一眼，捏住他的手腕，用罡气略一感知，吐了口气道：“幸亏没事，只是魂力虚弱，昏迷过去。”
<br/>　　转头看着坐在地上还在作法逼鬼气的叶少阳，冷冷道：“他到底是谁？”
<br/>　　“他……是刘老伯家的内侄，说是来自……来自茅山的道士，听说了我家小顺的事，非要过来看看，一来就闯祸了，多亏张道及时赶到，不然后果不堪设想啊……”
<br/>　　张道长眉头皱起来，冷冷说道：“俗话说一事不烦二主，刘四，我看你可怜，才出手帮你，你怎么半路就找了别人，我帮你不是为了钱，也不是为了名，只求积阴德而已，我并不怕你找人帮忙，但你也得找个有法力的，不然适得其反，只会耽误了你家孩子的性命。”
<br/>　　“是是是。”刘四见张道长生气，急的差点就跪下去了，一个劲地赔不是，表示自己并不是想找这个道士帮忙，只是碍着刘老头的面子，才让这个年轻人过来看看，没想到他一出手就闯祸了，自己心中也是后悔不迭。
<br/>　　刘四一边说着，还怕张道长不信，把刘老汉拉出来让他现身说法。刘老汉心中也是有些后悔，跟着赔不是，同时也替叶少阳解释：“他也是好心，只是没想到鬼怪这么难对付……”
<br/>　　“好心！万一害死了这孩子，难道也还是好心不成？”
<br/>　　刘老汉一时语塞。
<br/>　　叶少阳站了起来，用力吐出被逼出来、含在口中的最后一缕鬼气，抬头望着这位张道长，说道：“教训好了没有？”
<br/>　　张道长盯着他，以居高临下的姿态说道：“难道贫道说的不对？我看你最多也就是个方士，居然想自己对付三个恶鬼？用一句俗话说，二两银子的本钱，也想做十两黄金的买卖？”
<br/>　　叶少阳有点无语，愤然道：“你还觉得是你救了我是吧？”
<br/>　　张道长道：“难道不是？”
<br/>　　叶少阳苦笑。“要不是你突然闯进来，对那个小鬼出手，我必能杀了它，甚至还能一举杀死另外两只，不过念在你也是好心，算了吧。”
<br/>　　张道长噗嗤一声笑出来，他身边那些个弟子也跟着一起笑。
<br/>　　“都到这份上了，还这么大言不惭！”
<br/>　　“真是冥顽不灵，要不是我师父救你，你现在已经被恶鬼杀了！”
<br/>　　张道长嘴角也浮起了一抹无奈而又轻蔑的笑容，对弟子们摆了摆手，让他们不要再说下去，自己心平气和地对叶少阳说道：“像你这样的年轻人，我也见过很多，有些少年法师学了点法术的皮毛，就以为自己天下无敌，什么样的恶鬼厉妖都敢挑战，早晚要吃个大亏，很有可能会牺牲性命……年轻人，修道之法，在乎静心，分析局势，量力而行，我说这番话，希望对你有用。”
<br/>　　这番话说的心平气和，也没有刻意的讽刺，只是站在一个前辈长者的身份，对后辈进行淳淳教诲，不过这种教训的姿态，本来就是居高临下的。
<br/>　　叶少阳无奈地耸了耸肩。
<br/>　　张道长说道：“你是什么牌位，方士？”
<br/>　　叶少阳不想撒谎，但是说灵仙的话，未免有点惊世骇俗，于是说道：“天师。”
<br/>　　张道长几个弟子噗嗤一声笑出来。“就你还天师呢，做梦呢？”
<br/>(本章完)
