　　十多分钟后，戴安娜总算是回来了，
<br/>　　“叶前辈，我已经查明情况，您要找的那块万年玄冰，如今在阿里商会旗下的万象拍卖场，并且将于今晚进行拍卖……”
<br/>　　“拍卖？”
<br/>　　叶尘听完这话，眉头微微一挑，
<br/>　　“走！去拍卖场！”
<br/>　　说完，便直接朝外面走去。
<br/>　　纳兰柔然正求之不得，当即招呼众人，
<br/>　　“快快！快走！”
<br/>　　很快，
<br/>　　众人出来酒馆，乘坐马车，开始朝拍卖场的方向赶去。
<br/>　　而就在他们前脚刚走不久，整个就酒馆便被一队全副武装的人马所包围。
<br/>　　显然是赫连将军府的人到了！
<br/>　　“搜！给我搜！就算把整个魔皇城给我翻过来，要找到杀害我儿的凶手！”
<br/>　　一名煞气十足、一身黑色铠甲的中年男子，坐在一头高大威猛的灵兽上，大声怒吼道。
<br/>　　连同整个酒馆都一阵乱晃，似乎随时都要崩塌一般。
<br/>　　而整个酒馆里的客人，以及酒馆老板、掌柜等人，早已经被赫连将军府的金甲侍卫，统统抓了起来，全部跪在地上瑟瑟发抖。
<br/>　　原来，
<br/>　　此人便是魔皇城大名鼎鼎的赫连将军府的主人，赫连鹏飞的父亲，赫连铁树！
<br/>　　“找不到杀害我儿的凶手，你们这些人，一个也别想活命！”
<br/>　　……
<br/>　　半个小时之后，
<br/>　　叶尘一行人，已经来到魔皇城最大的拍卖场，万象拍卖场。
<br/>　　“叶前辈，这万象拍卖场的规格很高，寻常人就算是有钱也进不去，必须是贵族，才有这个资格进入！”
<br/>　　戴安娜说到这里，望向一旁的纳兰柔然，
<br/>　　“公主殿下，还得借您的身份玉牌一用！以您罗刹城公主的身份，应该可以带两名随从进入其中……”
<br/>　　纳兰柔然的俏脸之上，顿时露出一抹得意之色，当即从腰间取出了一块墨色玉牌，笑嘻嘻道：
<br/>　　“叶前辈，看来您带我一起来魔皇城，还真是有先见之明！”
<br/>　　“走吧，本公主带你们进去，其他人在外面等着！”
<br/>　　说完这话，纳兰柔然小手一挥，便率先朝拍卖场中走去，叶尘和戴安娜两人，则紧随其后，至于其他人，便只能在外面守候了。
<br/>　　“我是罗刹城公主纳兰柔然，这是我的身份玉牌！”
<br/>　　来到拍卖场的入口处，纳兰柔然自报身份之后，将自己的身份玉牌递了过去。
<br/>　　那两名工作人员，明显一愣，其中一人脱口而出道：
<br/>　　“你就是那个罗刹城公主？”
<br/>　　纳兰柔然不由得眨了眨漂亮的大眼睛，俏脸上露出一抹喜色，同时又有些狐疑，
<br/>　　“怎么？你认识我？”
<br/>　　那人刚要开口回答，另外一人却一口打断，笑道：
<br/>　　“罗刹城公主的芳名，我们自然是听说过的！公主殿下里面请！”
<br/>　　纳兰柔然顿时喜形于色，对两人随手抛出几枚闪闪发光的珠子，道：
<br/>　　“算你们会说话，这是赏你们的小费！”
<br/>　　叶尘见状，则眉头微微一皱，已经猜到了大概原因。
<br/>　　以赫连将军府的势力，此刻他们这一行人，恐怕早已经被全城通缉了吧？
<br/>　　果不其然，
<br/>　　等三人进入拍卖场之后，叶尘便听到了那两名工作人员的对话，
<br/>　　“你是不是傻？刚刚差点露馅！”
<br/>　　“我也是太激动了嘛！赫连将军府刚刚发出悬赏令，谁要是能找到罗刹城公主的下落，便悬赏一万魔珠！那可是一万魔珠啊！”
<br/>　　“我看他们三人年轻不大，修为也不过，不如我们索性将他们擒了，献给赫连将军，奖励岂不是会更丰厚？”
<br/>　　“不可！你难道忘了万象拍卖场的规矩了吗？一旦进入拍卖场，便是阿里商会的贵客，便是赫连将军，也不得在这里打斗，否则就是与整个阿里商会为敌！”
<br/>　　“不错不错！我差点把这个规矩给忘了！”
<br/>　　……
<br/>　　将两人的对话听在耳中，叶尘的眼眸中，顿时闪过一抹杀意，不过很快又放弃了。
<br/>　　以赫连将军府的势力，找到他们几人的行踪，应该只是时间问题，就算他把这两个杀了，也没有太大意义。
<br/>　　而且眼下最重要的事情，是要找到玫瑰的肉身。
<br/>　　……
<br/>　　不一会，
<br/>　　三人在工作人员的指引下，进入拍卖场之中。
<br/>　　此时已经是傍晚时分，晚上的拍卖会，也已经将要开始。
<br/>　　这万象拍卖场，不愧是魔界第一拍卖场，此时几乎已经座无虚席。
<br/>　　尤其是在最上方的，那些贵宾包厢，更是全部爆满。
<br/>　　而下方的普通席位，也有等级之分，分为一、二、三区域。
<br/>　　一区比较靠前，座位也比较宽大，三区则最靠后，座位也更加窄小。
<br/>　　此时，一区和二区已经全部爆满。
<br/>　　叶尘三人，也只能在三区，找到一个比较偏僻的位置，坐了下来。
<br/>　　“待会若真有我要的那件东西，记得务必帮我拍卖下来！”
<br/>　　三人坐定之后，叶尘冲着一旁，一脸郁闷之色的纳兰柔然，忽然开口道。
<br/>　　座位好坏对他来说，无关紧要，只要能得到她要的东西就行。
<br/>　　纳兰柔然听到这话，当即拍了拍自己，已经初具规模的胸脯，一脸豪气道：
<br/>　　“叶前辈尽管放心！我一定帮你拿下！”
<br/>　　……
<br/>　　很快，拍卖会开始，
<br/>　　一身紫色长袍的白发老者，缓缓走上抬起，朗声道：
<br/>　　“欢迎诸位，来到我万象拍卖场，一年一度的拍卖大会！我是首席拍卖师阿史那……”
<br/>　　老者寒暄了几句之后，正打算宣布开始。
<br/>　　然而，
<br/>　　就在这时，
<br/>　　入口处，忽然传来一道浑厚的声音，
<br/>　　“等一下！”
<br/>　　众人顿时全都一愣，显然没有想到，竟然有人敢在万象拍卖场闹事。
<br/>　　不过等众人，看清楚来人的面庞，顿时一个个又全都愣住了，
<br/>　　原来这个不速之客，赫然正是那位赫连将军！
<br/>　　只见，赫连铁树大踏步走到拍卖厅中央的高台之上，冲着紫袍老者拱手行了一礼，道：
<br/>　　“阿布多先生，小儿鹏飞，被恶徒所杀，凶手就在这拍卖厅之中，能否让本将军，先将此人揪出来，为我儿报仇？”
<br/>　　赫连铁树说完这话之后，在四周扫视了一圈之后。
<br/>　　最终，目光迅速锁定在叶尘三人的身上，眼眸之中，顿时几欲喷出火来！
<br/>　　叶尘不由得眉头一皱，暗道一声“麻烦！”
<br/>(本章完)
