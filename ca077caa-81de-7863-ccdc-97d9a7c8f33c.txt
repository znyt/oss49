　　其中这六翼蜈蚣，是一种天生异种，世间少有的独特生物，肉身强横到极致。
<br/>　　据说六翼蜈蚣，足以是以肉身蛮力，强行撕裂空间，就算是九劫古帝，甚至是半步天人境，碰到这六翼蜈蚣。
<br/>　　稍有不慎，都会让这六翼蜈蚣在身上钻一个眼。
<br/>　　在这小小的身躯之中，蕴含的却是强横到极点的肉身。
<br/>　　在古老的时代之中，六翼蜈蚣是无数炼体之道的强者梦寐以求的至宝，以六翼蜈蚣的身躯，所炼制的六翼圣体丹，乃是最好的炼体圣丹之一。
<br/>　　想要将炼体之道走到极致的强者，无一例外都需要这六翼圣体丹。
<br/>　　那而藏金精，则是厚土之精极致所化之物，一旦是炼化，足以凝聚肉身之力，使肉身蜕变。
<br/>　　六翼圣体丹，是纯粹提升肉身之用，而藏金精，则不止是提升肉身，更是足以提升灵脉，让灵脉变的更加粗壮，力量足以暴涨数成。这两件至宝，在古老的时代之中都是赫赫有名，只有天人境的强者才有机会得到。
<br/>　　天人境之下的人物，根本是没有资格得到这两件至宝。
<br/>　　在这佛寺开启出现的无数宝光之中，楚风眠第一眼便是就见到了这两件至宝。
<br/>　　这两件至宝得到，足以让楚风眠的身体变得更加的强横，距离冲击凝炼真正的不朽真身，都有着极大的帮助。
<br/>　　当然其中还有着许多宝物，对楚风眠有着大用，但是对楚风眠价值最高的，还是这六翼蜈蚣，跟藏金精。
<br/>　　“如果是佛祖洞府，有如此宝藏倒是并非不可思议！”
<br/>　　楚风眠并不意外，佛祖可是足以跟仙帝平起平坐的人物。
<br/>　　在任何时代之中，都是至强者，最多是有人足以与其平起平坐，却是没有任何人可以将之超越。
<br/>　　佛祖陨落，遗留之物也都是绝对的至宝。
<br/>　　当看到这无数宝光之后，在场的无数强者更是变的疯狂无比，甚至是有些跃跃欲试，想要冲击进去了。
<br/>　　但是就随着一名古帝，贸然的冲入其中，却是被这佛寺之上的禁制一瞬间的轰杀殆尽之后，在场的武者才终于是冷静了下来。
<br/>　　这宝光是虚，佛寺没有开启，任何人也别想要踏足其中。
<br/>　　但是这佛寺，已经是即将开启。
<br/>　　所有的宝光，慢慢的消失，当这最后一道宝光消失的一刻，一道古老的声音响彻而起。
<br/>　　这声音，楚风眠听不懂，但是却听得出来，这是梵文，佛门之中最为古老的语言。
<br/>　　梵文响彻而起的一刻。
<br/>　　这古老佛寺的大门，轰然之间打开，无数至宝，都在这佛寺之中。
<br/>　　透过无数光芒，足以清晰的看得见。
<br/>　　“冲啊！”
<br/>　　“这佛寺的宝藏是我的！”
<br/>　　“阴阳乾坤！逆转大阵！”
<br/>　　轰隆！
<br/>　　无数身影在这刹那之间同时向着那佛寺大门冲了过去。
<br/>　　这个时候，谁先第一个步入到佛寺之中，便是就足以占据先机，最有机会得到其中的宝物。
<br/>　　这其中的佛门至宝，最差的越是曾经半步天人境的灵器，许多甚至都是天人境强者的灵器，随身之物。
<br/>　　得到一件，就足以令一名九劫古帝一飞冲天了，提升无数的实力。
<br/>　　场面疯狂无比，五行天，沧海都，齐林山三方势力的九劫古帝最多，在这佛寺开启的一刻，当即联手布置出了一道古老的大阵，笼罩在了这佛寺入口前。
<br/>　　任何靠近那佛寺大门前的强者，都被这一道古老的大阵所攻击。
<br/>　　这三方势力，打算要将所有的武者拦在外面，将这佛寺之中的宝藏全部独吞！
<br/>　　“该死！这三大势力打算独吞！”
<br/>　　“打破这大阵，冲进去！”
<br/>　　在场的无数古帝，看着那佛寺的入口被疯狂，一个个疯狂无比，纷纷轰击这三大势力联手出来布置出的大阵。
<br/>　　一名九劫古帝，也许是在三大势力的面前，不值一提，可是在场属于其他势力的九劫古帝加起来，要比起三大势力的九劫古帝还要多上两倍。
<br/>　　三大势力现在的举动，无异是引发了群怒，不管是属于那一方的强者，都不愿意这佛寺之中的宝藏被三大势力所独吞。
<br/>　　无数的九劫古帝都是联合了起来，一齐出手去轰击着那阴阳颠倒大阵。
<br/>　　这些九劫古帝，虽然联手，但是一个个心并不齐，他们的目标都是这佛寺之中的宝藏，而并非是想要真正的联合。
<br/>　　以至于虽然出手的人数众多，但是谁也不愿意动用全力，以至于无法第一时间打破这阴阳逆转大阵。
<br/>　　场面一片混乱。
<br/>　　就在这混乱之中，一刀刀光，突然从天斩下。
<br/>　　这一道刀光，简直是天地说斩的一般，无声无息之间从天而降，在这光芒之中，甚至蕴含着几分灰色的雾气。
<br/>　　混沌气息！
<br/>　　这一刀的力量，已经是触及到了混沌。
<br/>　　五行便是混沌分化而出，五行天功法的极致，便是回归混沌。
<br/>　　之前那五行刀祖施展出的五行神刀，虽然玄妙，但是远远没有达到这个境界，但是眼前这的刀光，却是触及到了混沌。
<br/>　　一刀斩杀而来，快的让人无法反应过来。
<br/>　　一直潜伏在不远处，盯着楚风眠的五行刀圣，终于是出手了。
<br/>　　在这场面混乱起来，楚风眠都有些自顾不暇之际出手了，一刀直指楚风眠的心脏，强横的力量蕴含在刀锋之上，斩杀楚风眠而来。
<br/>　　楚风眠的身影，仿佛是站在那里，都没有任何一点点的反应。
<br/>　　出手的老者，脸色露出了几分冷笑。
<br/>　　就在这个时候，那刀锋也斩杀在了楚风眠的身上，但是随即，老者的脸庞却是突然僵住了。
<br/>　　那一刀斩杀过去，没有任何的鲜血，所斩杀的，只是一道虚影。
<br/>　　“躲开了？”
<br/>　　这老者的声音，无比震惊，但是震惊之中还能够听出无尽的怒火。
<br/>　　五行刀圣，可是半步天人境的强者，成名已久，地位显赫，在五行天都是老祖宗一般的存在。
<br/>　　他出手对付楚风眠一个小辈，还是偷袭，本就是为人所不齿，现在居然还被楚风眠躲过了。
<br/>　　对他来说，更是一种耻辱。
<br/>　　“逃到哪里去了？”
