　　第23章
<br/>　　这是一匹白马，全套马鞍，身上一缕杂毛都没有，从头到尾就像一道线一般，群贼当山贼这么多年，还真没见过这么俊的马。
<br/>　　高飞豹下巴都要掉在地上，这匹白马正是当初他想抢没有抢成功的，没想到蛤蟆寨竟拿出来拍卖，而这价钱才值……一石大豆。
<br/>　　要知道，一匹普通的骡子都能卖五六两银子，普通马匹要卖到十几两，可以做战马的起码在二三十两，像眼前这么俊的白马，起码能卖到五六十两。
<br/>　　许多人看得眼馋，心中想实在不行就抢了她的！
<br/>　　“怎么，想黑吃黑呀，问问姑奶奶的刀答不答应？”小白狼冷冷瞥了众人一眼，从怀里拿出一张纸：“说是可以打八折，能用不能用？”
<br/>　　“言出必行，童叟无欺自然能用。”黄三元笑道：“秦蛮，返给白当家二十斤大豆。”
<br/>　　在这个世界，一石越合一百斤，其实像青牛山这种欠发达地方，大家还是习惯以物易物，粮食要比货币坚挺许多。
<br/>　　如此，白马的价值要远远超过八十斤大豆，恐怕连马鞍都买不起。小白狼占了便宜，也不在这里耽搁，翻身上马道：“姐妹们，咱们走。”
<br/>　　临走之前，远远抛给程大雷一个媚眼：“谢啦。”
<br/>　　程大雷心神摇曳，望着小白狼离开的背影移不开目光，嘴里呐呐叹道：真白啊！
<br/>　　有小白狼这一幕，所有山贼的购买欲都被调动起来。
<br/>　　“黑子，我那张宣传单呢！”
<br/>　　“呃……好像被我擦屁股了。”
<br/>　　“混蛋！你怎么这么败家！”
<br/>　　“好像你也用了吧。”
<br/>　　“老八，你快把宣传单还我。”
<br/>　　“什么你的，明明是我抢到的。”
<br/>　　“咱们兄弟合伙，买到轮流骑……”
<br/>　　都是穷人，过惯了苦日子，在他们看来两成就是不少粮食。虽然他们中间很多人，连两成怎么算都不知道。
<br/>　　在这种情况下，第二匹马就被拍到一个很高的价格，不过第二匹马依旧强壮，仔细算下来，还是物超所值的。
<br/>　　其实这也是程大雷的安排，先在开始排几匹好马，调动一下大家的积极性，至于那些伤马，瘦马都搭配在中间。
<br/>　　见一切按自己所想的发展，程大雷收回目光，在这边向徐神机打听小白狼的事情。
<br/>　　据徐神机所说，小白狼没上山前，家里是练武的，自小她习的一身好武艺，擅使飞刀。在山下，她本是定了亲的，只是没过门丈夫就死了，后来她是和牌位成的亲。所以小白狼就被视作不详之人，人人都说是她克死丈夫。因此，在婆家她是受尽侮辱，在某个夜里，她公公想爬上她床的时候，她一刀捅了对方，逃到杏花岭落草。
<br/>　　“你的意思是说，杏花岭上的山贼都是女人？”程大雷敏锐捕捉到一件事。
<br/>　　“杏花岭上的山贼和小白狼的经历差不多，有些是丈夫死了的寡妇，有些是被夫家抛弃。”
<br/>　　“呔，这么重要的消息为什么之前不告诉我。”程大雷愤怒道，他没想到青牛山竟然还有这样一处好所在，程大雷决心日后一定要和杏花岭多亲近亲近，结成兄弟山寨，守望相助。
<br/>　　“关于杏花岭的事，青牛山的山贼都知道呐，怎么大当家你不知道……”徐神机道：“青牛山想对杏花岭下手的人也不是没有，但她们都对男人视若洪水猛兽，我劝大当家也不要动这心思，动了也白动。”
<br/>　　“不服，我不服，你们是故意的。”
<br/>　　正在此时，高飞豹忽然在那边大喊一声。程大雷扭过头去，看到高飞豹身边立着一匹瞎马，他原先就有一匹瞎马，现在又有一匹，一左一右十分对称。
<br/>　　“这是怎么会事？”程大雷走过去问。对于飞虎寨他也不想过分得罪，知道是瞎马，黄三元怎么没拦着高飞豹。
<br/>　　“我已经尽力拦着了，可他就像抽疯一样，拼命加价。”黄三元也很无语。
<br/>　　“不行，这次不算，你们是故意把瞎马卖给我。”高飞豹道。
<br/>　　高飞豹内心也很崩溃，刚才见黄三元阻止他，他还以为铁定是一匹好马，蛤蟆寨藏着不卖，就疯狂加价，心中还很得意。可结果……
<br/>　　程大雷也很无奈，摊开双手道：“事先我们已经定好规矩，大家都是同意了的，既然二当家觉得不算，那么所有人都要一起不算……”
<br/>　　“这当然不行！”
<br/>　　“大丈夫一口吐沫一口钉，怎么能说反悔就反悔。”
<br/>　　“这也只能怪自己运气不好。”
<br/>　　说来蛤蟆寨这次拍卖的马匹，还都是身强体壮的一等马，真正受伤老弱的也没有几只。所以大家都还是很满意今天的拍卖结果，现在一听程大雷说要作废所有拍卖，自然没人愿意。
<br/>　　众怒难犯，虽然飞虎寨的势力大，但也没办法和这么多人作对。
<br/>　　“把粮食给他们，我们走！”高飞虎目光冷冷的看了程大雷一眼。
<br/>　　高飞虎已经这么说了，高飞豹自然不敢再说什么。他郁闷的从怀里拿出一张榜文：“记得，给我打八折。”
<br/>　　目送飞虎寨的人离开，黄三元笑笑：“大家继续，继续。”
<br/>　　总之，拍卖会在一种宾主尽欢，其乐融融的气氛下进行。各山寨有买一匹的，有买两匹的，也不仅仅可以用粮食购买，兵器，盔甲，金银珠宝都可以抵数，反正黄三元见多识广，看一遍大概就可以评估出值多少粮食。
<br/>　　最后，每匹马大概以八石的价格卖出，事实上，这个价格还是远低于市价。换句话说，如果这样山寨真的有钱以市价购买马匹，也根本不用从蛤蟆寨买马。
<br/>　　到晚上，所有马都被卖光了，清点今天的收获，一共卖出马匹七十六匹，总共换得粮食五百石，兵器以及其他若干。
<br/>　　一石粮约等于一百斤，五百石就是五万斤，对于现在的蛤蟆寨来说，这简直就是一笔天文数字。可以说，蛤蟆寨从来没见过这么多粮食，现在粮仓已经没填满，到处都是小麦，大豆，黑豆，小米，高粱……
<br/>　　黄三元，林少羽还算是见多识广，而徐神机，徐灵儿，秦蛮则完全没这一幕震住，感觉自己陷入一种深深的幸福之中。
<br/>　　程大雷也是恍然明白一件事，战争果然是发家的最快道路。这五万斤粮食，若是种地，还不知道要种多久。
<br/>　　程大雷记得自己还有一次抽奖机会，是打退韩玄之后系统奖励的，只是他最近脸有些黑，一直没敢抽奖。
<br/>　　今天是个好天气，繁星满天，程大雷决定抽奖。
<br/>
