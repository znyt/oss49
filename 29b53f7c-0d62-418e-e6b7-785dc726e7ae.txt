　　摇光仙子是阐教金仙中年纪最小的一位，但为人极为骄横逞强，除了师姐黎山老母的话还会稍微听一点，其余任何人的面子她都不会给。不过这不代表她是白痴，大伙的事，让她一个人承担后果，她铁定是不干的。
<br/>　　黎山老母听她一说，沉吟道：“我们是为消灭转世鬼童而来，为我三界大义，并非哪一门派的事，如果要承担什么后果，自然是大家一起，谁也不能独善其身。”
<br/>　　众人缓缓点头，表示同意。
<br/>　　黎山老母手指道风，对钟馗说道：“钟天师可知道，这人就是转世鬼童？”
<br/>　　“听过传闻。”钟馗并不意外，看也不看道风一眼。
<br/>　　黎山老母道：“既是如此，阴司为何不擒杀他，不仅纵容他，反倒要做出有利于他的事情？”
<br/>　　钟馗道：“这个，你可以去问酆都大帝，也可以去找老崔……如果阴司让我抓他，我自然毫不客气，但现在阴司没让我动手，我犯不着去管这些。”
<br/>　　黎山老母哼了一声道：“钟天师一向不是自诩听调不听封，怎么这会儿又把阴司搬出来了？”
<br/>　　“那是针对鬼域之事，若太阴山的人在这，我自然毫不犹豫出手，道风从人间来，阴司对他尚能容忍，自有道理，这事轮不到我去操心。”
<br/>　　说到这，袖子一挥，“我这次来，什么都不关心，只是来救橙子姑娘，列位也别说没用的了，直说吧，你们放人不放？”
<br/>　　黎山老母跟摇光仙子等人互相看了一眼，黎山老母道：“列位，我等已然至此，今日不可退缩，否则必然被视为笑谈。”
<br/>　　摇光仙子道：“师姐所言不错，今日之后果，我等一起承担！他日就算崔府君追究起来，玉石俱焚而已！请列位为我护法！”
<br/>　　秋明子，普法天尊，一尘法师，几人都是青冥界道佛二宗的泰斗，所谓的功亏一篑，对他们其实不是很在乎，毕竟天劫降临，是所有人的事，天塌压大家。他们之所以参与行动，虽然有责任感在内，但也是因为事情没什么损失，成了能博一个好名声，失败了也没啥关系。
<br/>　　他们更在乎的是脸面。黎山老母的一席话说到他们心里了：虽然得罪钟馗和崔府君的代价很大，但如果乖乖听话，就这么半途而废的话，不免会给人一种惧怕阴司的印象，在依附着各自宗门的那些势力面前将名誉扫地，失去威信，这才是他们最最在乎的东西，当下互相看了看，一起站到了摇光仙子的前面，摇光仙子松开了覆盖在双鱼图上的手掌，原本缓慢运转的双鱼图，立刻又恢复了活力。
<br/>　　悬挂在山谷上方的灵光组成的北斗七星阵，再度焕发出活力，吸取着七座山峰的灵力，化作切割符，维持着阵法的运转。
<br/>　　“你们——”小九等人一直在旁观钟馗和空界几大势力联盟的谈判，本来看到了希望，耐心在一旁等待，结果摇光仙子居然再度启动了北斗七星阵，一个个愤然不已，立刻再度冲了过去，试图破坏阵眼，几大宗师死死抵挡。
<br/>　　黎山老母发动了幻灵花海，主动对道风展开了攻击，一边对钟馗说道：“事已至此，钟天师请回吧，以免坏了颜面。”
<br/>　　“哪里还有什么颜面。等着承受天子之怒吧。到时候……别怪我没说过。”
<br/>　　钟馗对这个结果并不算吃惊，只是心中仍然盛怒不已，拦住试图上前进攻的萧逸云等人，朗声说道：“道风你们挡住他们，其余人，随我下山救人。”
<br/>　　说完，从地上拔出旗杆，人飞入旗帜中，直接顺着悬崖飞了下去。
<br/>　　道风挡着黎山老母，小九照旧与摇光仙子缠斗，杨宫梓率领十二门徒中的几个，对星宿海的几大观主展开了攻击，其余人也各自找到对手，捉对厮杀。之前没有结束的大战，再度延续下来。
<br/>　　萧逸云等人则跟随钟馗一起飞下了悬崖。
<br/>　　“少阳，少阳你怎么样！”
<br/>　　芮冷玉在谷口大声喊着叶少阳的名字，一边用松纹古定剑劈砍着前面的灵力网。她之前就下了悬崖，尝试了很久也没有办法破除眼前的灵力网。
<br/>　　她不知道在山顶发生了什么，只是看到北斗七星阵瞬间减弱了攻势，似乎在怠速运行，稍微放下心来，跟老郭和扁头一起静静等待着，结果阵法突然再度启动，三人惊讶不已，情不自禁地各自运用手段，破解着面前层层叠叠的灵力网。
<br/>　　松纹古定剑虽然能斩断灵力网，但刚斩断一层，更多的切割符立刻潮水般涌来，堵住缺口，老郭和扁头站在她身后，扁头也尝试破阵，但触须都烧焦了，也是没用。
<br/>　　真的像叶少阳说的，这借用山体之势形成的北斗七星阵，根本就不是人力所能对抗。
<br/>　　身在阵法最中间的叶少阳，听到了芮冷玉的呼喊，但是分不出精力来回应，他跟橙子背靠着背，各自作法，抵挡着灵力网的收紧，两人都精疲力尽，但所做的一切，面对这灵力网这近乎无穷无尽的能量，也杯水车薪。
<br/>　　“老大，我不行了……”橙子气若游丝，她修为比叶少阳要差一截，面对愈发强烈的压力，比叶少阳先顶不住了。
<br/>　　“坚持！你家萧郎君不可能不来救你的，府君大人也不可能放任你不管的！”叶少阳干脆把她拉到自己怀里，让她躺在自己膝盖上，低头看去，橙子已经现出了人鱼真身，浑身瑟瑟发抖，看上去情况不妙。
<br/>　　“来不及了……就算他们来了，也赶不上了。”
<br/>　　橙子静静地望着叶少阳，凄然一笑：“老大，我是不是很让你失望啊，我一直没好好修炼……总是让你照顾，没有帮过你什么，到这份上，还要拖累你……”
<br/>　　“说什么呢，你知道我最疼你了。”
<br/>　　叶少阳冲她勉强笑了笑，双手托举着天风雷火旗，旗帜上贴着好几张暗金神符，加持着天风雷火旗的灵力，抵挡着灵力网的威压，营造出一个小小的空间，保护着两人。
