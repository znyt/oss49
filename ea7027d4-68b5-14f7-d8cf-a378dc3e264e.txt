　　第512章
<br/>　　钟伟虎心如电转，念头过了三过，道：“程当家言之有理，不过这件事事关重大，我还需要回去和王上商量一下，这件事我做不了主。”
<br/>　　“明白，明白。”程大雷点点头：“不过，下次凉州王能否派个可以做主的人来，如此，也省得麻烦。”
<br/>　　一句话说得钟伟虎满面通红，垂下头去，忙不迭声的答应。的确，程大雷算是琴川之主，百里胜是未来的三关统领，手中都握着兵权，凡事都可做主。而钟伟虎，连宋伯康的代言人都算不上，只能算一个传声筒，身份和二人还有些差距。
<br/>　　程大雷又领着钟伟虎回到大厅，双方接下来的谈话，便是宾主尽欢。三方心里各自揣着各自的打算，表面上谈笑风生。
<br/>　　钟伟虎急需把这里的情况报给凉州王。至于百里胜，也有他要准备的事情，所以双方同时起身告辞。
<br/>　　程大雷亲自将双方送出城，冲百里胜道：“愿侯爷此去一路顺风，等侯爷什么时候回来，在下再给侯爷接风。”
<br/>　　“程贤弟留步，留步，有日后我们兄弟相聚的时候。”百里胜骑在马上道。
<br/>　　程大雷又冲钟伟虎道：“钟大人回去之后，可要将事情据实禀告，我还在等凉州的回信。”
<br/>　　“程当家放心，程当家放心。”
<br/>　　程大雷心里算盘打得啪啪响，引得百里胜大军驻扎凉州边关，自己一手托两家，引他们狗咬狗，自己也方便从中取利，毕竟，平衡才是自己所需要的。
<br/>　　当然，想着一手托两家，也需要一定的实力基础。若李乐天有吞下凉州的兵力，也不需要和自己结盟，换在凉州城，也是同样道理。
<br/>　　百里胜和钟伟虎的人各自骑在马上，准备离开。程大雷冲他们拱手，说些一路顺风之类的话。
<br/>　　正这个时候，程大雷心里咯噔一下，脖子后面汗毛倒竖，感觉某种未知的危险正接近自己。
<br/>　　他下意识握紧匹夫剑的剑柄，身子像蓄满力的弓，任何风吹草动，就会把所有力气释放出来。
<br/>　　百里胜和钟伟虎要走还没走，感觉程大雷像变了个人，如果刚才他还是与人无害，那么他此刻就充满危险性。
<br/>　　怎么，程大雷这神经病突然发疯，要灭掉自己么。
<br/>　　程大雷不知从生死场中闯过多少次，早对危险有种下意识的直觉。这次，他的直觉也没有出错。
<br/>　　耳忽然响起马蹄声，就看到远处一匹快马奔来，马上一白影，抽弓搭箭，一箭直取程大雷的眉心。
<br/>　　锵啷一声，匹夫剑在这一刻出鞘，将射到面前的羽箭敲飞。
<br/>　　同时，那匹快马袭来，马上白影跃起，手中握着拂尘，向程大雷攻来。
<br/>　　程大雷同时暴起，一跃便是数丈，手中的匹夫剑亮出，直直一剑刺向空中的白影。
<br/>　　钟伟虎和百里胜愣在当场，见程大雷成天笑呵呵的，倒是忘了程大雷还是个杀胚。
<br/>　　双方在空中交手，各自落下时，却也是谁都没有占到便宜。
<br/>　　这时候，蛤蟆城的守军已集结完毕，手中握着强弓硬驽，当程大雷一和那神秘人分开，便立刻开始放箭。
<br/>　　但见那白影信步而行，手中拂尘轻摆，便将空中的羽箭打开，表情平静无比，似乎不费吹灰之力。
<br/>　　程大雷打量着这白影，见她身着白衣，手中一杆拂尘，脸像玉石一般冰冷。
<br/>　　程大雷利用这时间读取她的信息，心中咯噔一下：又一个绝世高手……
<br/>　　现在的绝世高手，就这么不值钱么，刚解决一个，怎么又冒出来一个。
<br/>　　“退！”
<br/>　　程大雷见弓箭威胁不到此人，便下令手下暂且退下。他持剑在队伍最前方，眼睛紧紧盯着这白衣尼姑。
<br/>　　白衣尼姑也停下脚步，刚才击退箭阵，很显然，对她来说也并不轻松。
<br/>　　“大当家小心，这人很是危险。”徐神机站在程大雷身后小声道。
<br/>　　钟伟虎和百里胜躲在人群身后，刚才他们见识到了程大雷出神入化的武艺，也同样见识到了蛤蟆城出色的组织反映能力。
<br/>　　但如此，还是没对这白衣尼姑造成威胁。匹马单枪，就敢来闯程大雷的蛤蟆城，她……就不怕死么。
<br/>　　程大雷持剑而立，目光盯着这白衣尼姑，道：“报个名字吧，我们好像不认识，难道是谁派你来杀我的？”
<br/>　　白衣尼姑道：“贼人，还我徒儿命来。”
<br/>　　“徒儿？”
<br/>　　程大雷一顿，大概猜出了这白衣尼姑的身份，原来是聂隐娘的师父。这还真是打了小的来了老的，聂隐娘脑子本就有些不清楚，如此，有这么一个脑子不清楚的师父也就理所应当了。
<br/>　　“抱歉，你徒弟已经进了鬼门关，如果你想讨她的命，也就只能去黄泉问阎王爷了。”
<br/>　　白衣尼眼底杀机立绽，道：“贼人，我送你下黄泉。”
<br/>　　嘟，收到任务，击杀白衣尼听月。
<br/>　　正这时候，程大雷收到一条系统提示。程大雷有些无语，不过，随着蛤蟆城的壮大，自然会有越来越多的人要杀自己，明的不行，暗杀这种事便会层出不穷，程大雷自己要学着习惯。
<br/>　　“请了。”程大雷摆平剑。
<br/>　　“请。”
<br/>　　双方同时暴起，如两头猛虎相逢。程大雷本就是个杀胚，他展现的如此暴力，倒也不令人觉得奇怪。关键是这白衣老尼，也能让人觉察危险，便实在的不简单了。
<br/>　　面对聂隐娘，程大雷不敢动手，可面对她的师父，程大雷却是未必会怕。没有师父一定要强过徒弟的道理，况且，她已经老了，毕竟，任你功夫再强，也同样敌不过时光。
<br/>　　二人捉对厮杀，就像两团旋风搅在一起，百里胜揉了揉眼睛，发现自己竟看不见二人的动作。
<br/>　　程大雷打得暗暗心惊，当真是不可小觑了这尼姑。她举手投足，吐纳自然，每一招每一式，都会给程大雷带来巨大压力。而相比之下，程大雷一位求快，就落了下风。
<br/>　　不过，从来都是乱拳打死老师傅，战斗的事情，和境界高低没多大关系。
<br/>　　于此同时，蛤蟆城的兵已经派了出来，将白衣尼姑围在当中。
<br/>(本章完)
