　　变异鸟惨叫都没有发生就扑棱棱的从空中直接掉下来。
<br/>　　“又是个完美的尸体，好像现在鸟类的羽毛也可以做特殊的道具。不说别的，做冬天穿的羽绒服填充物那是绝对没问题的！！”
<br/>　　林峰直接走了过去就好像是一个猎人在收集自己的战利品。
<br/>　　林峰挖出晶石然后把尸体丢入物品栏内。
<br/>　　随后林峰不断狙杀，当然他也不浪费子弹，除了天上飞的寻常够不着之外，其余的他不会开枪。
<br/>　　三个小时后。
<br/>　　火神帮全军覆没都没有搞定的老龙口被林峰轻松拿下，火神帮之所以打不过主要就是九阶怪物太厉害。可是这个A级怪物在林峰手中却不堪一击。
<br/>　　“林峰，这个以后就是我们的地盘了。”
<br/>　　陈勾兴奋道。
<br/>　　“没错，我们就是这个地方的主人，日后想干嘛干嘛。”林峰道。
<br/>　　“不过在此之前，把这些晶石都挖走。”
<br/>　　“是！”
<br/>　　李广等人立即开始挖晶石了。
<br/>　　整个小镇子里已经没有活人了，在小镇子里等人还搜集到了许多吃的。
<br/>　　什么腌萝卜之类的东西，外加一些米酒，都是一坛子一坛子的。都是老百姓自己酿造研制的。
<br/>　　这些东西不准备带回基地而是留在这里当物资。
<br/>　　林峰让李广回去把自己人都叫过来现在这里驻守。
<br/>　　“林峰，我觉得你那个房子买亏了，你说，我们这么大地方，可以造多少房子卖啊。”
<br/>　　陈勾道。
<br/>　　“亏？你想多了，你去看看我那个房价现在涨多少了吧，就算我现在卖转手就是赚，根本不可能亏的，而且那个房子也算是我们在基地内的一个据点。怎么看都不可能亏，不过你小子倒是给我提供了一个思路，那就是我们这里这么多房子也可以拿着来卖。”林峰道。
<br/>　　“卖？可有人买吗？基地那边是因为有军部保护够安全还有五十米高墙，我们这啥没有啊。”陈勾道。
<br/>　　“高墙我们是别想了，但是不代表没有高墙就代表不安全，不过这需要时间，现在这个地方百废待兴，我们要做的就是先把这里给彻底的把霸占下来，然后再以此为中心把附近的一切威胁给扫除。”林峰道。
<br/>　　接下来几天。
<br/>　　林峰就让陈勾他们在四周清楚威胁，四周还是有许多丧尸和变异怪物的。
<br/>　　“对了，这些玩意我就留在这里。”
<br/>　　林峰刷的一下拿出了五辆99主战坦克。
<br/>　　“我尼玛。”
<br/>　　陈勾和李广就看到忽然眼前几辆坦克出现都吓了一跳。
<br/>　　“别问别说，少说话多做事，这些坦克以后就留在据点里了，你们两个随便挑个自己喜欢的颜色，就当我给你们的福利了。剩下的三辆，你们也找些机灵点的人。一共五辆坦克，我想应该足够应付绝大部分怪物了。”
<br/>　　林峰笑道。
<br/>　　“林峰，别说了，我今天的菊花肯定给你了。”
<br/>　　陈勾二话不说随便挑了个坦克，至于什么颜色，坦克不都这个颜色。
<br/>　　“峰哥，我爱死你了。”
<br/>　　李广嘿嘿一笑。也随便找了个坦克。
<br/>　　坦克实际上不难，尤其现在坦克都是全自动的，很现代化，炮弹也都是自己装填的，不像之前，还需要有人在里面手动上弹。
<br/>　　陈勾李广等人熟悉了下后都已经游刃有余了。
<br/>　　甚至陈勾还会坦克飘逸。
<br/>　　林峰呵呵一笑，两辆主战坦克的出现，绝对大大的增加了老龙口的安全。
<br/>　　林峰然后又去了一次黑龙镇，这个地方也算他的半个地盘了。
<br/>　　“什么？林峰，你自己搞了个地盘？”
<br/>　　黑龙镇。
<br/>　　寒鸦等人都愣住了，这才几天功夫，这个林峰居然又自己打下了个地盘。
<br/>　　“牛逼！”
<br/>　　一群人都是佩服的五体投地，他们也都想着再去打地盘，可问题他们没有这个实力。
<br/>　　除了林峰可以做到外，放眼整个基地，也就军部有这个实力了。
<br/>　　几天下来，军部是南征北战，打下了一共足足五个据点，这五个据点全部都重兵把守，也算是一个前哨，拱卫基地的安全。
<br/>　　“对了会长，我有个坦克想要部署在这里增加这里的安全性。”林峰道。
<br/>　　“什么坦克？林峰你没有开玩笑吧？”寒鸦愣住了。
<br/>　　林峰也不墨迹指着外面的两辆99坦克。
<br/>　　一群人顿时傻了眼了。
<br/>　　“真家伙！”
<br/>　　一群人个个都狂喜。
<br/>　　“林峰，真有你的，坦克都可以搞来，莫非是陆将军送你的吗？”寒鸦好奇道。
<br/>　　“多的就别问了，反正这个坦克以后就部署在这里了，你们找个机灵点的人来学习开坦克。”林峰道。
<br/>　　“林峰，这你就小瞧我们战士会了，我们这里就有现成的坦克兵！”
<br/>　　寒鸦笑道。
<br/>　　“那就好。”
<br/>　　林峰也不墨迹，丢了两辆主战坦克部署在黑龙镇后。之所以要放坦克在这里，倒不是大公无私，而是因为这里也是林峰的半个地盘啊，那不能偏爱。
<br/>　　林峰回到曙光基地来到了新开的店，店内人很多，不过其中看热闹的还是绝大部分。
<br/>　　林峰把一些怪兽材料全部拿出来让吴三卖。
<br/>　　“快看，这好大的怪物尸体啊。”
<br/>　　“我尼玛，这都是九阶的吧。”
<br/>　　“厉害啊，传闻这新开的店背后老板势力很强，如今看来果然如此。”
<br/>　　“九阶的怪物估计拉泡屎都可以把我砸死，这家伙居然可以弄到这妖兽尸体。”
<br/>　　“恐怖如斯。”
<br/>　　许多店内的人看到那变异响尾蛇的尸体，看到那变异穿山甲的尸体都是震惊了。
<br/>　　林峰把东西上架后就又回去了。
<br/>　　回到小区内。
<br/>　　小区内苏云在。
<br/>　　老龙口据点还没百分百安全，所以林峰还是让苏云呆在小区内。
<br/>　　回到屋内后。
<br/>　　发现苏云老师居然在洗澡。
<br/>　　隔着雾化玻璃林峰可以隐约看到一个美妙玉体在后面。
<br/>　　林峰咽了口唾沫，想起自己物品栏里可还有个长腿女神在等着他召唤。
<br/>　　“这天使彦要十五阶才可以召唤，还是先突破吧，这一次肯定九阶没问题了。”
<br/>　　林峰二话不说就开始增加实力。
<br/>　　九阶晶石非常的大，和人脑袋没区别。
<br/>　　其中蕴含的基因原液是非常恐怖的。
<br/>　　“妈的，不会爆体而亡吧。”
<br/>　　林峰捏碎了这个九阶晶石倒在一个大锅里，绿油油的。林峰不敢一口闷了，害怕能量巨大，把自己身体撑爆了。
<br/>　　林峰小心翼翼的舔了一口，顿时就感觉到一股热量在体内游窜。
<br/>　　‘我去，这么强大的能量？不是我吹牛皮，这舔一口就似乎比几百颗晶石要强。还好没傻乎乎的一口喝了。不然估计尿的尿都是绿油油的。”
<br/>　　林峰顿时就像是那啥啥喝水的动作一样，调皮而灵活的小舌头，三百六十度的旋转，在那里不断的舔。
<br/>　　苏云洗完澡出来看到林峰回来了吓一跳，当看到林峰在那里趴着用舌头舔水的样子，顿时还脸红了一下，这个姿势太羞耻了。
<br/>　　林峰听到厕所开门的声音边舔边看着她。
<br/>　　“老师，你洗好了。”
