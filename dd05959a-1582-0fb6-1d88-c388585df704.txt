　　运气这东西在战争中往往可以起到决定性的做用。
<br/>　　在古代有东晋时期的恒温灭蜀，当时本应鸣金收兵的士兵竟然鬼使神差的跑去擂鼓进军，搞不清状况的晋军莫名其妙的便开始进攻，最后竟然神乎其神的取得了胜利，这其中若是没有运气的成份，怕是谁都不信。
<br/>　　而到了近代，小鬼子偷袭珍珠港，数百艘战舰横渡3500海里，最后竟然没有被任何人发现，这其中运气的成份占了多少？
<br/>　　阿里·伊本同样也是走了霉运，在抢了十余艘钢铁战舰之后，他立刻命令手下将这些战船全部拖回大食，岂图回去好好研究一下这种战舰的构造，以便日后仿造。
<br/>　　毕竟他们把李世民围在这里也不是长久之计，时间拖的越久大唐的援兵便会越近，万一等到大唐的援兵来了，只怕他一艘战舰都带不走。
<br/>　　可是理想是丰满的，现实却是骨感的，幸运女神这一次并没有关照阿里·伊本，反而站到了李佑的一方。
<br/>　　当拖着战舰离开的大食人离开无名海岛，打算换一条航线回国的时候，好死不死的与李佑和李震的北海舰队走了个碰头。
<br/>　　这个时候的李佑和李震早就已经没了耐性，只想着快一点赶路，如果不是遇到那些抢了李世民整个舰对的大食人，根本就不会想着转路去那个海岛看看情况。
<br/>　　毕竟他们这一路上已经查过了无数的岛屿，完全没有找到老头子的踪影，因此他们觉得老头子应该在更远的地方。哪怕阿里·伊本早一天，或者晚一天将舰队派出去，完全有可能与北海舰队擦肩而过。
<br/>　　但是，这个世界上并没有那么多如果与可能，运气不在就是运气不在，玩命赶路的李佑几乎在看到那支舰队的同时立刻就意识到了情况有些不对头。
<br/>　　原本应该带着滚滚浓烟前行的钢铁战舰竟然是被拖着走的，而且前面那些扬起风帆的运输舰上面竟然还挂着大食人的旗帜，如此种种足以说明很多问题。而且更重要的是，对面的战舰在看到他们的舰队之后不但没有迎上来，反而有调头就走的意思。
<br/>　　李佑和李震就是再傻，看到这样的情况也大致上猜到了一些什么，对视一眼立刻下达了围攻的命令，整个北海舰队狼一样便扑了上去。
<br/>　　那些拖着战舰向回赶的大食人在看到一排排冒着浓烟的战舰时也懵了，李世民在撤下战舰的时候把所有的物资全部丢进了海里，所以那些钢铁战舰根本没有机动力，也没有任何的战斗力，与大唐舰队相遇后果可想而知。
<br/>　　就这样，双方舰队一个想逃一个急追，没用一个时辰，那些大食人便被围了起来。
<br/>　　李佑早在看到这支舰的时候就已经红了眼睛，老头子所有的战舰都被大食人抢走了，那么老头子的下场会是什么不问可知。
<br/>　　所以炸了毛的李佑完全不顾李震的劝阻，直接下达了一个不留全部杀光的命令。
<br/>　　二十余艘战舰，数百门贞观炮几乎同一时间将炮口对准了那些木质的运输舰，只五轮炮击便将那些运输舰打沉了大半。
<br/>　　接着便是接舷战，北海舰队所有水兵一齐上阵，两舰夹一舰将那十余艘大食人抢来的战舰夹在中间，无数的水兵冲了上去，与迎上来的大食人战在一处。
<br/>　　这里不得不说的是，大食人的士兵的确不那么好对付，近战之下北海舰队几乎战不到任何的便宜，几乎每一寸甲板都是用血肉拼下来的。
<br/>　　距离太近了，步枪根本派不上什么用场，而大食人的数量又不比北海舰队的人少多少，一场血战下来，北海舰队虽然抢回了两艘战舰，却也损失了近千人手。
<br/>　　“殿下，不能再这样打了！”李震看着自己的手下一个又一个的倒下去，眼睛也红了，死死扯住李佑吼道：“他们已经没有任何的行动能力了，这些战船我们就是放在这里他们也跑不了，我没完全没有必要再打下去了！”
<br/>　　但李佑这个时候根本听不进去任何劝阻，努力挣扎着吼了回去：“老子不管，这些大食人敢动我父皇，如果不杀光他们，老子誓不为人！”
<br/>　　“哐”的一声，劝说无用之下，李震推着李佑，将他重重的压到战舰的舱壁上：“李佑，你冷静一点，你看看这四周，全都是大食人，根本没有我们的人，这说明什么你还不知道么？！”
<br/>　　李佑被压在舱壁上，瞪着眼睛，喘着粗气与李震对视着，看上去大有同归于尽的意思。
<br/>　　看到如此情况李震不得不再次对他吼道：“你想想清楚，这些大食人很可能是抢了太上皇的船只想要运走，太上皇很可能没事儿！”
<br/>　　听到老头子没事儿，李佑的状态恢复了一些，眨了眨眼睛讷讷问道：“没事儿？你说我父皇没事儿？”
<br/>　　见李佑终于安静了一点，李震松了口气，松开压着他的手：“现在还不能确定，水里有那么多大食人，弄上来几个问一下不就清楚了。但你现在是舰队的总指挥，你要让自己冷静，而不是像一个莽夫一样只知道杀敌。如果太上皇现在正在某一座岛上等着我们去支援，而你却把兵力浪费在这里，你觉得这样有意义么？！”
<br/>　　李佑沉默了一下，看了一眼外面还在继续的战斗：“传令下去，撤！”
<br/>　　“是！”刚刚躲的远远的传令兵见两位大佬终于不打了，这才敢靠过来，接了命令之后跑了出去。
<br/>　　李佑看着那个传令兵离开，抽了抽鼻子，一边整理着身上的衣服一边瞪着李震“咱们两个的事儿没完，你刚刚压着老子，等回到大唐，老子要跟你单挑！”
<br/>　　李震与李佑对视着：“没问题，接到太上皇之后，我原地不动让你打个爽快！”
<br/>　　很快就要有太上皇的消息了，对于找人找了两个来月的李震与李佑来说，简直就是一个天大的喜讯，不管老头子是生是死，总算是有消息了。
