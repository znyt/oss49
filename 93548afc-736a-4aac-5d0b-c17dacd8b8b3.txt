　　第763章
<br/>　　人群自动围成一个圈，程大雷晃动着肩膀，双脚跺地，迅速令自己身体进入战斗状态。面对卢俊义，他可不敢有丝毫大意，今天若是折在此地，输不输的程大雷倒是不太在意，关键是能不能活着走出这里。
<br/>　　卢俊义拉开架门，只说完『小心了』三个字，便向程大雷扑了过来。行家一出手，便知修为不浅。同样的一扑，速度、反应、角度方位都有不同，也便是高手和庸人的区别。
<br/>　　他脚下迈弧字步，身体向前进，眨眼便来到程大雷身边。双爪变拳，一记炮拳硬生生砸向程大雷。面对这势如破竹的两拳，程大雷哪里敢硬接，身体只好退，一退再退。
<br/>　　身子忽然跃起，想要从卢俊义头顶跃过去，绕到卢俊义背后。
<br/>　　“下来吧。”
<br/>　　卢俊义忽然虎吼一声，身体斜着一撞，用肩膀撞向半空中的程大雷。
<br/>　　好险！
<br/>　　程大雷倒抽一口凉气，身子在空中一折，于无力处生力，又移了半寸，双脚安稳落地。
<br/>　　一直提醒自己不要大意，终究还是大意了。面对同等级高手，选择跃空几乎等同于找死。也就是程大雷轻身功夫了得，否则这一招失了手，也就可以宣告程大雷的输在卢俊义手中。
<br/>　　卢俊义何尝不是震惊不已，能在半空中生出力气来，程大雷的轻身功夫可以说已经到出神入化的地步。
<br/>　　盛名之下无虚士。
<br/>　　程大雷双脚一沾地，便立刻向卢俊义攻来。他的拳脚功夫本来不如何，何况是面对卢俊义这种一等一的高手。无非是仗着力气大，速度快，反应和经验都足够而已。刚才自己接了卢俊义一招，且要看看，你能不能接我一招。
<br/>　　这一招程大雷没有用拳也没有用脚，而是用了撞。自己本就力气大，那就将这一点发挥到极致。
<br/>　　我不和你比拳脚，同你比力气。
<br/>　　这一招，还是他从黑牛身上悟出来的。此刻程大雷用出来，不输一头几百斤重的大黑牛。
<br/>　　卢俊义不敢大意，眼睛眯成一条线。他选择了和程大雷同样的应对方法，后退。同样的方式，策略却不一样。虽然在后退，但保持随时可以进攻的姿态。
<br/>　　强力难久，只要程大雷力气一泄，便是自己反攻的机会。
<br/>　　不过，程大雷的力气要比自己想得更大。退了三四丈，程大雷的速度丝毫没有放缓的趋势。卢俊义却已经是退无可退，只能反击。
<br/>　　他摁出一掌，推在程大雷肩膀上，身体像陀螺一般旋转。绕了四五圈，才将程大雷的力气泄去。
<br/>　　程大雷深吸一口气，双掌探出，口中道了一声：“破。”
<br/>　　卢俊义的拳头也正好砸了过来。
<br/>　　拳掌结结实实撞在一起，空气中发出沉闷声响。程大雷退了三步，卢俊义退了五步。
<br/>　　比力气，终究是程大雷更胜一筹，但也差不了多少。
<br/>　　短暂两次交手，不是行家根本看不明白双方在打什么。但对战斗的二人来说，彼此都是平生难得一见的高手。
<br/>　　打到现在程大雷也打出火气，都是年轻人，谁肯甘拜下风。此刻他早已将输赢的后果抛在脑后，只想着一个字：赢。
<br/>　　他头发冒出热气，晃动肩膀，骨节发出噼啪响声。
<br/>　　“来。”
<br/>　　上山虎碰着下山虎，出海龙碰上入云龙，龙虎相争，一个拳脚自然，章法浑然，一个天生魔头，神力千斤。
<br/>　　当真是棋逢对手将遇良才，谁也没有半分侥幸，谁也不敢说稳操胜券。
<br/>　　“来，来，来！”
<br/>　　程大雷发出一声又一声虎吼，一记又一记撞向卢俊义。卢俊义忽然发现自己被逼入一个尴尬的境地。自己不得不和程大雷比力气，而自己的力气本没有程大雷大。
<br/>　　外面观战的人也都是满头热汗，人人均想：卢俊义果然了得，这程大雷名不虚传。
<br/>　　“来。”
<br/>　　程大雷歇斯底里一声后，将全身力气凝于一拳，恶狠狠的砸向卢俊义。
<br/>　　输赢就在这一拳了。
<br/>　　卢俊义胳膊酸疼，刚才程大雷给他的压力不小。这个时候他便想着退，但程大雷速度太快，已经来不及，他只能双臂格在一起招架。
<br/>　　程大雷一拳砸塌了他的架子，紧跟着一拳闷在他的胸口，卢俊义身体重重跌了出去，口中吐出一口鲜血。
<br/>　　卢俊义心中懊恼非常，刚才如果硬接，未必接不住。也就是这一思一退，十成力气只用了七八成。满打满算的力气也不是程大雷对手，何况只是七八成。
<br/>　　自己力气不如程大雷，但程大雷仗着这一点赢不了自己。自己输在一个勇字，程大雷一往无前，而自己只想着稳扎稳打。
<br/>　　所以卢俊义虽然懊恼，但并不委屈，输，他输得心服口服。
<br/>　　双方交手到现在，谁也不记得那十招之约，也数不清双方究竟动过几次手，但大家都知道最后输的是卢俊义。
<br/>　　程大雷吐出一口浊气，目光冷冷扫过。凡是触及他目光的人，纷纷避开，不敢与他目光对视。
<br/>　　“百里胜死了，这世上还有你们需要怕的人，还有你们需要敬畏的道理。”程大雷略顿了顿：“想要去做贼，本当家是做贼的祖宗。”
<br/>　　卢俊义挣扎着站起，示意兵丁散开一条路。人群面面相觑，一是不敢违抗卢俊义的命令，二也是怕了程大雷的拳脚。
<br/>　　“送程当家。”
<br/>　　程大雷大摇大摆离开。
<br/>　　一路出了兵营，拐过一个接口，程大雷忽然停住脚步。
<br/>　　“刘发财，后面有没有人盯着？”
<br/>　　“好像没有。”刘发财回头看了看。
<br/>　　“来……”程大雷伸出手。
<br/>　　“怎么了大当家？”刘发财。
<br/>　　“扶我一把。”程大雷从齿缝中吐出四个字。
<br/>　　刚才与卢俊义交手，虽然侥幸胜了。但对这样等级的高手，一次胜负其实什么都说明不了。程大雷击倒卢俊义，但最后一拳给他造成的压力太大，现在胳膊已经拉伤，身体也是酸胀难忍。
<br/>　　刘发财一楞，他还未反应过来。李婉儿便已伸出手，搁在程大雷掌心中，搀扶住他的胳膊。
<br/>　　程大雷微微将身子向李婉儿倾了倾。
<br/>(本章完)
