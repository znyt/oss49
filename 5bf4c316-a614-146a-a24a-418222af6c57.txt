        苏漫雪在心里打起了这种如意算盘。
<br/>
<br/>        童筱颖一双透着怒火的大眼睛，愤恨地瞪着苏漫雪。
<br/>
<br/>        她的脸色也因为苏漫雪那话儿气得惨白如纸，胸口随着呼吸的程度起伏更加明显。
<br/>
<br/>        “苏漫雪，你疯了！”童筱颖忍不住对苏漫雪怒吼。
<br/>
<br/>        苏漫雪却只是白了童筱颖一眼，洋洋得意拉高了嗓音：“筱颖，你也别怪我！这一切，都是因你而起！谁叫你不好好在老家待着，偏偏要跑到滨江城来！我给过你机会的！是你自己不会珍惜！我恨透了你！”
<br/>
<br/>        “为什么？为什么你恨我？”童筱颖难以置信地质问。
<br/>
<br/>        一开始，苏漫雪若是因为玉坠的事情，才与她断绝来往，她能理解。
<br/>
<br/>        那么现在呢？
<br/>
<br/>        她总是对她忽冷忽热，说话也是阴阳怪气，时好时坏。
<br/>
<br/>        童筱颖觉得现在的苏漫雪，完全不是她以前所认识的那个苏漫雪了。
<br/>
<br/>        到底是什么让她变成现在这个样子？
<br/>
<br/>        还是说，她一直都是这个样子，只是她没有发现？
<br/>
<br/>        “因为……”苏漫雪欲言又止，顿了顿后，才接着说道，“你明明已经嫁给了傅昼景，却在背地里仗着跟我长着一模一样的脸去勾引我的夜沉！”
<br/>
<br/>        其实她本来是想说，因为傅夜沉爱的女人是她童筱颖，但是，她觉得这事情永远都不让童筱颖知道得好。
<br/>
<br/>        “我没有！”童筱颖咬了咬下唇，为自己辩解。
<br/>
<br/>        苏漫雪嗤之以鼻道：“行了！这世上最不要脸的女人就是你童筱颖了，吃着碗里瞧着锅里！别以为我不知道，你私底下跟我的夜沉在外过过夜，而且还不止一次！”
<br/>
<br/>        “我和大哥之间是清白的！”童筱颖斩钉截铁地说道。
<br/>
<br/>        苏漫雪冷笑了笑：“清白？呵？童筱颖，你的如意算盘是什么。我一清二楚！其实，你和傅昼景一直在暗中算计着我家夜沉吧！你一边仗着一张跟我长得一模一样的脸在暗地里勾引我的夜沉，一边把夜沉所有的事情都告诉傅昼景。噢，对了！你该不会是打算，到时候来个偷龙转凤？想办法绑架我，挟持我，然后把我弄失忆，跟我调换了身份。你好冒充我成为傅家的大少奶奶！然后，你再和傅昼景里应外合，将夜沉的傅氏集团占为己有！”
<br/>
<br/>        “……”童筱颖心悸地摇了摇头。
<br/>
<br/>        苏漫雪怎么可以如此可怕？
<br/>
<br/>        把歪念头想到她身上去也就算了，居然还要扯上昼景。
<br/>
<br/>        “苏漫雪，你不要血口喷人！我不许你冤枉昼景！”童筱颖不禁双手握拳，恼火地怒吼。
<br/>
<br/>        苏漫雪啧啧嘴，微微眯起了双眸，蔑视着童筱颖，咧嘴道：“哼！童筱颖啊童筱颖！你看人是什么心态，我可是一清二楚。就说说你那前男友吧！他那么穷，你还愿意跟他交往，为他勤工俭学，你图什么啊？只不过是图人家日后飞黄腾达，好将你娶进门，带你脱离贫困罢了！还有傅昼景，我想，你一定也是处心积虑的算计了他后，才逼得他跟你领证结婚的吧！像你这种女人，我岂能不恨得牙痒痒。”
<br/>
<br/>        “我突然替大哥感到悲哀了！”童筱颖忽然间收敛了怒火，淡定地冷冷一笑。
<br/>
<br/>        苏漫雪身心一怔，刚为自己站上风而沾沾自喜，但此刻，她觉得童筱颖有一丝不对劲：“你什么意思？”
<br/>
<br/>        “因为，你不懂爱，而且你还是一个蛇蝎心肠、口腹蜜剑的女人！”童筱颖咧着嘴，不痛不痒地说，“我童筱颖，还有我丈夫傅昼景，清者自清！”
<br/>
<br/>        “我说你两还不把她先奸后杀，金钥匙和钱，我都会给你两！”苏漫雪懒得再跟童筱颖浪费口舌，而是直接向这两个歹徒下命令。
<br/>
<br/>        “我要是的是两把金钥匙，傅大少奶奶可都有？”拿枪指着童筱颖的歹徒接着问道。
<br/>
<br/>        苏漫雪身心一怔，不假思索地反问：“什么两把？我这儿只有一把！”
<br/>
<br/>        “噢？看样子，傅大少奶奶要食言了！”歹徒没好气地说。
<br/>
<br/>        苏漫雪静下心来想了想，这金钥匙是傅老太爷给她的，倘若有两把的话，那么另外一把，傅老太爷一定是交给她童筱颖了！
<br/>
<br/>        想到这里，苏漫雪抬起手来，指着童筱颖的鼻子回答道：“另外一把，她知道在哪里！”
<br/>
<br/>        “是吗？”歹徒一听，立即扣动了手枪的扳指，蓄势待发地看着童筱颖，质问，“说！金钥匙在哪里？”
<br/>
<br/>        “你开枪吧！”童筱颖面不改色地说道，起初她还很害怕，但现在，她已经不害怕了。
