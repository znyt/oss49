　　“轰隆！”
<br/>　　暗古魔尊一掌之下，天地颤动，这一掌的力量，可怕到了极致。
<br/>　　这一掌之间，便是纯粹的力量碾压，没有任何的技巧，正因如此，才是最为可怕的。
<br/>　　若是以技巧取胜，却是可以利用手段，接招拆招。
<br/>　　但是这纯粹的力量，却是重剑无锋，令一切技巧，在这力量面前，都没有任何的意义，只能是纯粹的，对拼力量。
<br/>　　万古不败真魔体，虽然只能用来防御，但是却足以剩下大量的力量来，全部用来攻击。
<br/>　　暗古魔尊的这一掌的威力，却是要比起刚刚，还强上一倍之多。
<br/>　　“对拼力量？就算是生死境的强者，我也不怕！”
<br/>　　楚风眠眼神平静，手掌一翻，便是突然一掌拍了出去，直接便是凝聚战龙血脉，凝聚而出战龙之爪来。
<br/>　　瞬时之间，战龙之爪便是轰击过去。
<br/>　　“战！”
<br/>　　一个“战”字一出，突然间楚风眠身上的气势，便是与之前截然不同。
<br/>　　远古战龙，越战越勇，如今楚风眠拥有战龙血脉，本身的性格便是跟远古战龙一样。
<br/>　　越是强大的敌人，越是足以激发出楚风眠的力量来，令楚风眠的力量，都在节节攀升。
<br/>　　“轰！”
<br/>　　一爪之间，猛然无尽力量压迫而下，这一爪，跟暗古魔尊的一掌，直接相撞。
<br/>　　一爪一掌之间的力量，居然是不分上下。
<br/>　　“这暗古魔祖的实力，幸好只有生死境二重左右，若是在强大一点，我的力量，便是无法在跟他抗衡了。”
<br/>　　楚风眠心神一动，交手之间，楚风眠便是将这暗古魔尊的力量，给摸了个透。
<br/>　　不过这暗古魔尊，也算是楚风眠遇到的最大劲敌。
<br/>　　暗古魔尊的境界，虽然是跟楚风眠所击败过的虚家老祖一样，同为生死境二重，但是暗古魔祖的实力，却是要强大的多。
<br/>　　毕竟是地魔门出身的强者，只怕这暗古魔尊，挥手之间都可以灭杀无数像是虚家老祖那样的生死境强者。
<br/>　　同为相同境界，但是实际上的力量差距，却是天差地别，正如同楚风眠的实力，也是远超同境界武者一样。
<br/>　　“战龙之爪，给我轰！”
<br/>　　楚风眠心中思考之际，便是再次催动战龙之爪，压迫下去。
<br/>　　这暗古魔尊，若是让他积蓄力量，楚风眠也对他没有任何好办法，唯有强攻才行。
<br/>　　毫不犹豫，战龙之爪便是再一次轰击而下。
<br/>　　“想要轰杀我？笑话？除非是足以将我彻底镇压！不然任何人不可能击败的了我！”
<br/>　　面对着楚风眠的战龙之爪，暗古魔尊便是不闪不避，脸色露出几分讥讽来。
<br/>　　他根本便是不抵挡，直接便是以肉身，去抗这战龙之爪。
<br/>　　蕴含无尽龙威，力量极限的一爪，在打在暗古魔尊身边的一刻，却是直接消失了。
<br/>　　被他身边所围绕的无数恒古魔意，给直接吞噬，同时这洞窟之中的魔意，便是也少了不少。
<br/>　　万古不败真魔体，只要是拥有足够多的魔意，一切伤害，都可以尽数吸收，简直便是九域中最为难产的秘术之一。
<br/>　　“小子，你的实力的确是超乎我的想象，就连我地魔门中，都没有像你这样的天才。”
<br/>　　“能够以御风境的境界抗衡我，这简直是前所未闻！可惜今日你还是要死，因为我是不败的存在！”
<br/>　　暗古魔尊冷笑连连道。
<br/>　　“你的一切抵抗，都不过是在白费力气，在这地魔界中，我便是无敌的存在！”
<br/>　　开口之间，暗古魔尊又是一掌凝聚灵力，猛然对着楚风眠轰击过去。
<br/>　　顿时整个洞窟之中，便是魔意纵横，这些魔意，化身成为一尊尊魔头，便是向着楚风眠张牙舞爪的攻击而来。
<br/>　　“滚！”
<br/>　　楚风眠口中淡然吐出一个字，便是如同一声龙吟，彻底响彻而起。
<br/>　　“天龙八音！”
<br/>　　这天龙八音，本就是克制天下阴邪之物，那凝聚而出的无数魔头，在这一声龙吟之下，全部被毁灭。
<br/>　　“万古不败真魔体，的确是诡异难缠，但是并非无法破解，你还真当你是无敌的人物？一个地魔门的小魔头罢了，就这点旁枝末节的魔道，也敢大话说杀了我？”
<br/>　　楚风眠看着暗古魔尊，一字一字的说道。
<br/>　　“也罢，今日我便是破解了你的万古不败真魔体！看你还如何嚣张！”
<br/>　　开口只见，楚风眠的手中，赤炎剑便是一剑斩出，顿时整个洞窟之中，便是被无尽剑锋所充满。
<br/>　　楚风眠的一剑斩出，便是分化万剑，一剑万重，这等惊天动地的剑意，也唯有楚风眠，才能如此轻易的斩出。
<br/>　　这其中的每一道剑芒，便是对着的无数魔意，直接斩杀过去。
<br/>　　“不用白费力气了，这些魔意，根本不是你的剑锋可以斩破的！就算是你斩杀他们一千次，一万次，他们也都会从新凝聚的！”
<br/>　　暗古魔尊表情嚣张到极点，脸上尽是阴冷道。
<br/>　　“我看你如何破解了我的万古不败真魔体！你的实力，可以达到这一步，身上的奇遇绝对是无穷无尽！你的血脉，甚至是超越了真龙！乃是祖龙，甚至是龙皇的血脉！”
<br/>　　“得到了你的血脉，我便是可以炼化成魔龙血脉，令我的实力，再进一步！”
<br/>　　暗古魔尊开口，便是再一次的凝聚灵力，一掌轰击而去，他的每一掌，都没有丝毫的技巧，完全便是硬碰硬之间。
<br/>　　要以巨大的力量，将楚风眠活生生的打死！
<br/>　　“杀戮！”
<br/>　　楚风眠眼神一变，双瞳变的通红，一身杀意凝聚，无尽血气，充斥在楚风眠的剑锋之上。
<br/>　　猛然一剑，斩了出去，与这一掌，再度交锋，将这暗古魔尊一掌的力量，直接斩破。
<br/>　　其中蕴含的无尽血气，更是直接冲着暗古魔尊的身躯，斩杀过去。
<br/>　　但是便是跟刚刚一样，这无尽血气，在即将斩在暗古魔尊的身旁一刻，便是被直接全部吸收了。
<br/>　　在暗古魔尊身边的恒古魔意，在强大的力量，都能直接吸收，除非是令他身边的魔意全部消失，不然的话，近乎无敌。
