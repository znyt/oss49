　　阴阳谷，阴谷入口的沙滩上，楚涵和水族圣女并肩从小道走来。
<br/>　　“这次还是多谢水族相助。”楚涵随意的说道，表情却带着一丝可惜，神色有有些游离。
<br/>　　鱼柔女表情淡然：“你冒着危险再次进入阳谷的水族墓地，我们水族助你也是理所应当，银净水而已，这片海域多的就是。”
<br/>　　楚涵摇摇头：“可惜，墓地完全被水淹没之后，那里的磁场和水压都成了问题，哪怕我有备而来也潜入不了，要不是你将那些黄金贝壳都放在我这里，恐怕还找不到原路返回。”
<br/>　　鱼柔女的眼神一动，带着莫名的笑意道：“你可惜的最终目的，还是因为火族圣物的线索又丢了。”
<br/>　　楚涵没说话，当做是默认，但这时候他又忽然转身，目光如炬：“你们水族镇守这阴阳谷的入口，这几个月可有陌生人进来？”
<br/>　　楚涵的话中意思，自然就是当初在大峡谷遇到的那批火族人，自从火山爆发之后就彻底消失，以楚涵的推断火族人应该是回到了阴阳谷，但水族没有道理不发现，毕竟这入口虽有两处，但却都在水族的注视之下。
<br/>　　莫非死在了火山喷发中？
<br/>　　楚涵皱起眉，想到这一可能的他，顿时心情变得很不好。
<br/>　　鱼柔女笑盈盈的看着楚涵，与他并肩继续往前走：“你杀罗生的那天，正好有一批人进来。”
<br/>　　楚涵的脚步一顿，心中顿然的看着鱼柔女，那副表情显然就是在质问：“你怎么不早说？”
<br/>　　“你想说那些人是火族么？”鱼柔女不慌不忙，面对楚涵的直视处变不惊：“哪怕是你也找不到。”
<br/>　　楚涵皱眉，不明所以。
<br/>　　“你应该知道这阴谷内有一重禁制，普通人进来找不到任何旧址，怎么走最后都是通往阴谷出口。”鱼柔女解释着道：“也只有八门血脉者才能在阴谷内发现旧址踪迹，但也只能找到本家的踪迹，其余七门是完全看不到的，也就是说水族再如何在这阴谷内搜寻，也只能找到水族旧址而已，根本摸索不到火族居住地，哪怕我们知道他们就在阴谷出口附近。”
<br/>　　听着鱼柔女的话，楚涵忽然想到自己曾与白允儿一同去过白家旧址，甚至还住了好几天，如果说只有白家人才能进入那处地方，那发生在他身上的意外是怎么回事？
<br/>　　看着楚涵陷入沉思，鱼柔女并不知道这一层秘密，只当楚涵这才明白过来原因。
<br/>　　于是继续解释道：“而如果那些人不是火族人只是普通人，这时候不是死在了阳谷就应该早已不在这片地带了，所以再怎么找也是白费力气。”
<br/>　　楚涵忽然有些不理解这个女人的想法：“所以你就没说，包括水族的族长和长老，你也没说火族人可能已经进入了阴谷？”
<br/>　　“为什么要说？”鱼柔女反问。
<br/>　　楚涵当即哑口无言，只能摇摇头继续问道：“那有什么办法可以让阴谷的这古怪禁制解除？”
<br/>　　鱼柔女眉头一挑，好似听到了什么好笑的事情一般看着楚涵：“如果能解除，在外界混的风生水起的那四大偏门家族，早就想办法回来了。”
<br/>　　听着鱼柔女有所指向的话，楚涵忽然想到四大神秘家族只出动了几人来过阴阳谷，甚至最热闹的那次还是几个家族中的年轻一辈打赌所致，只是谁也没想到这次回阴谷，让三家的嫡系都返祖，随后罗家才会让罗生前来调查。
<br/>　　神秘家族当然不可能只有这么几个人，但他们的重点似乎并不在阴阳谷，而是放在外界。
<br/>　　这很奇怪，尤其在楚涵看来很奇怪。
<br/>　　他看着鱼柔女，问出了最大疑问：“为什么？是什么原因让你们都放弃了接触禁制的想法？”
<br/>　　鱼柔女轻轻一笑望向远方：“阴谷禁制的开启是因为这片地域的自我保护，因为谁也没有见过的那个回炉一门，当年出了事故，那处旧址第一个出现雾霾，紧接着就是禁制开启。”
<br/>　　“事故？”楚涵还有很多疑问，但这会儿却抓·住了鱼柔女话语的中心：“那一门真的有人啊？”
<br/>　　他深深记得高少辉说过，回炉一门的人压根就没出现过，从神秘家族祖祖辈辈开始，往前到曾住在阴阳谷内的好几千年，都没有人见过回炉一门的人。
<br/>　　鱼柔女眼神灵动，似笑非笑的说道：“的确没人，但是有守护者啊！事故的发生就是因为守护者死了，原本在阴谷内的平衡，就是回炉一门由守护者镇守，其余七门相互牵制，但回炉一门毕竟是阴阳谷的中心，守护者一死，哪怕七门之间的平衡早好也没用，我们只是那一门之下的臣子而已。”
<br/>　　楚涵震惊无比，这是他第一次知道回炉八门的秘密。
<br/>　　“我一直以为回炉的守护者是制裁家族。”他呐呐说道：“但听你这么说，是另有其人？”
<br/>　　鱼柔女停下脚步，眼神随意的看向了旁边。
<br/>　　楚涵望去，那是他第一次进入阴谷时见到的灰兔，也是阴谷的原住民，在这里生活的无忧无虑。
<br/>　　鱼柔女话中有话：“守护者不是人，我们甚至不能确定它是不是生命。”
<br/>　　楚涵脑海中‘轰’的一声炸响，守护者不是人，鱼柔女的目光在看向那些灰兔……
<br/>　　这还不够明显吗？！
<br/>　　当楚涵正在无尽的震撼之中时，鱼柔女已经缓和了情绪，继续道：“当年那场事故的发生，是因为回炉四偏门出了异心，想要夺取守护者的限权，取代回炉一门的地位然后打开通往银河的大门，让我们离开这个异乡，上回罗生的话你应该都想明白了，我们都不是这个星球的原住民。”
<br/>　　楚涵再次震撼，他呆傻的看着鱼柔女，好半天才反应过来刚刚他听到了什么。
<br/>　　也就是说，这世上真的有办法冲出地球，有办法与星河之中其余有生命的星球接上轨？！
<br/>　　“但是他们失败了。”鱼柔女继续道：“虽然他们的步骤很完善，将所有能找到的千月石都扔进了深海以牵制制裁者，再联合而起冲入回炉旧址，但他们没有想到守护者采取了玉石俱焚的方式，碎成了十块碎片消失不见。”
