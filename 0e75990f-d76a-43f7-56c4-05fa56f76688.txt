　　就算是楚风眠，现在他的遁光都已经是慢到了极点，几乎只是神力境，甚至是御风境级别武者遁光的速度。
<br/>　　每前进一里，都要耗费无数灵力。
<br/>　　又是花费了足足四五个时辰的时间，楚风眠才前进到了六千里的距离。
<br/>　　每通过一千里，这独桥之上的压力就会剧增。
<br/>　　在踏入到六千里之后，楚风眠身上的压力已经是让他再也无法催动遁光的了。
<br/>　　楚风眠已经是感觉到了，他已经来到了这洞府的深处。
<br/>　　距离这独桥的尽头，应该不远了。
<br/>　　因为楚风眠已经是听见了，阵阵狮吼声，在这洞府的深处传递了出来。
<br/>　　“走！”
<br/>　　无法飞行，楚风眠便是降落了下来，催动身法，向着独桥的深处直接跑了过去。
<br/>　　纵然是承受着如此巨大的压力，楚风眠身形的速度依然不满。
<br/>　　魅影身法被楚风眠催动到了极致，一步一个虚影，让人都有些摸不清楚，哪一个才是真正的楚风眠。
<br/>　　现在楚风眠的实力，早已经将这魅影身法修炼到了极致，一步一幻，穿行数里。
<br/>　　这周围的光幕，已经是变得无比稀少了，大约百里，才有一团光幕的出现。
<br/>　　但是楚风眠利用灵识，透过这光幕，便是足以看见其中摆放的一件件宝物。
<br/>　　黑龙木，星河砂，无数天级灵器，都摆放在了其中。
<br/>　　这其中的宝物，不止是九劫古帝，甚至是让半步天人境级别的强者看见，都要为之疯狂，想要得到。
<br/>　　“这霸狮武祖到底是得到了多少的宝藏……”
<br/>　　楚风眠一路上看见这光幕之中蕴含的无数宝藏，都是一个天文数字了，绝对是比起九玄仙尊洞府之中所蕴含的宝藏加起来，还要更多。
<br/>　　不过面对这些光幕之中的宝藏，楚风眠都是全然无视，他的目标，是这独桥的尽头。
<br/>　　至于这些，楚风眠都不在乎。
<br/>　　一百里，两百里。
<br/>　　对古帝来说，几乎是弹指一挥间的距离，如今在这杜桥之上，却是无比艰难。
<br/>　　每前进一步，不止是要承受巨大无比的压力，对灵力的消耗，也是无比巨大。
<br/>　　就算是让一名九劫古帝到来，都是走不了几步，都必须要坐在原地休息，恢复灵力了。
<br/>　　而楚风眠凭借神脉，可以源源不断的汲取灵气，恢复灵力，可是逐渐的，楚风眠也感觉到了一丝灵力不支。
<br/>　　在这独桥之上，楚风眠消耗的灵力，比起神脉汲取的灵气，还要更多。
<br/>　　楚风眠也只能够是将空戒之中之前凝聚出的圣品灵石，大量的拿出来，吞噬灵石来恢复灵力。
<br/>　　幸好是之前楚风眠没事的时候，就以神脉在凝聚灵石，现在楚风眠空戒之中的圣品灵石数量，足以百亿，千亿来计。
<br/>　　不过楚风眠每时每刻所吞噬的圣品灵石数量，也是无比惊人的，几乎是一息之间，就足以吞噬千万枚圣品灵石。
<br/>　　这才足以支撑住楚风眠的灵力消耗，让楚风眠全力赶路进去。
<br/>　　六千九百里了！
<br/>　　楚风眠一边赶路，一边也是记着他现在的位置所在。
<br/>　　突然间就在楚风眠目光抬起头，看向前方的一刻，一道宫殿，突然映入到了楚风眠的眼帘之中。
<br/>　　这一道宫殿，跟之前所见的那些宫殿，截然不同，高大数百倍，气势磅礴。
<br/>　　在这宫殿的最上方，还有着一座雕刻的雄狮，位于宫殿之上，这虽然是一尊雄狮雕像，但是其中蕴含的武道，比起许多武祖的雕像，还要恐怖。
<br/>　　霸狮武祖的雕像！
<br/>　　一尊雄狮雕像，能够拥有如此可怕的武道，唯有霸狮武祖的雕像。
<br/>　　到了武祖这个级别的强者，所留下的雕像，都会蕴含武意，一旦是武道登峰造极，便是会跟武者，融为一体。
<br/>　　像是以现在楚风眠的剑道，如果楚风眠自己亲手雕刻出一枚石像来，这一枚石像上，也会拥有极为恐怖的剑意。
<br/>　　万古流传下去，也会是一件至宝。
<br/>　　当然那始祖天龙雕像，则更加夸张，在其中不止是足以领悟出始祖天龙的武道，就连始祖天龙最强的始祖九式，都足以从中领悟出来。
<br/>　　想到这里，楚风眠眼神之中也露出了几分炙热之色。
<br/>　　他现在的实力再进一步，想要在领悟始祖第三式，甚至是第四式，都绝对足以了。
<br/>　　只是现在楚风眠不好会圣龙秘境，找到一个机会，如果可以讲那始祖天龙雕像，弄到身边来的话。
<br/>　　随着楚风眠的实力提升，终究有一日，楚风眠可能将始祖九式，全部参悟出来。
<br/>　　等找到一个机会，楚风眠还是要回圣龙秘境一趟，起码是去参悟一下这始祖天龙雕像。
<br/>　　楚风眠一边想着，一边也来到了这一座宫殿前。
<br/>　　百里之外，楚风眠都足以感觉到那霸狮武祖雕像其中蕴含的气势。
<br/>　　如今亲眼所见，这霸狮武祖雕像之中的神韵，简直是如同活的一样。
<br/>　　仿佛是这一头雄狮，只是在沉睡，一旦是被人惊扰，机会苏醒，撕裂眼前的一切。
<br/>　　“七千里，这独桥尽头，居然是足足七千里，才足以来到这独桥的尽头。”
<br/>　　当楚风眠最后一步，踏入到了这宫殿前方的一刻，身上的压力，轰然散去。
<br/>　　楚风眠也明白，这里便是独桥的尽头，也算是完成了这霸狮武祖的考验。
<br/>　　楚风眠身上的衣衫，都是湿透了，疲惫无比，不过有着神脉的力量，一旦是压力消失。
<br/>　　十几个呼吸的时间，楚风眠身上的灵力，便是全然恢复了过来，旋即楚风眠睁开双眼，看向这宫殿。
<br/>　　这一座宫殿的大门，完全的敞开着。
<br/>　　在上面，没有任何的禁制，阵法，任何一个人，都足以踏足其中。
<br/>　　不过楚风眠可明白，想要踏入这其中的困难，何止是几道禁制，阵法可以比拟的。
<br/>　　独桥七千里，九劫古帝都无法踏入其中，只怕是唯有半步天人境，才有可能真正的来到这里。
<br/>　　这霸狮武祖的考验，也的确可怕，唯有接近半步天人境的实力，才足以完整他的考核。
<br/>　　楚风眠踏入到了宫殿之中，一时间天旋地转。
<br/>　　这一方宫殿，另有乾坤！
