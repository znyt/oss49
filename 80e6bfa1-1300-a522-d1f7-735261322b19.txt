　　罗昊看向站在门口的詹姆斯。
<br/>　　尽管已经从麦克米兰口中有所了解，可罗昊还是希望能够从詹姆斯得到“官方”的解释。
<br/>　　“你应该已经从麦克米兰那边得到了消息。”詹姆斯走进房间，看着目光呆滞，甚至还嘴角留着口水的Parker，说道：“我们给他使用了药物让他开口，而这种药物会直接摧毁人的大脑。”
<br/>　　詹姆斯指了指自己脑袋的位置，说道：“他现在的智商，只相当于一个五六岁孩子的水平。明天他就会被送回国，运气好或许会在一个疗养院里度过他剩下的日子；运气不好，就是被送进联邦监狱。”
<br/>　　罗昊跟詹姆斯一起离开了关押Parker的房间，返回联合行动司令部指挥室的路上，罗昊向詹姆斯打听了关于西伦科特的下落，毕竟西伦科特才是这次联合行动的最终目标。
<br/>　　詹姆斯无奈的摇了摇头，说道：“西伦科特这次非常谨慎，原本我们以为可以从Parker这儿得到西伦科特的下落，但是很遗憾，这次就连Parker也不知道西伦科特去了什么地方，本来我们都认为西伦科特还在中东，但是结合目前所得到的情报，西伦科特很有可能已经离开了中东。不过Parker还是交代了几个我们并没有掌握的西伦科特设立的安全屋，海军陆战队情报室（MCIA）已经对Parker所交代的那几个安全屋进行调查，希望可以从那里得到一些线索。”
<br/>　　麦克米兰他们返回联合行动司令部已经是第二天凌晨，房间中，罗昊听到从外面传来的直升机涡桨发动机轰鸣声，立刻从床上翻身坐起，快速穿好衣服，在云熙额角上亲吻了一下，让她继续睡觉后，离开了房间。
<br/>　　航母甲板上灯光照得很亮，麦克米兰带着海豹六队队员以及海军陆战队队员从SH-53E“超种马”运输直升机机舱中走出来，他们身上的迷彩服很脏，显然是又经历了一场激烈的战斗。
<br/>　　麦克米兰走下运输直升机后，立刻招呼着在航母甲板上待命的医疗队上前救治被他们救出来的伤员。麦克米兰看到了站在航母甲板上的罗昊，对着身边队员招呼了一声，便是朝着罗昊走去，说道：“我们找到了你们的队员，他们……”
<br/>　　听到麦克米兰的话，罗昊整颗心都揪了起来，强行让自己冷静下来，盯着麦克米兰等着他后面所要说的话。
<br/>　　“他们没有生命危险，但是伤得很重。”麦克米兰看着罗昊说道：“至于他们恢复后还能不能继续在特种部队服役，只能看治疗后的情况而定。”
<br/>　　“谢谢！”罗昊用力握了握麦克米兰的手，随后跟他拥抱了一下，说道：“他们能活着就是最好的，至于能不能继续在特种部队服役无所谓，其实如果能因为这次事情他们能离开特种部队，对于他们自己还是他们的家人来说，未必不是一个好的选择。”
<br/>　　“还有一件事。”麦克米兰指了指那些躺在担架上，被抬着去接受治疗，并且在后面几天中会安排送回各自国家的特战队员，说道：“我们这次只是找到了一部分被抓的人质，你们的队员也只是找到了五个，剩下的目前还不清楚被关在什么地方。西伦科特很狡猾，他把人质分开进行关押，这样不至于手中完全没有筹码。”
<br/>　　罗昊点了点头，转身走向舰桥去看望被营救出来的五个人。这次被救出来的五个人分别是刘子锦、司徒空、张子航、陈应天和林晓阳，五个人受的上都不轻，身上的伤口除了在遭到战斧导弹攻击时留下的之外，更多是遭到严刑折磨后所留下的，因为伤口没有得到及时处理加上这里天气炎热的关系，有些伤口甚至出现了溃烂的趋势，这并不是什么好情况，伤口溃烂感染对于伤员来说，就是致命的杀手。
<br/>　　“他们都会没事的。”云熙走到罗昊身边，安慰道。
<br/>　　尽管罗昊离开房间的时候叮嘱了让云熙好好休息，可是云熙放心不下罗昊，还是穿上衣服追了出来。
<br/>　　罗昊深吸一口气，随后点了点头，对云熙说道：“我突然想吃你做得早餐了。”
<br/>　　“我去跟厨房说，想吃什么，我帮你做。”
<br/>　　云熙声音轻柔，听着就像是被一阵暖洋洋的春风从心头吹过一样，非常的舒服。
<br/>　　“粥，或者馒头。”罗昊想也没想的就说道。
<br/>　　“好。”云熙轻轻点了点头，这几天在船上，吃得都是西式餐点，被罗昊这么一提及，云熙也是被勾起了馋虫。为了照顾家里那两个小的，云熙学了一手的好厨艺，在罗战和云悠还只有四岁的时候，两个小家伙能为了一个白馒头吵起来，最后云悠一哭，罗昊就会把抢走的馒头乖乖的放回到云悠面前，而作为姐姐的云悠还是会掰一半分给罗战。
<br/>　　厨房里，正在准备着早餐的厨师听到了云熙的要求后，很快就给她提供了她所需要的食材。在家的时候，云熙是变着花样做各种粥和面点，但是在这里只有白面。
<br/>　　粥是做不成了，只能做馒头了。
<br/>　　对于外国人来说，中国美食充满着神奇的魔力，各个城市各不相同的美食文化，对于西方人充满着诱惑，所以看到云熙动手做面点，厨房里那个负责管理厨房的中士停下了自己手中的工作，看着云熙做早点。
<br/>　　当云熙把蒸好的馒头拿出锅的时候，中士走上前，看着云熙手中用白面配上厨房中一些其他食材做好的馒头，说道：“长官，虽然我知道这样很不礼貌，但我还是想问您能把您做的包子留下一个吗？”
<br/>　　很多美国人是分不清馒头和包子这两个完全不同的面点的。
<br/>　　“抱歉中士。”云熙微笑着摇头，说道：“这是我为我的丈夫所准备的。”
<br/>　　中士很是遗憾的说道：“我真的很羡慕您的丈夫，他能有您这样一个妻子。”
<br/>　　云熙端着做好的馒头，在一众人羡慕的注视下，走进餐厅。
