夏夕绾一直盼着林婶苏醒，想弄清当年的真相，可是现在接二连三的听到这么劲爆性的消息，她都觉得匪夷所思了。

<br/>        爷爷是…妈咪的管家？

<br/>        这是什么意思？

<br/>        后来妈咪嫁给了管家的儿子…
<br/>
<br/>        “林婶，你每个字我都可以听懂，但是连成一句话我就听不懂了，我心里有好多疑问，想要你一一的说给我听。”
<br/>
<br/>        林婶疼爱的看着夏夕绾，“小小姐，只要你拿回小姐留给你的箱子，这一切你都会明白的。”
<br/>
<br/>        说完，林婶疲倦的闭上了眼，再度陷入了昏睡。
<br/>
<br/>        医生来给林婶检查了身体，奇怪道，“夏小姐，病人的身体其实一直不太好，但是病人好像曾经服用过什么药丸，这个药丸一直保持病人心脏不衰竭。”
<br/>
<br/>        夏夕绾刚才为林婶号了脉，就如医生说的，林婶体内有一种保命的药丸。
<br/>
<br/>        半个月前她为林婶号脉的时候竟然还没有察觉出来，能如此神不知鬼不觉的为林婶续命这么多年，这个人的医术相当了得，夏夕绾知道是自己的妈咪。
<br/>
<br/>        十年前妈咪逝去之前应该给林婶服用了这个保命的药丸。
<br/>
<br/>        夏夕绾觉得现在事情越来越扑朔迷离了，她的周围好像有一个巨大的网，将她给笼罩住了。
<br/>
<br/>        这时一串悠扬的手机铃声响起了，来电话了。
<br/>
<br/>        是夏振国打来的。
<br/>
<br/>        夏夕绾没有任何的意外，夏振国肯定会打电话给她的，询问她跟陆寒霆的事情。

<br/>        夏夕绾按键接通了电话，“喂，爸。”
<br/>
<br/>        “绾绾，你现在赶紧回家一趟，我有事要好好的问问你！”夏振国火急火燎道。
<br/>
<br/>        夏夕绾勾了一下红唇，“好啊，我正好也有事情想要跟爸说。”

<br/>        ……

<br/>
<br/>        夏夕绾又施了第二次针，老爷子年纪大了，又做了十年的植物人，需要时间来苏醒。
<br/>
<br/>        将针收起来，夏夕绾静静的看着夏老爷子，林婶说爷爷是妈咪的管家，这是什么意思？
<br/>
<br/>        林婶还说妈咪来到这座四九城后…难道妈咪不是海城人，而是从某个地方而来？

<br/>        令林婶那般惊恐颤栗的“他”又是谁？

<br/>        “爸，是不是夏妍妍跟你说了什么，夏妍妍说我刷了陆寒霆的黑金卡，夏妍妍让你来问我的？”
<br/>
<br/>        夏振国没有否认，“是妍妍告诉我的，妍妍说你手里有陆寒霆的卡，我记得那一次小蝶在帝豪六星级酒店过生日，酒店经理曾经派出劳斯莱斯送你回去，你是不是早就勾搭上陆寒霆了？”
<br/>
<br/>        “勾搭？”夏夕绾澄亮的翦瞳里溢出了几分冷笑，“爸，你也是这样说夏妍妍的么，说夏妍妍想要勾搭陆寒霆？”
