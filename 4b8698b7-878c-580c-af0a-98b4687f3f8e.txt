陆寒霆回到了party大厅，他在人群里寻找夏夕绾那道纤柔的身影，但是找了一圈，他都没有找到夏夕绾。
<br/>
<br/>        夏夕绾去哪里了？
<br/>
<br/>        陆寒霆隐隐有一种很不好的预感，他当即伸手抓住了一个侍者，“夏夕绾呢，你们有没有看到夏夕绾？”
<br/>
<br/>        “没有啊。”侍者摇了摇头。
<br/>
<br/>        陆寒霆松开了侍者，他看了一圈这个party大厅，大家有些跳舞，有些谈笑风生，各忙各的，根本就没有人注意到夏夕绾的失踪。
<br/>
<br/>        陆寒霆伸腿，一脚踹翻了一个桌子，“他妈的都给我闭嘴，夏夕绾呢，你们都眼睛瞎了没发现夏夕绾不见了？”
<br/>
<br/>        桌子倒了，上面的美食和美酒全部摔在了地上，发出了巨响。
<br/>
<br/>        轰，一声，上一秒还嘈杂的party大厅现在瞬间安静了下来，雅雀无声。
<br/>
<br/>        大家面面相觑的看着陆寒霆，陆寒霆发了好大的脾气，现在一张精致的俊脸覆着一层寒霜，几乎是阴沉的能滴出水来了。
<br/>
<br/>        夏夕绾不见了吗？
<br/>
<br/>        她不是刚刚还在这里的吗？
<br/>
<br/>        “你们还愣着干什么，快点给我去找夏夕绾，如果夏夕绾出点什么事，你们都别想逃！”陆寒霆眼角猩红的吼道。
<br/>
<br/>        大家吓得肩膀一抖，夏夕绾出事了跟他们有什么关系，这位陆总大人是要连坐罪啊，这…这也太不可理喻了。
<br/>
<br/>        不过陆寒霆现在的样子实在太吓人了，他们印象里的这位陆总大人一直是喜怒不形于色，成熟而稳重的，大家还没有见过他发脾气，现在的他紧紧的拽着拳，一双狭眸犹如漆染，深沉到可怕。
<br/>
<br/>        不能惹。
<br/>
<br/>        不敢惹。
<br/>
<br/>        不好惹。
<br/>
<br/>        大家迅速去找夏夕绾了。
<br/>
<br/>        这时有人找到了游泳池，迅速大叫道，“陆总，夏小姐在这里！不好了，夏小姐落水了！”
<br/>
<br/>        这个游泳池比较偏僻，大家又在party上谈笑风生，所以夏夕绾落水也没有引起大家的注意。
<br/>
<br/>        如果不是陆寒霆的到来搞出了这么大的动静，大家是不会发现夏夕绾的。
<br/>
<br/>        陆寒霆迅速拔开长腿走了过来，每一步都带出了强大而凌厉的腿风，他来到了游泳池边，池里的水清澈见底，他一眼就看到了下面的夏夕绾。
<br/>
<br/>        夏夕绾纤柔的身体已经沉在了水底，她身上的星空吊带长裙和那一头清纯的长发全部在水里散落开了，唇红齿白，眉眼如画的她闭着眼，整个画面绝美到妖冶，好像她就是水里的妖姬。
<br/>
<br/>        陆寒霆瞳仁骤缩，来不及思考，他直接一跃跳入了游泳池里。
<br/>
<br/>        陆寒霆身手矫健而快速的游了过去，他当即看出夏夕绾的脚被卡住了，他游过去，将她的脚拔了出来。
<br/>
<br/>        长臂伸过去，箍住了她盈盈一握的纤柔腰肢，然后带着她游了上去。
<br/>
<br/>        陆总出来了！
<br/>
<br/>        陆总将夏小姐救上来了！
<br/>
<br/>        陆寒霆浑身潮湿，他将夏夕绾抱放在了地面上，迅速有人过来探了一下夏夕绾的鼻翼。
<br/>
<br/>        夏夕绾已经没了呼吸！
