        &lt;h3&gt;第三卷第二百一十二章  礼物&lt;/h3&gt;
<br/>
<br/>        江俞轩看着大大咧咧的陈子昂，心里有那么一丝丝难过，她大概不知道他的心思吧？她也大概不知道这么多年他是用“卧底”的身份呆在她的身边吧。
<br/>
<br/>        如果陈子昂知道了他这么多年事无巨细的将她的行踪汇报给张函，陈子昂会怎么想？还会什么事情都和他商量，还会告诉他吗？
<br/>
<br/>        江俞轩那曾经意气风发神采飞扬被一种淡淡的忧伤所代替，他掩下眼里所有的情绪，看着陈子昂，他们俩人个头相差无几，只是陈子昂此时坐在他的办公椅上，而他站在办公桌旁边，他们的相处方式还是一点都没有变，在陈子昂的心中，他从来都不是江家大少。
<br/>
<br/>        “咦，你站着干嘛？一副老实的表情？”
<br/>
<br/>        “我哪有？这不是要听你指导工作吗？”
<br/>
<br/>        “装？这么几年还没有装够？”
<br/>
<br/>        “你？”
<br/>
<br/>        “这个新年我捋清楚了很多的事情，张函知道我所有的一切大概都是你说的吧？为了讨女朋友欢心，你真是费尽心思呢。”陈子昂言语犀利，不给江俞轩说话的机会。
<br/>
<br/>        “不过这些年有你的帮助我才有今天的这个地步，江大少，谢谢你。”
<br/>
<br/>        “子昂，你听我说。”江俞轩想解释。
<br/>
<br/>        “什么都不用说了，你和张倩楠准备什么时候举行婚礼？”陈子昂看着江俞轩，曾经他们在一起那么默契，如今他有了自己的幸福就祝福他吧！
<br/>
<br/>        “我们大概可能不会举行婚礼了，我们在张家已经走了一个形式。”江俞轩老实的回答。
<br/>
<br/>        陈子昂站起来，走到衣架前，拿起她的小包，从里面拿出一份文件和一块百丽翡达的手表：“俞轩，你现在是江家大少，什么都不缺，我也什么都给不了你，这份文件是你的股份，这快手表是我送你的礼物，无论你们举不举行婚礼，我都是不能参加了。”
<br/>
<br/>        “子昂，我.....”
<br/>
<br/>        “别推辞，是我的心意，我认识张函差不多十一年了，正儿八经在一起呆了三年，不可否认那三年是我最快乐的时光，可是后来，发生了很多的事情，我已经不是当初的陈子昂，他也不是当初的张函了。你说我心里有没有他，那是不可能的，我怎么会忘得了那快乐的时光呢，平淡而幸福，快乐而短暂，我知道你们都在撮合我和张函复合，可是可能吗？张函是有夫之妇啊，你们难道让我背着小三的罪名，背着破坏别人家庭的罪名吗？”
<br/>
<br/>        江俞轩无言以对。
<br/>
<br/>        “俞轩，你们想的不一定就是我想要的，我想要的是安安宁宁的生活，有没有爱情对我来说无所谓，没有爱情的生活是乏味的枯燥的，但是鸡肋式的爱情会让人心如死灰，伤痕累累。”陈子昂看着江俞轩，拉过江俞轩的手，将手表给江俞轩带上。
<br/>
<br/>        “此生，只送给一个人礼物，这个人就是你，感谢这么多年的陪伴，祝安好！”能记住我们在一起的时光吗？
<br/>
<br/>        人生苦短且漫长，我们终于是要走向不同的方向，过自己的生活走自己的路。
<br/>
<br/>        “子昂，我.....我.....”江俞轩似有千言万语说不出口，在平时那张叱咤风云的嘴，此时一个字也说不出来。
<br/>
<br/>        “好了，我们现在该去见乔小东了，那个人时间观念比较重，还是早点过去，西城交通你是知道的，堵车严重。”陈子昂将车钥匙交到江俞轩的手中。
<br/>
<br/>        “你开车，让我在车上休息会，我从泉城开回来的。”她穿上大衣，拿起包转身就离开了江俞轩的办公室。
<br/>
<br/>        江俞轩看着手中的腕表和车钥匙，长长的叹了一口气，陈子昂，你的心思谁能摸得透？
<br/>
<br/>        陈子昂快步走出江俞轩的办公室，来到广场前，郁郁葱葱的绿植在冬天散发着蓬勃的生命力，只是西北风呼啸而来，刮起陈子昂那三千青丝在寒风中舞动着。
<br/>
<br/>        江俞轩开车到陈子昂的身边，看着陈子昂在寒风中那消瘦的身影飞舞的青丝，眼眶发热。这么多年以来一直是他陪在她的身边，看着她成长，陪着她成长，如今他们真的是要各奔东西了。
<br/>
<br/>        “这几天跑来跑去的，感觉比平时还累，真的是老了。”陈子昂上车，打开车里的音箱，放了一首歌：“如果你愿意一层一层，一层的剥开我的心，你会发现你会讶异，你是我最压抑最深处的秘密，如果你愿意一层一层，一层的剥开我的心，你会鼻酸你会流泪，只要你能听到我看到我的全心全意。”
<br/>
<br/>        陈子昂闭上眼睛，静静的靠在椅背上，那首“洋葱”就那样反复循环着直到他们到达约定吃饭的地方。
<br/>
<br/>        乔小东见到陈子昂哈哈大笑：“你们两人还真是形影不离，我又没有叫你，你跑来干什么？”
<br/>
<br/>        陈子昂：“有喝酒的这等好事，不叫我，你太不够朋友了吧。”
<br/>
<br/>        乔小东：“谁知道你回来了呢。”
<br/>
<br/>        江俞轩：“子昂知道我喝不了酒，特意从北京赶回来陪你喝酒。”
<br/>
<br/>        乔小东：“吹吧你，中午我给你打的电话，晚上她就从北京回来了？”
<br/>
<br/>        陈子昂：“我从泉城赶回来的，行了吧。”
<br/>
<br/>        乔小东：“这还差不多。”
<br/>
<br/>        乔小东心里藏着事情，吃饭是小事，八卦是大事，见到陈子昂和江俞轩一起，更是好奇俩个人为什么孩子都有了，江俞轩和陈子昂还是单身状态，他可是从网上用功的找了一番俩个人的资料。
<br/>
<br/>        江俞轩喝不了酒，而且他又开车，于是就乔小东和陈子昂俩个人喝起来了，乔小东看着江俞轩：“江总，男人多少还是得喝点酒，总不能让老婆天天喝酒应酬哦。”
<br/>
<br/>        陈子昂：“他三杯倒，还不如不让他喝，就算是我喝醉了还有人照顾我。”
<br/>
<br/>        乔小东：“哈哈哈，你们俩个还真是有意思，孩子没有带回来吧？”
<br/>
<br/>        陈子昂：“孩子在北京呢。”
<br/>
<br/>        乔小东：“羡慕你们夫妻俩这种生活啊。”
<br/>
<br/>        陈子昂终于注意到乔小东的话了：“我们夫妻俩？夫子你搞错状况了哦。”
<br/>
<br/>        乔小东：“不是，你们孩子都有了。”
<br/>
<br/>        陈子昂：“原来今天夫子是想听故事，那就把酒满上，有酒没故事怎么行呢？”
<br/>
<br/>        陈子昂也就没有隐瞒乔小东，将她这半生的故事细细的讲给乔小东听，听到夏颜的时候，乔小东气愤的说：“如此不知廉耻，还好意思招摇过市？从此有我的地方就没有她夏颜，有夏颜的地方就没有我，道德败坏之人，不配与人为伍！”
<br/>
<br/>        陈子昂：“夫子也别激动，事情已经过去了，俞轩上次也是为了给我解围，倒是让大家误会了。我和俞轩就是朋友，承蒙俞轩多年关照，子昂感激不尽！”
<br/>
<br/>        故事没有结束，酒已经恰到好处，告别了乔小东，江俞轩又开车将陈子昂送回西城的家，在陈子昂的楼下他在车上坐了一晚，陈子昂看着楼下隐在黑暗中的车辆，轻轻的拉上窗帘，隔绝了外面的世界。
