　　第36章
<br/>　　这天早上，程大雷如往常一样，起床后先去菜地看了看，辣椒长势喜人。随后又看了看山寨养的猪和马，几只鸡仔咯咯在地上啄着小虫。
<br/>　　一切风平浪静，和过去的每一天一样。
<br/>　　不过，不知怎地，程大雷总觉得心绪不宁，左眼皮砰砰直跳，总觉得有事情要发生。
<br/>　　“大当家，大事不好，大事不好……”
<br/>　　正这个时候，徐神机急匆匆跑来，跑得气喘吁吁，上气不接下气。
<br/>　　“出什么事了？”程大雷问。
<br/>　　“大当家，朱家寨被人灭了！”徐神机脸色苍白，没有半点血色。
<br/>　　“那家山寨动的手？”
<br/>　　“你去看看吧，你去看看就知道了，朱家寨上下三十几口都被杀了。”
<br/>　　“走，带我去看看！”
<br/>　　徐神机脚步匆匆，在前面引路，中途又碰上秦蛮和林少羽。李行哉从屋里出来，眼前蛤蟆寨全体都出动了，当然，蛤蟆寨一共就六个人。他也不知道什么事，也跟在大家后面。
<br/>　　到朱家寨就知道了，程大雷之前闻到火药味是假，可如今的血腥味却是扑面而来。
<br/>　　在一片山坡之上，到处倒卧的都是尸体，一具具倒在血泊中，绿草上挂着血珠。上到八十老妇，下到黄口小儿，程大雷看到草丛中卧着一具孩子的尸体，看模样还不到五六岁。
<br/>　　如今每具尸体都被砍掉头，只有尸体倒在地上。
<br/>　　他下意识捂住嘴巴，喉咙中发出一声干呕。
<br/>　　“怎么，没见过死人呐？”
<br/>　　一个声音突然响起，高飞虎黑着一张脸，瞪了程大雷一眼。
<br/>　　今天来到朱家寨的，不止蛤蟆寨一家，飞虎寨，杏花岭，秃头坡，狗头岭等青牛山数得上来的山寨头目都到了。
<br/>　　程大雷没有理睬高飞虎的讽刺，依旧捂着嘴巴。
<br/>　　“喂，是那家那寨动的手，大家都在青牛山吃饭，没必要把事情做这么绝吧？”小白狼突然道。
<br/>　　“还特么没看出来嘛，不是我们动的手，是外人！”高飞豹大叫。
<br/>　　“谁？”程大雷问。
<br/>　　飞虎寨的于虬然道：“黑石城，陆哼。”
<br/>　　程大雷还没见过他，问：“这位是？”
<br/>　　“程大家，在下飞虎寨军师于虬然，还是第一次和程当家见面。”
<br/>　　程大雷道：“于军师怎么知道是陆哼动得手？”
<br/>　　“我们来得早，在朱家寨发现一封血书，上面留着陆哼的名字，上面还说，还说……”
<br/>　　“还说什么，你快说呀？”秃头坡的熊大问。
<br/>　　“他还说，三天后要从青牛山过，要青牛山有一个算一个，在山下迎接！”
<br/>　　血书从于虬然身上拿出来，挨着给大家传看。黄三元看过，是陆哼的字迹，证明这封血书不假。
<br/>　　众山贼中有认字的，看过之后，脸都沉得像冰，如今青牛山五十多家山寨的当家都在这里，黑压压一片，却是没有任何人说话。每个人都黑着脸，看着地上的死人，都有种兔死狐悲的感觉。
<br/>　　这段时间青牛山草木皆兵，各家山寨都在想着如何吞了薛半川这批寿礼，也在琢磨着陆哼会如何应对青牛山的山寨，无非是兵来将敌水来土堰而已。
<br/>　　可谁也没有想到，陆哼根本没有与大家兵来将敌水来土堰的意思。他只用了一招……杀鸡儆猴。
<br/>　　连夜奔袭，血屠朱家寨，可偌大一个青牛山，五十多股山贼，却没有一家听到半点动静。杀人而来，杀人归，陆哼就是要用这种手段告诉所有人。他平得了朱家寨，也平得了青牛山任何一家山寨。
<br/>　　想到这里，谁不是噤若寒蝉，感觉头顶悬着一柄刀，随时都会落下。
<br/>　　本来一帮人都吵吵着如何打劫陆哼，磨刀霍霍，还生怕别人抢了先。可如今，所有人都偃旗息鼓，没人心里敢再动这个念头。
<br/>　　这个时候，有人跌跌撞撞跑进朱家寨，手里还拎着一只野兔。
<br/>　　程大雷还真认得这个人，他是朱家寨的朱小朋，前几天还从蛤蟆寨借走一些辣椒面，二人见过一面。
<br/>　　朱小朋昨晚在山里打猎，侥幸躲过一劫，可见到山寨的惨状，他瞬间崩溃，噗通一声倒在地上，嚎哭不止。
<br/>　　这哭声响彻在众人耳边，每个人心里都不太好受。朱家寨一共三十多口人，有老人有小孩，平素也不下山打劫，关键是人数太少，说不好打劫不成反被人劫。而今天，朱家寨三十多口却被屠得干干净净，包括小孩，老人，女人……
<br/>　　朱小朋突然站起来，慢慢走到程大雷面前。
<br/>　　“程当家，欠你的债我是还不了了，您别往心里去。”
<br/>　　“不算什么，你好好活着。”事到如今，程大雷也只能说这些话。
<br/>　　“我爹死了，我娘死了，我的女人和孩子都死了，我还怎么活。”
<br/>　　说着，朱小朋忽然大吼一声，狠狠的向一块大石撞去，程大雷反应慢了半拍，朱小朋已经撞破脑袋，尸体正倒在程大雷面前。
<br/>　　程大雷猛地呆住，不敢相信眼前发生的这一幕。
<br/>　　“喂，大家都搭把手，把人埋了。”高飞豹喊道。
<br/>　　几家山寨的头目用刀挖土，忽然抬着将朱家寨的尸体埋进地里。
<br/>　　程大雷挥挥手，驱散鼻尖的血腥，有气无力道：“走吧，我们回去。”
<br/>　　“喂，你这个人怎么这么没人性，就一点不管嘛。”高飞豹道。
<br/>　　程大雷没有回头，挥挥手带着人回了蛤蟆寨。回到蛤蟆寨，程大雷就病倒了，窝在屋里，冷汗一层一层往外冒。
<br/>　　“大当家，你没有大碍吧？”徐灵儿给程大雷煮了一碗姜汤，捧着走进屋里。
<br/>　　程大雷摇摇头，手捧着姜汤，咕咚咚咽了一口，一时手腕发颤，哐当一声，粗瓷碗摔在地上，砸了个稀巴烂。
<br/>　　“大当家……”
<br/>　　程大雷挥挥手，不让徐灵儿靠前，眼睛呆呆望着地上的瓷片发愣。
<br/>　　徐灵儿立在一旁，也不知道该怎么办，只好道：“大当家，你还好吧，不行就早早歇着。”
<br/>　　程大雷让徐灵儿把地上的残渣收拾了，抬脚上了炕，半晌，他望着窗外，说了回到蛤蟆寨后的第一句话。
<br/>　　“山贼的命……还真是比狗都贱呐。”
<br/>　　
<br/>(本章完)
