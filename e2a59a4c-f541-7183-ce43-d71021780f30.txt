        &lt;h3&gt;第三卷第一百八十五章  宁雅在韩国&lt;/h3&gt;
<br/>
<br/>        宁雅的韩国之行收获满满，三个人同行她就是百瓦电灯泡，明晃晃的照耀着胡婷和狸猫，宁雅已经见惯了胡婷和狸猫撒狗粮的方式，对他们有时候是会视而不见听而不闻。
<br/>
<br/>        狸猫陪着妻子逛了几天，因为手中存稿不多，便和胡婷商量着：“你明天和宁雅一起出去逛逛，我码上一天字？”
<br/>
<br/>        “天天码字，你烦不烦啊？出来了还不得闲。”
<br/>
<br/>        “我不是怕人寄刀片吗？”
<br/>
<br/>        “难道你就不怕我有刀片？”
<br/>
<br/>        “你干嘛要给我刀片啊，你又不看我的书。”狸猫笑着说。
<br/>
<br/>        “哼，要不是看在你想要造小栖桐的份上，我早都给你准备上刀片了。”
<br/>
<br/>        “老婆，造人是小事，你得为你以后的幸福生活着想啊，以后千万记得不要再提刀片的事情了。好了，你就陪着宁雅逛一逛，你看看我这样要真是有读者给我寄刀片，我接还是不接啊？”
<br/>
<br/>        胡婷不情不愿的去找宁雅：“小雅，你说狸猫讨厌不讨厌？出来了还不闲着。”
<br/>
<br/>        宁雅：“你这天天缠人的毛病是得改一改，人家做的也是正经事情，你怎不能托人家后腿吧。”
<br/>
<br/>        “我怎么就拖后腿了，也就放假这几天，他还要码字。我还不是想和他多呆点时间。”胡婷表示很委屈。
<br/>
<br/>        “好了，好了，我不是陪着你吗？走我们去免税店看看去，这不马上就要回家了，看看能不能扫一些货回去。”
<br/>
<br/>        胡婷听说要扫货立马多云转晴，拉着宁雅的手俩个人去了扫货了，再呆两天他们就可以返程。
<br/>
<br/>        “小雅，这几天凌辰一直都没有联系你吗？”胡婷边看手中的化妆品边问宁雅，那次狸猫骂了她，她也觉得那件事情做得有些过。
<br/>
<br/>        “没有，想了一想也还真是那次你接电话之后就没有了消息，想一想这人啊真的是经不起考验。”宁雅叹了口气。
<br/>
<br/>        “你就告诉我，你到底爱不爱凌辰？”胡婷心里那个愧疚啊。
<br/>
<br/>        “怎么说呢？这么多年，心里一直有他的存在，因为有了他我拒绝和任何人交往，这个算不算是爱？”宁雅反问胡婷。
<br/>
<br/>        “这个当然是爱啊，不然你怎么会将他放在心上呢？”胡婷突然就闭口了，她真是猪队友。
<br/>
<br/>        “先不说这个事情了，回去后要好好工作，我这些天都想明白了，女人啊，无论你出生时是什么样，一定要自强自立，就像陈子昂一样努力的改变自己的命运。”宁雅有些想念陈子昂了，和陈子昂在一起总是给人一种积极向上上的感觉。
<br/>
<br/>        宁雅每次去送文件的时候，就静静的站在陈子昂的旁边看着她，她永远都是一头长发，偶尔扎起来，偶尔柔顺的披散着，看文件的时候眉眼低垂，长长的睫毛呼闪着，侧面看起来线条柔和，下巴微微的翘着，神情专注而认真，抬起头时，那美丽的丹凤眼闪着凌凌波光，似乎能让人陷进去。
<br/>
<br/>        “想什么呢？”胡婷见宁雅一副沉思的模样。
<br/>
<br/>        “我在想陈子昂呢。”宁雅笑着说。
<br/>
<br/>        “我的天，你这个时候不是应该想你和凌辰以后的事情吗？你怎么会去想一个女人呢？”胡婷大惊小怪的喊。
<br/>
<br/>        “我怎么就不能想一个女人了？难道你不是女人？我有时候也会想你的啊。”
<br/>
<br/>        “明白了，你就说你和凌辰有没有可能？如果有可能了我哪怕飞到山西去找他解释都行。”
<br/>
<br/>        “俩个人的感情无需第三方干涉，有时候别人干涉反而适得其反，你也不要过多的内疚，我如果和凌辰有缘分的话，总会有解开误会的那一天，没有缘分我和他不是还是同学吗？反正这一段时间也没有时间来谈情说爱的，话说谈恋爱也是需要付出成本的。”
<br/>
<br/>        “需要什么成本啊？”胡婷不明白了，好像她和狸猫没有要什么成本啊，一切都是水到渠成。
<br/>
<br/>        “物质成本啊，俩个人在一起不需要花钱吗？时间成本啊，俩个人谈恋爱不需要时间吗？时间用来赚钱不香吗？用到哪些无聊的卿卿我我上，有结果还好说，没结果浪费精力还伤心。”
<br/>
<br/>        “艾玛，小雅，你什么时候变得这么现实了？受什么刺激了？”胡婷踮起脚尖伸手摸了摸宁雅的额头。“没发烧啊！”
<br/>
<br/>        “不是我现实啊，我和凌辰俩个人这么多年了，物质就不说了，他不缺钱，而我也不缺钱，偶尔，过年过节他会给我买一些东西要么送给我，要么寄给我，这个不花钱吗？还有俩个人离得很远，他到处飞，见面的次数也不多，就是打打电话，聊聊天，说实话有时候非常想找他肩膀靠一下都不行。感情是有了，可靠吗？这么几年还不是一场空。”宁雅的语气有些忧伤。
<br/>
<br/>        “你说的也对，要是你和凌辰离得近，估计也成了吧？”
<br/>
<br/>        “也不一定的。”她的父亲看中的是门当户对。
<br/>
<br/>        “小雅，你心里是不是还在怪我？”
<br/>
<br/>        “我有那么玻璃心吗，走吧，回去看看你的狸猫怎么样了？”
<br/>
<br/>        “小雅，狸猫怕我给你添乱，所以他不让我和你一起工作。”
<br/>
<br/>        “你就这样的窝在家里？”
<br/>
<br/>        “我想一想，如果现在要孩子的话，我怕是不能工作了。”
<br/>
<br/>        “那就要孩子吧，你们家你不工作也可以，而且你和狸猫相亲相爱的。”
<br/>
<br/>        狸猫忽悠着胡婷和宁雅出去了，打开电脑，他写了会文章，思路总是有点乱，刚巧也是写到男女主感情纠葛上，他停下来，想到了凌辰和宁雅，俩个人的感情这几年大家都心知肚明，可是俩个人谁都不愿意向对方前进一步，躲在自己的壳里，如今被她老婆一搅和，怕又是困难重重，不如他联系一下凌辰，看看是个什么情况，如果凌辰真有女朋友了，那现在这样的状况就是最好的结局，如果没有女朋友了，还是想办法让俩个人解除误会为好。
<br/>
<br/>        狸猫打开qq给凌辰发了个信息：“听说你小子又女朋友了什么时候带出来晒一晒？”
<br/>
<br/>        凌辰：“正在饱尝失恋的滋味，哪里来的女朋友？”
<br/>
<br/>        狸猫：“这是还没有开始就失恋了？”
<br/>
<br/>        凌辰：“宁雅有男朋友了。”
<br/>
<br/>        狸猫：“谁告诉你的？”
<br/>
<br/>        凌辰：“.......”他觉得他不知道该如何给狸猫解释那件事情。
<br/>
<br/>        狸猫：“宁雅和我这几天在韩国，后天回国，当然还有胡婷。”
<br/>
<br/>        凌辰：“宁雅没有带男朋友？”
<br/>
<br/>        狸猫：“你哪只眼睛看到宁雅有男朋友了？”
<br/>
<br/>        凌辰：“你说的是真的？”
<br/>
<br/>        狸猫：“滚犊子，不想理你了。”
<br/>
<br/>        狸猫关了qq，一个个的都不让人省心，我好难啊！
