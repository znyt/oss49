　　第523章
<br/>　　徐神机的存在，一直是山寨的大BUG。在山寨所有军师中，他永远是排在首位的第一军师。但对于山寨的发展，他几乎没发挥过任何作用。
<br/>　　说也奇怪，徐神机也不是一点脑子没有，况且经历过这么多事，就算块木头也该甩开窍了。关于山寨发展，徐神机也层出不穷的出过很多主意，有些时候，听上去还很有道理，程大雷都会被说服。
<br/>　　然而，一旦具体实施起来，就绝对和徐神机想得不一样。
<br/>　　这么多年过去，徐神机算无一策的属性有增无减，甚至有愈演愈烈的趋势。
<br/>　　程大雷每次出去做事，都将他带在身边，虽然看在眼里有些烦人，但终究比留他在家里祸害山寨好。
<br/>　　这次出门远行，程大雷也同样想将徐神机带在身边。然而，关鱼的身体状况十分糟糕，那华氏一族究竟能不能寻到还是个问题，总之是路上越快越好。
<br/>　　程大雷骑着黑牛，单枪匹马出发，以黑牛的脚力，赶到东海也浪费不了太多时间。但假若是带上徐神机，速度自然是快不了。可将他留在山寨，程大雷又实在放心不下。
<br/>　　“大当家，你找我？”
<br/>　　这个时候，徐神机推开书房的门走了进来。
<br/>　　程大雷收回心思，点点头道：“你来的正好，我就是有件大事要和你商量。”
<br/>　　“大当家尽管说，究竟有怎样的事拿不定主意，由我帮大当家参谋参谋。”
<br/>　　“嗯呐，你来了我就放心了。”
<br/>　　徐神机昂起头，笑眯眯道：“究竟是什么事呐？”
<br/>　　“走。”程大雷站起身：“咱们到地方聊。”
<br/>　　徐神机稀里糊涂，但因为被程大雷捧得有些找不着北，也没有太关心这件事。二人一前一后离开了书房，黑衣尼姑鱼洞真在后面跟着。
<br/>　　对于鱼洞真的存在，徐神机没有太当会事，只不过随口问了一句。
<br/>　　“大当家，她是怎么会事？”
<br/>　　“喔，江湖上的高手，慕名而来，你不用管她。”
<br/>　　程大雷也算是明白了，这黑尼姑就是个机器人，你让她干什么就干什么，不让她做什么她就什么都不做。而且，这黑尼姑还有个珍贵的品质，那就是话少。
<br/>　　徐神机离开城主府，看方向是去城中的大牢。
<br/>　　“大当家，咱们是要去见什么人么？”
<br/>　　“喔，不是，马上就要到了，你到了不就知道了么。”
<br/>　　一路来到地牢，目前牢里也没关什么人。程大雷振振肩膀，冲徐神机道：
<br/>　　“你看，这里环境怎样？”
<br/>　　“很不错啊。”徐神机感慨道：“咱们的大牢也没什么犯人，都干净得很，牢头都每天闲得晒太阳。”
<br/>　　“你满意就很好。”
<br/>　　“哈哈……咦……”徐神机眼睛睁大：“什么叫我满意就很好。”
<br/>　　程大雷一把将徐神机推进一间牢房中，瞬间将牢门锁上。
<br/>　　徐神机一头雾水，道：“大当家，你这是做什么啊。”
<br/>　　程大雷松了口气，道：“你以后就住在这里了，我什么时候回来，你什么时候出来。”
<br/>　　说着，程大雷转向鱼洞真，道：“以后你在这里看着他，不能让他离开牢房半步，明白么？”
<br/>　　鱼洞真双手合十，轻轻点头。
<br/>　　虽然她一句话没说，但很明显比那些说了千言万语的人还要靠谱。程大雷相信，她一定会将自己的命令，一五一十的执行下去。
<br/>　　程大雷想来想去，也就想到这个法子。不让徐神机给山寨添乱，就只有将他关起来，不然的话，还得带着他走这场路，他的老胳膊老腿，怕也经不起这场折腾。
<br/>　　徐神机真的懵了：“大当家，你不要开玩笑了，快快放我出去。”
<br/>　　“好啦，你都说了，这里环境很好的，每天衣食住行都有人照顾，最多半年，我就回来啦。”
<br/>　　“可这究竟是为什么啊！”
<br/>　　“徐军师！”程大雷拉长声音：“你一直是山寨的第一军师，论智谋，论算计谁比得上你。像你这种有天赋的人，就该有更高的追求，山寨第一军师就能满足你么，你要放眼全国，争取成为帝国第一军师。”
<br/>　　“呃……我有希望么？”
<br/>　　“不要妄自菲薄，以你的天赋，绝对可以取得更高的成就，千万不可浪费掉。”程大雷冲徐神机握紧拳头：“我看好你喔。”
<br/>　　“大当家是要我……”徐神机睁大眼睛：“闭关？”
<br/>　　“对，你好好闭关思考一下，等我回来，且看你能成长到哪一步。”
<br/>　　说完这些话，程大雷神清气爽的离开了大牢。总觉得，自己这条计策是绝对万无一失的。
<br/>　　关鱼的身体等不及，程大雷必须连夜出发，早一天总有早一天的好处。程大雷回到城主府，苏樱已经帮他收拾好行李。
<br/>　　这次没办法携带鬼面斧，鬼面斧太过沉重，出远门携带不便。而鬼面斧的目标又大，现在天下谁不知道自己是骑黑牛，背斧头的，出门太拉仇恨值。
<br/>　　他无非是换了一套便装，打扮成儒生公子的模样，腰上挂着匹夫剑。
<br/>　　当然，就是长得不像儒生公子而已。
<br/>　　原先的时候，程大雷还有些清秀，可这些年是越长越糙了。本以为自己的人设就算不是诸葛亮，也是俏周郎，但现在感觉越来越像李逵了。
<br/>　　和珅一行人将程大雷送到城门口。
<br/>　　“大当家，这一路去风雨坎坷，你可要保重身体。”
<br/>　　“好啦，好啦。”程大雷骑在牛背上挥挥手：“知道啦，你们回去吧，我在外面耽搁不了什么时间，找到人我就回来了。”
<br/>　　“徐军师呢？”和珅奇怪道。程大雷和徐神机向来形影不离，这次出门，怎么舍得不将徐神机带在身边。况且，程大雷要走了，徐神机怎么送都不送。
<br/>　　“喔，对了。”程大雷道：“你们有所不知，徐军师最近若有所悟，要闭关潜修，你们没事的时候，可千万不要去打扰徐军师，别误了徐军师参悟大道。”
<br/>　　“明白，明白。”
<br/>　　和珅等人忙不迭的点头答应。大家都知道徐军师的智慧，心里也很是佩服。徐军师既然有这么重要的事，大家怎么敢去打扰。
<br/>(本章完)
