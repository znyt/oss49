　　“好快的速度。”胡海吓了一跳。
<br/>　　当他感受到脖子上传来的冰凉感觉时，更是面露恐慌。
<br/>　　“你...你...小子，识相地便放下你的剑，否则，我一声令下，这数十武者足以将你击杀。”胡海威胁道。
<br/>　　萧逸冷笑一声，手中利剑稍稍在胡海脖子上一划，顿时出现一条血线。
<br/>　　“你倒说说看，怎么杀我？”
<br/>　　萧逸的紫云精铁剑，只需轻轻一动，立刻就可以要了这胡海的小命。
<br/>　　胡海感受到脖子的疼痛，惊恐道，“别...别杀我。大不了，我不要你的分数，放你走就是。”
<br/>　　这时，那数十原本围攻林劲和铁牛的武者，见胡海有危险，纷纷停下了攻击，而后围拢到萧逸和胡海身边。
<br/>　　林劲和铁牛喘了口气，对女子问道，“嫣然，你认识这个少年？”
<br/>　　女子摇了摇头。
<br/>　　另一边，萧逸架着胡海，挪揄道，“哦？照你说来，我应该放了你，还要谢你的不杀之恩了？”
<br/>　　说着，紫云精铁剑再度在胡海的脖子上划出一条血线。
<br/>　　胡海吃疼，连声道，“不敢不敢，只是...”
<br/>　　“废话少说。”萧逸冷声道，“我好不容易睡个觉，被你一掌震下来了...”
<br/>　　胡海打断道，“我那一掌，打的不是你阿。”
<br/>　　“什么？”萧逸冷声道，“你的意思是，要我一个堂堂大男人去和一个弱质芊芊的女子计较？你在看不起我是不是？”
<br/>　　“不是不是。”
<br/>　　原本嚣张无比的胡海，如今哭丧着嘴脸。
<br/>　　“哼，谅你也不敢。”萧逸冷声道，“把你令牌上的分数交出来，这事便算了，否则，我倒要看看你的脖子能挨我几剑。”
<br/>　　“分数？”胡海脸色一变，道，“不可能。小子，你休要威胁我。你知不知道我是何人？我乃是长风公子麾下的，你...”
<br/>　　萧逸没有再废话，嘶的一声，胡海脖子上又多一条血痕。
<br/>　　嘶的一声，又是一条血痕。
<br/>　　嘶...嘶...嘶....
<br/>　　数秒过后，鲜血染红了胡海整条脖子。
<br/>　　“别划了，我将分数给你便是。”胡海连声求饶，他毫不怀疑，若自己再敢废话，那少年会杀了他。
<br/>　　只见胡海拿出令牌，交给了萧逸，令牌上写着80。
<br/>　　萧逸也拿出了自己原本的令牌，只不过，令牌上写得是1。
<br/>　　两块令牌重叠，一阵气息涌动。
<br/>　　而后，萧逸的令牌变成了81，胡海的令牌变成了0。
<br/>　　“早这样便不用受苦了。”萧逸淡淡一笑，将令牌还给胡海，而后收剑离开。
<br/>　　胡海脱离危险，竟立刻后退，回到那数十武者身旁，恶狠狠地说道，“别让那小子跑了，他抢了我们整整80点分数。”
<br/>　　数十武者，立刻蜂拥而上。脸色狰狞道，“敢抢我们的分数，真是找死。”
<br/>　　“找死。”萧逸脸色一冷，手中紫云精铁剑舞出一道道剑花。
<br/>　　一道道剑罡凭空而现。
<br/>　　正是清风剑罡。
<br/>　　自己来裂天剑派，是想学剑术的，便使出了清风剑罡。
<br/>　　对比控火兽武魂，就算进阶成绿色了，也远远比不上冰鸾剑武魂，那才是自己最大的底牌。
<br/>　　毫无疑问，剑术和剑技，才是最能提升自己实力的东西。
<br/>　　而且，自己的紫炎，还有火属性武技，都是易霄的招牌，轻易使用不得，免得招来不必要的麻烦。
<br/>　　再者，虽说自己只有先天三重修为，但哪怕只靠真气，战力也不逊色于先天六重。
<br/>　　配合玄阶中级剑法，清风剑罡，收拾这群家伙轻而易举。
<br/>　　同一时间，一旁的林劲等人见萧逸被围攻，立刻道，“快去助他。”
<br/>　　三人瞬间出手，来到萧逸身边，一并杀敌。
<br/>　　“狂刀斩。”林劲大喝一声，一道巨型刀刃劈下，瞬间秒杀数人。
<br/>　　“地动山摇。”铁牛那魁梧的身躯，一把捉起数人，力量巨大，将那数人按在地面，猛地一顿轰。
<br/>　　轰得地面摇晃，泥土龟裂，那数人淹没在泥土中，只有出的气，没有进的气。
<br/>　　“柳絮千条。”女子也轻喝了一声，一阵绿光闪过，千条柳絮凭空而现，将十数人捆绑起来。
<br/>　　虽无法击杀这十数人，却也让他们动弹不得。
<br/>　　这时，萧逸一道道清风剑罡劈过，结束了他们的性命。
<br/>　　林劲三人，本来就强，只不过之前以少敌多，才落入了下风。
<br/>　　有了萧逸的分担后，三人的战力展露无遗。
<br/>　　不过几分钟，原本猖狂无比的数十武者尽皆死亡，只剩下胡海一人。
<br/>　　萧逸一剑刺出。
<br/>　　林劲立刻喝道，“小兄弟，不要杀他。”
<br/>　　然而，迟了，萧逸的剑，已经刺穿了胡海的喉咙。
<br/>　　萧逸眉头一皱，推开胡海的尸体，道，“这等小人，杀了便杀了，免得日后多增后患。我看此人的言语，也是个心肠歹毒之人。”
<br/>　　“哎。”林劲叹了口气。
<br/>　　而后，三人都来到萧逸面前，拱了拱手，道，“谢过刚才的救命之恩了。”
<br/>　　“没事。”萧逸摆摆手，转身便走。
<br/>　　“慢着。”林劲叫住了萧逸。
<br/>　　“有事？”萧逸微微转过了头。
<br/>　　“你就这样走了？”林劲疑惑道，“不选择加入我们吗？我们联手，安全得多，也能抢得更多分数。”
<br/>　　萧逸淡淡道，“不必了，我更想靠自己的实力来通过考核。”
<br/>　　林劲劝道，“兄台，最近几日开始，森林里危机四伏，一个个落单的武者都吃了大亏，被别人联手起来抢了分数。”
<br/>　　“你这样单独行动，危险得很。”
<br/>　　“哦？”萧逸一愣，道，“怎么，很多人都联手了吗？”
<br/>　　“阿？”这下轮到林劲一愣，道，“兄台不知吗？几乎九成九以上的武者都联合起来了。”
<br/>　　“是吗？”萧逸面露疑惑。
<br/>　　一旁的铁牛嗡声道，“兄台你这几日干什么去了？这事没人不知道吧。”
<br/>　　“额。”萧逸脸色微微尴尬。
<br/>　　女子似是想起了什么，捂嘴笑道，“你该不会是这几日都在睡觉吧。”
<br/>　　萧逸撇撇嘴，不语。
<br/>　　林劲见状，解释道，“这次的考核，除了分数前五百者通过外，前十名，还有额外的奖励，排名越前，奖励越高。”
<br/>　　“有些家伙，为了确保自己能得到这些奖励，便聚集了大量的武者为其效命，疯狂掠夺其他武者的分数，好让自己的分数足够高。”
<br/>　　“像那长风公子，本身就北山榜前二十的有名天才。凭着自己身的名气和实力，麾下足有数百武者为他效命，分数每日都在疯狂暴涨。”
<br/>　　萧逸闻言，瞬间明白过来。
<br/>　　这次的考核，毕竟不是只有一人通过，而是前五百者。
<br/>　　有些实力低弱的武者，本身没有绝对的把握。那么，联合起来，或者背靠大树，是最好的选择。
<br/>　　他们不求能得到前十的奖励，只求通过。
<br/>　　这时，有长风公子，或者其他有名的武者牵头的话，他们会马上靠拢。
<br/>　　只要抢光了其他参加考核的武者的分数，那么，就算他们只有一分、两分，也足以通过考核了。
<br/>　　林劲继续说道，“现在森林内，有十多个这样的团伙。”
<br/>　　“很多单独行走的武者，都吃了大亏，迫不得已下，都在被这些团伙追杀的过程中，互相联手起来。”
<br/>　　事实上，这几天森林内的战斗骤然变得激烈，就是因为这些团伙疯狂地袭击个人武者。
<br/>　　“像我们三个，就是昨天在追杀途中相互认识的，并暂时联手。分数靠自己实力得到，敌人是谁打败的，便归谁。”
<br/>　　“怎么样，兄台，为了安全起见，还是加入我们把。”林劲认真地说道。
<br/>　　一旁的女子也柔声道，“兄台，一起吧，那些团伙很过分的，动辄杀人，你单独行动，会很危险。”
<br/>　　“危险？”萧逸淡淡一笑，道，“也罢，反正在森林里呆一个月也是无聊，就一起吧。”
<br/>　　林劲三人顿时一喜，他们都是心高气傲之辈，不屑去加入那些团伙，想靠自己的实力通过考核，但又不希望被那些团伙趁人之危，只能暂时联合起来。
<br/>　　如今，多了萧逸这个强大的战力，安全性大大提高。
<br/>　　林劲道，“兄台，我们三人也是昨日才认识，彼此都不是很了解，不如趁这个机会，再互相介绍一番吧。”
<br/>　　“我先来。”林劲道，“我叫林劲，18岁，先天七重修为，绿色武魂，玄铁重刀。”
<br/>　　铁牛道，“我叫铁牛，18岁，先天六重修为，绿色武魂，撼地神牛。”
<br/>　　女子道，“我叫柳嫣然，16岁，先天六重，绿色武魂，烟雨碧柳。”
<br/>　　三人看向萧逸。
<br/>　　萧逸淡淡道，“萧逸，16，先天三重。武魂嘛，下次再告诉你们。”
<br/>　　“阿？你耍赖。”三人表情一瘪。
<br/>　　恰在此时，远处传来一阵激烈的战斗声响。
<br/>　　听其声音，战斗的人数，起码在百人以上。
