        “我让你帮我查那个‘连少’是什么来头，你查到了没有？”隔着一扇灰不溜秋的铁栅栏门，季运恒坐在里面，双手抱臂，没好气地问。
<br/>
<br/>        季思妍甩了甩手，同样也没好气地回答：“你自己都查不清对方底细，你让我上哪里查去？再说了，你可是这滨江城土生土长的，混了么多年，居然搞不定一个外地人！”
<br/>
<br/>        她的这位大哥，以前在任何夜店里都是霸王，只有他欺负别人的份，哪能轮到别人欺压到他头上来。
<br/>
<br/>        而季运恒自己在外边干的那些龌龊事，当然没脸拿回家去炫耀。
<br/>
<br/>        倘若他告诉自己的妹妹，他这次是为了夜店里的一个歌女，才跟别人打的群架，而且还未打赢，那岂不是真的丢脸丢到家了！
<br/>
<br/>        “哎呀！你别废话了！快点和老爸一起想想办法，让我早点出去！我可不想在这破地方被关十五天！”季运恒恼火道。
<br/>
<br/>        季思妍抿了抿唇，转身去了坐在那边值班的两个男警员那里，偷偷地从手提包里拿出了两包高档香烟，塞到了这两个警员的办公桌下的抽屉里，希望他们多多关照一下她哥哥。
<br/>
<br/>        然而，两个小警员却是中规中矩，将季思妍强行塞到他们两抽屉里的香烟又拿了出来，还给了季思妍。
<br/>
<br/>        “季大小姐就先回去吧！你留在这里，只会影响我们办案！”男警员劝说道。
<br/>
<br/>        季思妍尴尬地笑了笑，不得不收起自己的香烟，重新放回自己的包包里，踩着高跟鞋离去。
<br/>
<br/>        回到家后，季思妍便听到自己的父亲在抱怨，说上头压根就不收他的礼。
<br/>
<br/>        既然那上头都不肯收礼，那么这下头就更不会收了。
<br/>
<br/>        这件事情摆明了，有人就是要让季运恒蹲那么十多天的监狱。
<br/>
<br/>        季思妍百思不得其解，压根就猜不着那位“连少”是什么来头。难道，是某个高官家的子弟？！
<br/>
<br/>        她也就只能这么想想，不然实在是想不到，她家这位“刘阿斗”哥哥还能惹到谁。
<br/>
<br/>        但也总不能就让季运恒被关在拘留所里，毕竟季氏集团里还有很多公事等着她哥哥去处理。
<br/>
<br/>        季思妍思前顾后，权衡利弊，经过一番思想斗争后才想到一个人。
<br/>
<br/>        在这滨江城，也就只有他有权利能提前放她哥哥从拘留所里出来了。
<br/>
<br/>        翌日。
<br/>
<br/>        天还未亮，季思妍便守在了傅氏集团大厦，傅夜沉的办公室门口。
<br/>
<br/>        傅昼景这几日和赵斌一起出差了，因为对她似乎还不够信任，所以这次傅昼景出差并未带上她一起，以至于她被搁置在地利置业无人管。
<br/>
<br/>        季思妍决定趁着这个机会，好好地跟傅夜沉处处。
<br/>
<br/>        在这两兄弟之间周旋着，她真觉得自己一个头两个大。
<br/>
<br/>        傅夜沉的住所很隐秘，她花钱请人查了一晚，也未查到。
<br/>
<br/>        为了把自己弄得狼狈一点，季思妍今天只化了淡妆，连高跟鞋都没穿，就直接蹲在了傅夜沉的办公室门边。
<br/>
<br/>        她希望自己这招苦肉计能有用！
<br/>
<br/>        其实，她蹲在这里的体力已经到了极限，双脚都开始发麻了。
<br/>
<br/>        就在她快要坚持不下去的时候，电梯的楼层抵达声，几乎是给了她夹缝丛生的希望。
<br/>
<br/>        傅夜沉看到季思妍蹲在自己的办公室门口，一点都不诧异。
<br/>
<br/>        季思妍见傅夜沉朝这边走过来，身边还跟着特助郑忠威，立即从地上站起身来，哪知眼前开始发一阵黑眼晕，让有些头晕脑胀地找不着方向，打了个趔趄。
<br/>
<br/>        明明她自己的双腿已经不受大脑控制了，她却仍旧借着这次机会，有意识地将自己的身子往傅夜沉身上扑去。
<br/>
<br/>        在跌入到一个温暖的怀抱里后，季思妍立即抱住这个怀抱，一边委屈着，一边又略带撒娇地哭诉：“夜沉哥哥，你一定要救救我那个不争气的哥哥啊！”
<br/>
<br/>        “季小姐，有话好好说啊！”耳畔忽然响起一个肥腻的声音。
<br/>
<br/>        季思妍这才定睛抬眸，看到闯入自己眼帘的男人的脸，居然是郑忠威后，瞬间吓得花容失色。
<br/>
<br/>        余光里，只见傅夜沉正推门进入办公室。
<br/>
<br/>        季思妍慌忙推开郑忠威，站稳脚后，恼羞成怒地瞪了郑忠威一眼，连忙紧跟傅夜沉的身后。
<br/>
<br/>        “夜沉哥哥，我哥哥他也不知道是招惹了谁，现在被关在了拘留所里。要关十五天！我爸的公司，是不能没有我哥的啊！夜沉哥哥，你帮帮我好不好？”季思妍踱步上前，拦在了傅夜沉的面前后，伸出手来，轻轻地抓着傅夜沉的西装衣袖口，撒娇似的摇了摇。
<br/>
<br/>        她瘪着粉唇，黯然神伤地垂着眸子，吸了吸鼻子，一副欲哭无泪的样子。
<br/>
<br/>        傅夜沉抬了抬手，轻而易举地甩开了季思妍的手，坐回到自己的老板椅上，并对刚走进来的郑忠威命令道：“送客。”
<br/>
<br/>        郑忠威颔首，大步上前，走到了季思妍的跟前，向她迎了迎手：“季小姐，这边请。”
<br/>
<br/>        “夜沉哥哥，求你！”季思妍咬了咬下唇，努力让自己挤出了一滴眼泪出来。
<br/>
<br/>        郑忠威见她这楚楚可怜的样子，回头于心不忍地看了一眼傅夜沉的态度。
<br/>
<br/>        原本都以为，他傅夜沉会怜香惜玉。
<br/>
<br/>        郑忠威更以为大少爷会开口说句好话。
<br/>
<br/>        结果……
<br/>
<br/>        傅夜沉面不改色，冷酷无情地说了一个字：“滚。”
<br/>
<br/>        他这一个字，倒是真的把季思妍给吓得惊愕。
<br/>
<br/>        是她哭得不像吗？
<br/>
<br/>        还是她装得不够惨？
<br/>
<br/>        这个傅夜沉是怎么回事？
<br/>
<br/>        不知道什么叫“怜香惜玉”吗？
<br/>
<br/>        季思妍满脑子的疑惑与恼火，无处发泄不说，还得一口气全部咽到肚子里去。
<br/>
<br/>        郑忠威已经看明白了大少爷的意思，立即毫不留情地说道：“季小姐，请回吧！”
<br/>
<br/>        季思妍感觉自己已经到了无话可说的地步，但她又不能就这样空手而归，也不知道自己是哪儿来的勇气，趁着面前这两个男人不备，径直往窗户那边走去。
<br/>
<br/>        郑忠威吓得脸色都白了，眼疾手快，二话不说便直接从后抱住了季思妍。
<br/>
<br/>        可偏偏郑忠威的这双手不歪不着，直接抱到了季思妍的胸部上。
